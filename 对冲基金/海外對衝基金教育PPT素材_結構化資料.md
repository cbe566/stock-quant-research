## 一、對衝基金的定義與核心邏輯

### 1.1 什麼是對衝基金？

對衝基金（Hedge Fund）是一種以「絕對回報」為目標的私募投資基金，通過多元化策略（包括做多、做空、槓桿、衍生品等）在各種市場環境下追求正收益。

**核心特徵：**
- **絕對收益導向**：不以跑贏基準指數為目標，而是追求無論牛熊市都能獲得正回報
- **策略靈活**：可做多、做空、使用槓桿、衍生品、跨市場套利
- **有限合夥制**：通常以有限合夥（LP/GP）結構運作
- **高門檻准入**：僅面向合格投資者（高淨值個人及機構投資者）
- **流動性限制**：通常設有鎖定期（Lock-up Period），贖回頻率為季度或年度

### 1.2 對衝基金 vs 公募基金 vs 私募股權基金 — 三方比較

| 比較維度 | 對衝基金（Hedge Fund） | 公募基金（Mutual Fund） | 私募股權基金（Private Equity） |
|---------|----------------------|----------------------|---------------------------|
| **投資標的** | 股票、債券、商品、外匯、衍生品等全資產類別 | 主要為股票、債券 | 非上市公司股權 |
| **策略** | 做多做空、槓桿、套利、全球宏觀等 | 主要做多（Long-only） | 收購、重組、改善經營後退出 |
| **投資者門檻** | 高（通常 $100萬～$500萬美元以上） | 低（任何人均可投資） | 極高（$500萬～$1,000萬美元以上） |
| **監管程度** | 較低，2010年後逐漸加強 | 嚴格監管（SEC、CSRC等） | 較低 |
| **流動性** | 較低（季度/年度贖回、鎖定期） | 高（每日可申購贖回） | 極低（5-10年鎖定期） |
| **費用結構** | 2%管理費 + 20%業績報酬 | 1%-2%管理費 | 2%管理費 + 20%附帶收益 |
| **投資期限** | 中短期（數日至數年） | 靈活 | 長期（5-10年） |
| **收益目標** | 絕對收益（牛熊市皆正回報） | 相對收益（跑贏基準） | 高回報（通過經營改善） |
| **信息透明度** | 較低 | 高（定期披露持倉） | 較低 |

### 1.3 對衝基金的核心邏輯

**「對衝」的本義：** 通過建立相反的頭寸來降低市場整體風險，專注於個股選擇的超額收益（Alpha），而非市場整體漲跌（Beta）。

**公式表達：**
> 投資組合收益 = Alpha（選股能力） + Beta（市場風險） × 市場收益
>
> 對衝基金目標：最大化 Alpha，降低或消除 Beta

## 二、對衝基金發展史（1949年至今）

### 2.1 完整發展時間線

#### 📌 創始期（1949-1965）

**1949年 — 第一支對衝基金誕生**
- **創始人**：Alfred Winslow Jones（阿爾弗雷德·溫斯洛·瓊斯）
- **背景**：社會學家兼財經記者，在為《財富》雜誌撰寫市場技術分析文章時，受啟發創立投資合夥企業
- **初始資金**：10萬美元（自有4萬 + 朋友6萬）
- **核心創新**：
  - 首次將「做多」與「做空」結合在同一個投資組合中
  - 使用槓桿放大收益
  - 基金經理自身投入資金（利益綁定）
  - 發明了「2/20」收費模式（2%管理費 + 20%業績報酬）
- **策略邏輯**：買入看好的股票（做多），同時賣空看淡的股票（做空），從而「對衝」掉市場整體的系統性風險，收益完全來自選股能力

#### 📌 早期成長期（1966-1979）

**1966年 — 對衝基金引爆關注**
- 記者 Carol Loomis 在《財富》雜誌發表文章《The Jones Nobody Keeps Up With》
- 文章揭示：Jones 的基金在過去5年中跑贏最優秀的公募基金44%
- 首次使用「Hedge Fund（對衝基金）」一詞
- **影響**：文章發表後3年內，至少130支對衝基金成立

**1969年 — 傳奇基金誕生**
- George Soros（喬治·索羅斯）創立 Quantum Fund（量子基金）
- Michael Steinhardt 創立 Steinhardt Partners

**1970年代 — 第一次行業洗牌**
- 1973-1974年石油危機引發股市暴跌
- 許多對衝基金因過度使用槓桿遭受重創
- 行業規模大幅收縮，但為日後更健全的風險管理奠定基礎

#### 📌 崛起與繁榮期（1980-1999）

**1980年代 — 全球宏觀策略興起**
- Julian Robertson 創立 Tiger Management（老虎基金），成為全球宏觀策略標桿
- 金融去監管化推動行業快速發展

**1990年 — Citadel 成立**
- Ken Griffin（肯·格里芬）年僅22歲在哈佛宿舍開始交易
- 創立 Citadel（城堡投資），後發展為全球最大對衝基金之一

**1992年 — 索羅斯「擊垮英格蘭銀行」**（詳見第四章案例）

**1994年 — LTCM 成立**
- John Meriwether 與兩位諾貝爾經濟學獎得主共同創立
- 成為量化策略的典範

**1998年 — LTCM 崩盤**（詳見第四章案例）
- 行業第二次重大洗牌
- 引發對槓桿風險的深刻反思

**1990年代關鍵數據**
- 1990年：全球約600支對衝基金，管理資產約390億美元
- 1999年：增長至約4,000支基金，管理資產超過4,500億美元

#### 📌 機構化與主流化時期（2000-2007）

**2000-2002年 — 科網泡沫破裂**
- 對衝基金因可做空而表現優於公募基金
- 養老金、保險公司、主權財富基金開始大規模配置對衝基金
- 行業進入「機構化」時代

**2005年 — 行業規模突破1萬億美元**
- AIMA（另類投資管理協會）會員管理資產首次達到1萬億美元

**2007年 — 巔峰時刻**
- 管理資產接近2萬億美元
- 全球對衝基金數量超過10,000支

#### 📌 金融危機與重塑（2008-2012）

**2008年 — 全球金融危機**
- 對衝基金平均虧損15%-20%
- 投資者大規模贖回，部分基金被迫暫停贖回（Gate）
- Bernie Madoff（麥道夫）騙局曝光，涉及金額650億美元
- 雖然Madoff並非嚴格意義上的對衝基金經理，但其作為多個對衝基金的子顧問，嚴重打擊了行業信譽

**2010年 — Dodd-Frank 法案通過**
- 美國實施《多德-弗蘭克華爾街改革法案》
- 要求對衝基金向SEC註冊
- 引入Form PF報告制度，定期披露持倉和風險敞口
- 加強系統性風險監控

#### 📌 復甦與創新時代（2013-至今）

**2013-2019年 — 穩步增長**
- 量化策略和系統化交易快速崛起
- AI、機器學習深度融入投資策略
- 多策略基金（Multi-Strategy）成為主流

**2020年 — 新冠疫情衝擊**
- 全球宏觀策略表現優異
- 量化基金展現韌性

**2021年 — GameStop 事件**
- 散戶對抗對衝基金，Melvin Capital 遭受重創
- 引發關於市場公平性的廣泛討論

**2022年 — Citadel 創紀錄**
- Citadel 創造160億美元利潤，為單一對衝基金年度最高紀錄

**2025年 — 歷史性里程碑**
- 全球對衝基金管理資產首次突破5萬億美元（HFR數據：$5.15萬億）
- 行業實現2009-2010年以來首次連續兩年雙位數回報
- 2025年平均回報率11.2%

**2026年展望**
- 市場規模達5.71萬億美元，預計2031年達8.83萬億美元（CAGR 9.12%）
- 多策略基金首次超越股票策略成為最大類別
- 量化和系統化策略增長最快（CAGR 11.63%至2031年）

## 三、主要策略類型

### 3.1 股票多空策略（Equity Long/Short）

**策略邏輯**：同時做多被低估的股票和做空被高估的股票
- **佔比**：約佔對衝基金策略的30%，為最普遍的策略類型
- **優勢**：降低市場系統性風險敞口，專注Alpha收益
- **2025年表現**：回報率17.6%，為所有策略中表現最佳
- **代表基金**：Tiger Global、Lone Pine Capital、Viking Global

### 3.2 全球宏觀策略（Global Macro）

**策略邏輯**：基於對全球宏觀經濟趨勢的判斷，在股票、債券、外匯、商品等資產類別間進行方向性投資
- **關注因素**：通脹、利率、GDP增長、地緣政治、央行政策
- **特點**：投資範圍最廣，可跨市場、跨資產類別操作
- **2025年表現**：回報率7.16%
- **代表基金**：Bridgewater Associates、Soros Fund Management、Brevan Howard

### 3.3 事件驅動策略（Event-Driven）

**策略邏輯**：利用企業重大事件（併購、重組、破產、分拆等）造成的股價暫時錯誤定價來獲利
- **子策略**：
  - **併購套利（Merger Arbitrage）**：買入被收購方股票，做空收購方股票
  - **困境證券（Distressed Securities）**：投資陷入財務困難的公司債券或股票
  - **激進投資（Activist）**：買入大量股份後推動公司變革
- **2026年趨勢**：隨M&A活動加速，事件驅動策略需求上升
- **代表基金**：Elliott Management、Paulson & Co.

### 3.4 相對價值套利策略（Relative Value Arbitrage）

**策略邏輯**：利用相關證券之間的價格偏差進行套利
- **子策略**：
  - **固定收益套利（Fixed Income Arbitrage）**：利用債券定價偏差
  - **可轉債套利（Convertible Arbitrage）**：做多可轉債，做空對應股票
  - **統計套利（Statistical Arbitrage）**：利用數學模型發現價格偏差
- **特點**：低淨敞口，收益穩定，但對黑天鵝事件敏感
- **2025年表現**：回報率超過11%，優於2024年的7%
- **代表基金**：Citadel、LTCM（歷史案例）

### 3.5 量化策略（Quantitative / Systematic）

**策略邏輯**：完全依賴數學模型、統計分析和計算機程序進行投資決策
- **技術工具**：人工智能、機器學習、分布式計算、大數據分析
- **子策略**：
  - **量化股票（Quant Equity）**：因子模型驅動的股票選擇
  - **CTA / 管理期貨（Managed Futures）**：趨勢跟蹤策略
  - **高頻交易（HFT）**：毫秒級交易
- **增長趨勢**：量化策略CAGR 11.63%（至2031年），為增長最快的策略類別
- **代表基金**：Renaissance Technologies、Two Sigma、D.E. Shaw、AQR

### 3.6 多策略基金（Multi-Strategy）

**策略邏輯**：在同一基金內配置多種策略，動態調整各策略權重
- **優勢**：分散風險、降低波動、適應不同市場環境
- **2025年趨勢**：首次超越股票策略成為最大策略類別
- **代表基金**：Millennium Management、Citadel、Balyasny Asset Management

### 3.7 策略表現總覽（2025年）

| 策略類型 | 2025年回報率 | 特點 |
|---------|-------------|------|
| 股票多空 | 17.6% | 表現最佳 |
| 全球宏觀 | 7.16% | 穩健 |
| 相對價值 | >11% | 超越上年表現 |
| 行業平均 | 11.2%-12.64% | 連續兩年雙位數回報 |

## 四、知名對衝基金案例

### 4.1 Bridgewater Associates 橋水基金 — Ray Dalio（瑞·達利歐）

| 項目 | 內容 |
|------|------|
| **成立時間** | 1975年 |
| **創始人** | Ray Dalio（瑞·達利歐） |
| **總部** | 美國康涅狄格州韋斯特波特 |
| **管理資產** | 約$1,500億美元（歷史最大對衝基金之一） |
| **累計淨利潤** | $497億美元（創立以來） |
| **核心策略** | 系統化宏觀策略 |

**旗艦基金**：
- **Pure Alpha**：主動宏觀策略，追求絕對收益
- **All Weather**：全天候策略，風險平價（Risk Parity），不同經濟環境下均衡配置
  - 2016年回報率11.6%

**核心理念**：
- 「原則」（Principles）：極度透明、系統化決策、從錯誤中學習
- 經濟機器理論：將經濟視為一台機器，通過理解信貸周期、去槓桿化等機制來預測市場
- 風險平價：根據風險而非資金金額來平均分配各資產類別

### 4.2 Renaissance Technologies 文藝復興科技 — Jim Simons（吉姆·西蒙斯）

| 項目 | 內容 |
|------|------|
| **成立時間** | 1982年 |
| **創始人** | Jim Simons（吉姆·西蒙斯），數學家、前NSA密碼破譯員 |
| **總部** | 美國紐約長島 |
| **管理資產** | 約$1,300億美元 |
| **核心策略** | 純量化 / 數學模型驅動 |

**旗艦基金 — Medallion Fund（大獎章基金）**：
- **被譽為歷史上最成功的對衝基金**
- 平均年化回報率：費前 >60%，費後 ~40%
- 自1988年至2018年間，幾乎每年都實現正收益
- **2025年已對外部投資者關閉**，僅限員工投資

**核心競爭力**：
- 僱用數學家、物理學家、統計學家、計算機科學家（非傳統金融人才）
- 使用複雜的數學模型和統計方法分析海量數據
- 極度保密，外界對其具體策略知之甚少
- Simons 曾獲Oswald Veblen幾何學獎，是陳-西蒙斯定理的聯合創立者

### 4.3 Citadel 城堡投資 — Ken Griffin（肯·格里芬）

| 項目 | 內容 |
|------|------|
| **成立時間** | 1990年 |
| **創始人** | Ken Griffin（肯·格里芬） |
| **總部** | 美國佛羅里達州邁阿密 |
| **管理資產** | ~$397億美元（對衝基金）；另有Citadel Securities（做市業務） |
| **累計淨利潤** | $740億美元（創立以來） |
| **核心策略** | 多策略（股票、固收、商品、量化） |

**關鍵成就**：
- 2022年創造$160億美元利潤 — 單一對衝基金年度利潤歷史紀錄
- 多策略架構：同時運營股票、固定收益、商品、量化等多個策略團隊
- Citadel Securities 是全球最大的做市商之一，處理約佔美國股票交易量的25%

**創業傳奇**：
- Griffin 在哈佛大學宿舍安裝衛星天線接收即時報價開始交易
- 22歲創立 Citadel，起始資金460萬美元
- 2008年金融危機中基金一度虧損55%，但成功復甦

### 4.4 Soros Fund Management 索羅斯基金 — 1992年做空英鎊

| 項目 | 內容 |
|------|------|
| **創始人** | George Soros（喬治·索羅斯） |
| **核心基金** | Quantum Fund（量子基金） |
| **核心策略** | 全球宏觀 |
| **最知名交易** | 1992年做空英鎊 |

**1992年「黑色星期三」— 史上最經典的宏觀交易**

**背景**：
- 英國1990年加入歐洲匯率機制（ERM），英鎊被要求維持在對德國馬克的特定匯率區間
- 索羅斯認為英國加入ERM的匯率過高，難以維持
- 英國通脹率是德國的3倍，高利率正在傷害英國經濟
- 德國因統一後的通脹壓力維持高利率，與英國的經濟需求矛盾

**交易執行**：
- 1992年9月15日晚，德國央行行長在接受記者採訪時暗示「一兩種貨幣可能面臨壓力」
- 9月16日早晨，索羅斯將做空英鎊的頭寸從15億美元急劇增加到**100億美元**
- 英格蘭銀行拼命買入英鎊試圖維持匯率，當天花費數十億英鎊
- 英國政府在一天內兩次緊急加息（從10%到12%再到15%），仍無法阻止英鎊暴跌

**結果**：
- 英鎊對德國馬克下跌15%，對美元下跌25%
- 英國被迫退出ERM
- **索羅斯獲利超過10億英鎊**
- 量子基金價值從150億美元飆升至220億美元
- 英國財政部估計損失33億英鎊
- 索羅斯被稱為「擊垮英格蘭銀行的人」

**歷史影響**：
- 諷刺的是，脫離ERM後英國經濟反而復甦，低利率和競爭力匯率推動了多年增長
- 此事件成為全球宏觀策略的經典教材
- 展示了對衝基金在全球貨幣市場中的巨大影響力

### 4.5 LTCM 長期資本管理公司 — 1998年崩盤

| 項目 | 內容 |
|------|------|
| **成立時間** | 1994年2月 |
| **創始人** | John Meriwether（前所羅門兄弟副董事長） |
| **核心團隊** | 包括兩位諾貝爾經濟學獎得主（Myron Scholes、Robert Merton） |
| **核心策略** | 固定收益套利 / 相對價值策略 |

**輝煌時期**：
| 年份 | 回報率 |
|------|--------|
| 1994 | 20% |
| 1995 | 43% |
| 1996 | 41% |
| 1997 | 17% |

**崩盤過程**：

**根本原因 — 過度槓桿**：
- LTCM的策略本質是利用微小的定價偏差套利，每筆交易收益極低
- 為獲得高回報，LTCM使用極高槓桿（資產/資本比率高達25:1）
- 加上場外衍生品頭寸，實際槓桿遠超表面數字

**導火索 — 1998年俄羅斯債務違約**：
- 1998年8月，俄羅斯政府宣布債務違約
- LTCM雖然對衝了俄羅斯政府債券（GKO）的頭寸，通過做空盧布來對衝
- 但俄羅斯政府同時實施外匯管制，盧布做空無法兌現
- 全球「逃向安全資產」（Flight to Quality），市場流動性急劇萎縮
- LTCM幾乎所有策略同時虧損

**危機時刻**：
- 1998年8月單月虧損44%（21億美元）
- 全年虧損超過52%
- 其巨大的頭寸若被強制平倉，可能引發全球金融系統崩潰

**美聯儲出手救助**：
- 1998年9月23日，在美聯儲的協調下（但未動用美聯儲自有資金）
- 14家金融機構聯合注資**36.5億美元**進行重組
- 避免了LTCM資產在動蕩市場中被拋售引發連鎖反應
- 2000年初，基金完成清算，救助銀團收回資金

**教訓（PPT重點）**：
1. **槓桿是雙刃劍**：可以放大收益，同樣放大虧損
2. **模型有局限性**：再優秀的數學模型也無法預測極端事件（「黑天鵝」）
3. **流動性風險不可忽視**：正常時期的相關性在危機時可能完全改變
4. **諾貝爾獎也救不了**：學術榮譽不等於市場實戰能力
5. **大而不倒**：LTCM的案例預示了2008年金融危機中「系統性風險」的概念

## 五、對衝基金的費用結構

### 5.1 經典「2/20」收費模式

**管理費（Management Fee）— 2%**
- 按管理資產總額（AUM）的百分比每年收取
- **不論基金盈虧均需支付**
- 用途：支付員工薪酬、辦公場地、行政管理等運營費用
- 計算方式：若AUM為10億美元，年管理費 = 10億 × 2% = 2,000萬美元

**業績報酬（Performance Fee / Incentive Fee）— 20%**
- 按基金利潤的百分比收取
- **僅在基金獲利時才收取**
- 計算方式：若基金獲利5億美元，業績報酬 = 5億 × 20% = 1億美元
- 部分基金設有**門檻收益率（Hurdle Rate）**，如8%，僅超出門檻的部分才收取業績報酬

### 5.2 高水位線（High-Water Mark）

**定義**：確保基金經理不會因為「恢復」之前的虧損而重複收取業績報酬

**運作機制**：
- 基金淨值必須超過歷史最高點（高水位線），才能收取業績報酬
- 若基金從100跌至80，再漲回100，這段「恢復」不收取業績報酬
- 僅當淨值超過100（新的利潤部分）才收取20%業績報酬

**示例**：
| 時期 | 淨值 | 高水位線 | 是否收取業績報酬？ |
|------|------|---------|-------------------|
| 第1年末 | $120 | $120 | 是（對$20利潤收取） |
| 第2年末 | $100 | $120 | 否（低於高水位線） |
| 第3年末 | $110 | $120 | 否（仍低於高水位線） |
| 第4年末 | $140 | $140 | 是（僅對超出$120的$20收取） |

### 5.3 行業費率趨勢

| 時期 | 平均管理費 | 平均業績報酬 |
|------|----------|-------------|
| 傳統模式 | 2.00% | 20.00% |
| 2019年行業平均 | 1.50% | 19.00% |
| 2025年行業平均 | <1.50% | ~15% |
| 頂級基金（如Medallion） | 5% | 44% |

**趨勢**：
- 整體費率持續下降，投資者議價能力增強
- 但頂級表現基金（如Renaissance、Citadel）反而提高費率，因為需求遠超供給
- 機構投資者要求更多「定制化」費率安排（如管理費折扣 + 聯合投資權）

## 六、對衝基金與中國投資者的關係

### 6.1 中國投資者參與海外對衝基金的主要渠道

#### 渠道一：QDII（合格境內機構投資者）
- **機構**：經中國證監會（CSRC）批准的基金管理公司、保險公司、證券公司等
- **投資範圍**：境外證券市場的股票、債券等（對衝基金受限）
- **特點**：門檻相對較低，可通過銀行、基金公司購買QDII產品間接參與
- **局限**：受額度限制，且多數QDII產品不直接投資對衝基金

#### 渠道二：QDLP（合格境內有限合夥人）
- **試點城市**：上海、天津等
- **特點**：允許境外基金管理人在中國設立有限合夥企業，向境內合格投資者募集人民幣，投資境外對衝基金、私募股權等
- **優勢**：可直接投資海外對衝基金
- **門檻**：較高，面向機構投資者和高淨值個人

#### 渠道三：QDIE（合格境內投資企業）
- **試點城市**：深圳前海
- **特點**：與QDLP類似，為境內資金投資境外另類資產提供通道
- **投資範圍**：海外對衝基金、PE基金、VC基金、房地產基金、結構化產品等

#### 渠道四：直接海外開戶
- 部分高淨值投資者通過海外銀行賬戶、家族辦公室直接投資
- 需符合個人外匯管理規定（每人每年5萬美元便利額度）
- 大額資金需合規通道

### 6.2 投資門檻

| 投資方式 | 最低投資金額 | 適合人群 |
|---------|-------------|---------|
| QDII基金產品 | 1,000-10,000人民幣 | 一般投資者 |
| QDLP/QDIE | 100萬-300萬人民幣 | 合格投資者 |
| 直接投資海外對衝基金 | $100萬-$500萬美元 | 超高淨值/機構 |
| 通過FOF（母基金） | $25萬-$100萬美元 | 高淨值投資者 |

### 6.3 中國投資者需注意的風險

1. **外匯風險**：投資以美元計價的基金，人民幣匯率波動會影響實際收益
2. **監管風險**：中國外匯管制政策可能變化，影響資金跨境流動
3. **信息不對稱**：海外基金的信息披露制度與國內不同，語言和文化障礙
4. **法律管轄風險**：多數海外對衝基金註冊在開曼群島、BVI等離岸地區
5. **流動性風險**：鎖定期和贖回限制可能在需要資金時無法及時退出
6. **稅務合規風險**：需了解中國CRS（共同申報準則）和海外投資稅務申報要求

## 七、對衝基金的風險與監管

### 7.1 主要風險類型

#### 市場風險（Market Risk）
- 資產價格波動造成的損失
- 即使做了對衝，在極端市場中相關性可能突變

#### 槓桿風險（Leverage Risk）
- 與公募基金不同，對衝基金的槓桿**沒有法定上限**
- 依賴市場紀律和交易對手的風控約束
- LTCM案例：25:1的槓桿在危機中成為致命因素

#### 流動性風險（Liquidity Risk）
- 當高槓桿與低流動性資產結合，若投資者集中贖回，基金可能被迫低價拋售
- 2008年金融危機中，多家基金被迫「閘門」（Gate）暫停贖回

#### 交易對手風險（Counterparty Risk）
- 對衝基金的交易對手（銀行、主經紀商等）可能違約
- 複雜的場外衍生品交易中尤為突出

#### 操作風險（Operational Risk）
- 估值方法不當、內控制度薄弱、合規風險
- Madoff騙局即為極端的操作/欺詐風險案例

#### 模型風險（Model Risk）
- 量化模型在歷史數據中表現良好，但在未知的市場環境中可能失效
- 「尾部風險」（Tail Risk）難以被模型完全捕捉

### 7.2 全球監管框架

#### 美國
- **SEC（美國證券交易委員會）**：
  - 2010年《Dodd-Frank法案》要求管理資產超過1.5億美元的對衝基金顧問向SEC註冊
  - **Form PF**：定期報告基金策略、持倉、槓桿使用、交易對手風險等
  - FSOC（金融穩定監督委員會）和OFR（金融研究辦公室）有權查閱所有報告
- **CFTC（商品期貨交易委員會）**：監管涉及期貨和衍生品的對衝基金

#### 歐洲
- **AIFMD（另類投資基金經理指令）**：2011年起要求對衝基金經理在歐盟註冊，定期報告
- 報告要求涵蓋：槓桿使用、風險管理、流動性安排等

#### 亞洲
- **香港SFC（證監會）**：持牌制度，要求對衝基金經理持有第9類（資產管理）牌照
- **新加坡MAS（金融管理局）**：基金管理公司需持有CMS牌照

#### 離岸中心
- **開曼群島CIMA**：全球最受歡迎的對衝基金註冊地，監管框架相對寬鬆但逐步加強
- **BVI、百慕大**：亦為常見註冊地

### 7.3 投資者保護機制

| 保護機制 | 說明 |
|---------|------|
| **盡職調查（Due Diligence）** | 審查基金文件、管理人履歷、業績歷史、風控體系 |
| **獨立託管人（Custodian）** | 資產由獨立機構託管，防止挪用 |
| **獨立審計** | 年度由四大會計師事務所審計 |
| **獨立行政管理人（Administrator）** | 獨立計算淨值，防止人為操控 |
| **投資者信函（Side Letters）** | 大型投資者可協商額外保護條款 |
| **合規報告** | Form PF等定期監管報告 |
