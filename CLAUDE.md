## 二、數學與科學量化框架

### 核心因子模型（Fama-French + Carhart）
1. **市場因子（Market Beta）** — 市場系統性風險
2. **規模因子（SMB）** — 小型股溢價
3. **價值因子（HML）** — 低估股溢價（低P/B）
4. **動量因子（MOM）** — 過去12個月強勢股
5. **質量因子（QMJ）** — 高ROE、低負債、穩定獲利
6. **低波動因子（BAB）** — 低Beta股票溢價

### QVM 多因子組合策略（最推薦）
**步驟一：質量篩選（Quality Gate）**
- 排除 ROE < 10%、負債率 > 70%、自由現金流為負的公司

**步驟二：價值篩選（Value Screen）**
- 篩選 EV/EBIT < 15、P/B < 2、P/S 低於行業中位數

**步驟三：動量確認（Momentum Confirm）**
- 選取過去6個月漲幅前50%，且近1個月無過度超漲

**步驟四：調倉頻率**
- 每季重新審視（3/6/9/12月）

### 數學統計工具
- **回歸分析**：多因子線性回歸、Ridge/Lasso 正則化
- **時間序列**：ARIMA、GARCH（波動率預測）
- **機器學習**：隨機森林、XGBoost、LSTM（深度學習）
- **風險模型**：夏普比率、最大回撤、Calmar Ratio、VaR
- **投資組合優化**：馬可維茲均值方差、Black-Litterman模型
- **統計套利**：配對交易、協整分析

### 技術分析指標
- **趨勢**：MA20/50/200、EMA、MACD
- **動量**：RSI、KD隨機指標、威廉指標
- **波動**：布林通道、ATR平均真實波幅
- **成交量**：OBV、VWAP、成交量均線
- **支撐壓力**：斐波那契回撤、關鍵價位、籌碼密集區

## 三、籌碼分析框架

### 三大法人追蹤
- **外資**：資金規模最大，偏好大型權值股，中長線操作
  - 外資連續買超 → 中長線看多訊號
  - 外資連續賣超 → 注意系統性風險
- **投信**：波段操作為主，季底（3/6/9/12月）作帳行情
  - 投信連續買超個股 → 提早嗅出佈局線索
- **自營商**：短線套利為主，含避險部位
  - 自營商大量買進 → 可能是短線交易或期權避險

### 籌碼集中度分析
- **千張大戶持股比例**：
  - 大戶持股增加（連續幾週）→ 籌碼集中，後勢可期
  - 籌碼分散到散戶 → 主力出貨訊號，風險上升
- **券商分點追蹤**：
  - 觀察特定券商分點連續買超
  - 注意「隔日沖」慣性（當日拉高隔日出貨）
- **融資融券變化**：
  - 融資增加 + 股價上漲 → 多頭健康
  - 融資增加 + 股價不漲 → 警惕主力出貨

### 籌碼分析原則
- 觀察「連續性」，不看單日數據
- 三面向整合：基本面選股 + 籌碼面確認主力 + 技術面找進場點
- 注意假外資、主力誘多陷阱
- 嚴格執行停損，籌碼再漂亮也設好風控

## 五、分析流程（標準SOP）

1. **宏觀掃描**：確認當前景氣階段、央行政策方向
2. **板塊篩選**：根據景氣選擇強勢板塊
3. **量化選股**：執行QVM多因子篩選
4. **籌碼確認**：三大法人、千張大戶、券商分點
5. **技術確認**：趨勢、動量、成交量確認進場點
6. **風險評估**：計算夏普比率、最大回撤、VaR
7. **輸出報告**：Excel（數據）+ Word（分析）+ PPT（摘要）
