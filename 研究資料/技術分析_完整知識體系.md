# 一、價格行為分析（Price Action）

價格行為分析是技術分析的基石，直接觀察市場參與者的買賣行為在K線圖上留下的痕跡，不依賴衍生指標，被許多專業交易員視為最純粹的分析方法。

---

## 1.1 K線型態完整分類

### 1.1.1 單根K線型態

#### (A) 錘子線（Hammer）

```
形態描述：
  小實體位於K線上端，下影線至少為實體的2倍長，上影線極短或不存在。

  示意圖：
    ─      ← 微小上影線（可無）
   ███     ← 小實體（紅/綠皆可）
    │
    │      ← 長下影線（>= 2倍實體）
    │
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 下跌趨勢末端 |
| **意義** | 看漲反轉信號；空方一度將價格壓至極低，但多方強力反擊收回 |
| **可靠度** | 中高（約60-65%） |
| **確認條件** | 次日收盤價高於錘子線實體；配合放量更可靠 |
| **實體顏色** | 紅色（陽線）錘子更可靠；綠色（陰線）錘子也有效但稍弱 |
| **最佳參數** | 下影線 >= 實體2倍，上影線 < 實體10% |
| **搭配建議** | 配合RSI超賣、支撐位、成交量放大確認 |

#### (B) 倒錘子線（Inverted Hammer）

```
形態描述：
    │
    │      ← 長上影線（>= 2倍實體）
    │
   ███     ← 小實體位於K線下端
  ─        ← 微小下影線（可無）
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 下跌趨勢末端 |
| **意義** | 潛在看漲反轉；多方嘗試推升但遭壓回，仍顯示買壓出現 |
| **可靠度** | 中（約55-60%） |
| **確認條件** | 必須等待次日確認陽線；單獨出現可靠度不足 |
| **搭配建議** | 與成交量放大、支撐位共振確認 |

#### (C) 吊頸線（Hanging Man）

```
形態描述：
  與錘子線完全相同的形態，但出現在上升趨勢末端。

    ─
   ███     ← 小實體
    │
    │      ← 長下影線
    │
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 上升趨勢末端 |
| **意義** | 看跌反轉警告；多方優勢中空方開始嘗試壓低價格 |
| **可靠度** | 中（約55-60%） |
| **確認條件** | 次日收盤價低於吊頸線實體；陰線吊頸更危險 |
| **注意事項** | 容易與錘子線混淆，關鍵在於出現的趨勢環境 |

#### (D) 射擊之星（Shooting Star）

```
形態描述：
    │
    │      ← 長上影線（>= 2倍實體）
    │
   ███     ← 小實體位於K線下端
  ─        ← 微小下影線（可無）
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 上升趨勢末端 |
| **意義** | 強看跌反轉信號；多方推升至高點但遭空方強力壓回 |
| **可靠度** | 中高（約60-65%） |
| **確認條件** | 次日低開或收陰確認；配合壓力位更可靠 |
| **成功率統計** | 在關鍵壓力位出現時，成功率可達65-70% |

#### (E) 十字星（Doji）

```
形態描述：
  開盤價與收盤價幾乎相同（實體極小），上下影線長度各異。

  基本十字星      長腳十字星      蜻蜓十字星      墓碑十字星
      │               │                              │
      │               │                              │
      ┼               ┼               ┼              │
      │               │               │              ┼
      │               │               │
```

**十字星的四種子類型：**

| 子類型 | 形態特徵 | 出現位置 | 意義 |
|--------|----------|----------|------|
| **基本十字星** | 上下影線等長，實體為一橫線 | 任何位置 | 市場猶豫不決，多空平衡 |
| **長腳十字星** | 上下影線極長，波動劇烈後回到開盤價 | 趨勢末端 | 劇烈多空搏殺，強反轉信號 |
| **蜻蜓十字星** | 僅有長下影線，無上影線 | 下跌末端 | 類似錘子線的看漲信號 |
| **墓碑十字星** | 僅有長上影線，無下影線 | 上升末端 | 類似射擊之星的看跌信號 |

| 項目 | 說明 |
|------|------|
| **可靠度** | 單獨看為中性（需確認），在趨勢末端出現可靠度提升至中高 |
| **確認條件** | 必須等待後續K線方向確認；出現在關鍵位更有價值 |
| **成功率** | 配合其他信號時約60-70% |

#### (F) 陀螺線（Spinning Top）

```
形態描述：
    │
   ███     ← 小實體（比十字星略大）
    │
  上下影線長度相近，均大於實體
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 任何位置（趨勢末端更有意義） |
| **意義** | 市場猶豫，多空均未取得優勢；趨勢可能即將改變 |
| **可靠度** | 低（單獨使用不可靠，約45-50%） |
| **確認條件** | 需要後續K線方向確認，僅作為警告信號 |

#### (G) 光頭光腳陽線（Marubozu Bullish）

```
形態描述：
   ███
   ███     ← 長實體（陽線），無上下影線
   ███
```

| 項目 | 說明 |
|------|------|
| **意義** | 極強多方控制，從開盤漲到收盤無回檔 |
| **可靠度** | 高（約70%在趨勢方向上持續） |
| **應用** | 在底部出現為強烈看漲信號；在頂部出現可能是最後一衝（衰竭） |

#### (H) 光頭光腳陰線（Marubozu Bearish）

```
形態描述：
   ███
   ███     ← 長實體（陰線），無上下影線
   ███
```

| 項目 | 說明 |
|------|------|
| **意義** | 極強空方控制，從開盤跌到收盤無反彈 |
| **可靠度** | 高（約70%在趨勢方向上持續） |
| **應用** | 在頂部出現為強烈看跌信號；在底部可能是恐慌性拋售尾聲 |

### 1.1.2 雙K線組合型態

#### (A) 看漲吞沒（Bullish Engulfing）

```
形態描述：
  第一根為小陰線，第二根為大陽線，第二根實體完全包覆第一根實體。

   ▓▓
   ▓▓      ███
           ███
           ███
           ███
  Day 1    Day 2
  (陰)     (陽)
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 下跌趨勢末端 |
| **意義** | 強看漲反轉信號；多方一根K線完全吞噬空方前一日的戰果 |
| **可靠度** | 高（約70-75%） |
| **確認條件** | 第二根陽線實體越大越可靠；第二根成交量放大確認 |
| **最佳條件** | 出現在關鍵支撐位、RSI<30區域 |
| **成功率統計** | 在支撐位出現時成功率約70-75%；一般位置約60-65% |
| **搭配建議** | 配合成交量放大 + 趨勢線支撐 + RSI超賣 |

#### (B) 看跌吞沒（Bearish Engulfing）

| 項目 | 說明 |
|------|------|
| **出現位置** | 上升趨勢末端 |
| **意義** | 強看跌反轉信號；空方完全吞噬多方前一日的上漲 |
| **可靠度** | 高（約70-75%） |
| **確認條件** | 第二根陰線實體越大越可靠；配合放量 |
| **搭配建議** | 配合壓力位 + RSI超買 + MACD頂背離 |

#### (C) 看漲孕線（Bullish Harami）

```
形態描述：
  第一根為大陰線，第二根為小陽線，第二根實體完全被第一根實體包含。

  ▓▓▓▓
  ▓▓▓▓
  ▓▓▓▓  ██
  ▓▓▓▓
  Day 1   Day 2
  (大陰)  (小陽)
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 下跌趨勢末端 |
| **意義** | 潛在看漲反轉；下跌動能減弱，市場開始猶豫 |
| **可靠度** | 中（約55-60%），弱於吞沒形態 |
| **確認條件** | 需等待第三根確認陽線；孕線十字星（Harami Cross）可靠度更高 |

#### (D) 看跌孕線（Bearish Harami）

| 項目 | 說明 |
|------|------|
| **出現位置** | 上升趨勢末端 |
| **意義** | 潛在看跌反轉；上漲動能減弱 |
| **可靠度** | 中（約55-60%） |
| **確認條件** | 需要後續陰線確認 |

#### (E) 刺透線（Piercing Pattern）

```
形態描述：
  第一根為大陰線，第二根低開後強力反彈，收盤價超過第一根實體的50%以上。

  ▓▓▓▓
  ▓▓▓▓  ███
  ▓▓▓▓  ███    ← 第二根收盤至少穿過第一根實體50%
        ███
  Day 1  Day 2
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 下跌趨勢末端 |
| **意義** | 看漲反轉信號；空方跳空低開後多方強力反攻 |
| **可靠度** | 中高（約60-65%） |
| **關鍵條件** | 第二根收盤必須穿過第一根實體中點以上，穿越越多越可靠 |
| **與吞沒差異** | 刺透線第二根未完全吞沒第一根，可靠度略低 |

#### (F) 烏雲蓋頂（Dark Cloud Cover）

| 項目 | 說明 |
|------|------|
| **出現位置** | 上升趨勢末端 |
| **意義** | 看跌反轉信號；刺透線的鏡像形態 |
| **可靠度** | 中高（約60-65%） |
| **關鍵條件** | 第二根陰線開盤高於前一日高點，收盤深入前一日陽線實體50%以下 |

#### (G) 平頭頂部（Tweezer Top）

| 項目 | 說明 |

| **可靠度** | 中（約55-60%） |
| **確認條件** | 第二根為陰線更可靠；配合壓力位確認 |

### 1.1.3 三K線組合型態

#### (A) 晨星（Morning Star）

```
形態描述：
  三根K線組合：大陰線 + 小實體（星）+ 大陽線

  ▓▓▓▓          ███
  ▓▓▓▓          ███
  ▓▓▓▓          ███
  ▓▓▓▓   ██     ███
         ──
  Day 1  Day 2  Day 3
  (大陰) (星)   (大陽)
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 下跌趨勢末端 |
| **意義** | 強看漲反轉信號；代表空方動能耗盡，多方接管 |
| **可靠度** | 高（約70-75%） |
| **確認條件** | 第三根陽線收盤越接近第一根陰線開盤價越強；第二根與第一、三根有缺口更佳 |
| **變體** | 晨星十字（Morning Doji Star）：中間為十字星，可靠度更高（約75-80%） |
| **成功率統計** | 整體成功率約70-75%，在關鍵支撐位可達75-80% |
| **搭配建議** | 配合成交量第三根放大 + RSI底背離 |

#### (B) 暮星（Evening Star）

| 項目 | 說明 |
|------|------|
| **出現位置** | 上升趨勢末端 |
| **意義** | 強看跌反轉信號；晨星的鏡像 |
| **可靠度** | 高（約70-75%） |
| **確認條件** | 第三根陰線收盤越接近第一根陽線開盤價越強 |
| **變體** | 暮星十字（Evening Doji Star）：可靠度更高 |

#### (C) 三隻烏鴉（Three Black Crows）

```
形態描述：
  連續三根長陰線，每根開盤價在前一根實體內，收盤價創新低。

  ▓▓▓▓
  ▓▓▓▓
  ▓▓▓▓
    ▓▓▓▓
    ▓▓▓▓
    ▓▓▓▓
      ▓▓▓▓
      ▓▓▓▓
      ▓▓▓▓
  Day1 Day2 Day3
```

| 項目 | 說明 |
|------|------|
| **出現位置** | 上升趨勢末端或高位盤整後 |
| **意義** | 強看跌信號；空方持續施壓，多方無力反抗 |
| **可靠度** | 高（約70-75%） |
| **確認條件** | 三根陰線實體大小相近，無長下影線；成交量遞增更可靠 |
| **注意事項** | 在已下跌很久後出現可能是超賣後的持續，非最佳做空點 |

#### (D) 三個白武士（Three White Soldiers）

| 項目 | 說明 |
|------|------|
| **出現位置** | 下跌趨勢末端或低位盤整後 |
| **意義** | 強看漲信號；三隻烏鴉的鏡像 |
| **可靠度** | 高（約70-75%） |
| **確認條件** | 三根陽線實體大小相近，無長上影線；放量確認 |
| **注意事項** | 第三根若出現長上影線（受阻），可靠度下降 |

#### (E) 三法形態（Three Methods）

**上升三法（Rising Three Methods）：**
- 長陽線後跟隨2-3根小陰線（在陽線範圍內），然後一根長陽線收盤突破前高
- 意義：持續看漲形態，短暫休息後繼續上攻
- 可靠度：中高（約65%）

**下降三法（Falling Three Methods）：**
- 長陰線後跟隨2-3根小陽線（在陰線範圍內），然後一根長陰線收盤跌破前低
- 意義：持續看跌形態
- 可靠度：中高（約65%）

#### (F) 棄嬰形態（Abandoned Baby）

| 項目 | 說明 |
|------|------|
| **形態** | 類似晨星/暮星，但中間的十字星與前後K線之間有明顯缺口 |
| **出現位置** | 趨勢末端 |
| **意義** | 極強反轉信號 |
| **可靠度** | 極高（約75-80%），但出現頻率極低 |
| **成功率統計** | 看跌棄嬰成功率約77%（Quantified Strategies回測數據） |

## 1.2 經典圖表形態

### 1.2.1 反轉形態

#### (A) 頭肩頂（Head and Shoulders Top）

```
形態描述：
        頭
       /  \
  左肩/    \右肩
  /  \      /  \
 /    \    /    \
/      \  /      \
────────\/────────── ← 頸線（Neckline）
```

| 項目 | 說明 |
|------|------|
| **結構** | 左肩 → 頭（最高點）→ 右肩 → 跌破頸線 |
| **成交量特徵** | 左肩放量 → 頭部量能萎縮 → 右肩量能最低 → 跌破頸線放量 |
| **測量目標價** | 從頸線向下測量「頭部至頸線」的距離 |
| **公式** | 目標價 = 頸線價位 - (頭部最高價 - 頸線價位) |
| **成功率** | 約83-89%（多項研究一致認為是最可靠的圖表形態） |
| **失敗率** | 約11-17% |
| **確認條件** | 必須等價格有效跌破頸線（收盤價基準），回測頸線後再跌更可靠 |
| **時間框架** | 日線以上更可靠，周線最佳 |
| **注意事項** | 頸線可以是水平、向上傾斜或向下傾斜，向下傾斜的頸線更偏空 |

#### (B) 頭肩底（Inverse Head and Shoulders）

| 項目 | 說明 |
|------|------|
| **結構** | 頭肩頂的鏡像；左肩 → 頭（最低點）→ 右肩 → 突破頸線 |
| **測量目標價** | 目標價 = 頸線價位 + (頸線價位 - 頭部最低價) |
| **成功率** | 約83-89% |
| **確認條件** | 突破頸線時成交量放大 |

#### (C) 雙頂（Double Top / M頂）

```
形態描述：
    /\      /\
   /  \    /  \
  /    \  /    \
 /      \/      \
─────────────────── ← 頸線（兩峰之間的低點）
```

| 項目 | 說明 |
|------|------|
| **結構** | 兩個相近高度的峰值，中間有一個回調低點 |
| **測量目標價** | 目標價 = 頸線 - (峰值 - 頸線) |
| **成功率** | 約72-75% |
| **確認條件** | 跌破頸線確認；第二個峰值成交量低於第一個更可靠 |
| **兩峰間距** | 通常間隔1-3個月，間距過近可靠度降低 |
| **兩峰高度差** | 允許3-4%的差異，完全相同並非必要 |

#### (D) 雙底（Double Bottom / W底）

| 項目 | 說明 |
|------|------|
| **結構** | 兩個相近深度的谷值 |
| **測量目標價** | 目標價 = 頸線 + (頸線 - 谷值) |
| **成功率** | 約78-79%（略高於雙頂） |
| **確認條件** | 突破頸線時成交量放大 |

#### (E) 三重頂 / 三重底（Triple Top / Triple Bottom）

| 項目 | 說明 |
|------|------|
| **結構** | 三個相近高度的峰值（或三個谷值） |
| **成功率** | 三重底約87%，三重頂約80-85% |
| **特殊性** | 比雙頂/雙底更少見，但更可靠 |
| **測量目標價** | 與雙頂/雙底相同方法 |

#### (F) 圓弧頂 / 圓弧底（Rounding Top / Bottom）

| 項目 | 說明 |
|------|------|
| **結構** | 價格緩慢形成弧形轉向 |
| **成交量特徵** | 圓弧底的成交量呈U型（兩端大中間小） |
| **時間框架** | 通常需要較長時間形成（數週到數月） |
| **成功率** | 約65-70%，因形成時間長，一旦完成動能較強 |
| **測量目標價** | 圓弧深度/高度即為目標價測量距離 |

#### (G) V型反轉（V Reversal）

| 項目 | 說明 |
|------|------|
| **結構** | 急跌後急漲（V底）或急漲後急跌（倒V頂） |
| **特殊性** | 沒有明顯的底部/頂部構築過程 |
| **觸發條件** | 通常由突發事件驅動（財報、政策等） |
| **可靠度** | 一旦確認動能極強，但極難提前預測 |
| **交易方法** | 等待反轉初始動能確認後追入，而非預測底部 |

### 1.2.2 持續形態

#### (A) 旗形（Flag）

```
看漲旗形：                   看跌旗形：
                                /\
    /=====\                    /  \
   / ====== \              \======/
  /  =====   \             \======/
 /   旗面     \              旗面
/    (向下傾斜) \
旗桿                         旗桿
```

| 項目 | 說明 |
|------|------|
| **結構** | 急漲/急跌後（旗桿），形成小幅反向平行通道（旗面） |
| **測量目標價** | 從旗面突破點加上旗桿長度 |
| **成功率** | 看漲旗形約67-70%，看跌旗形約68% |
| **確認條件** | 突破旗面上/下邊界時放量 |
| **時間特徵** | 旗面通常持續1-3週 |
| **注意事項** | 旗面傾斜方向應與主趨勢相反 |

#### (B) 三角形

**對稱三角形（Symmetrical Triangle）：**

```
    /\
   /  \
  /    \
 /  /\  \      ← 上邊向下傾斜，下邊向上傾斜
/  /  \  \
  /    ↗
 / 突破
```

| 項目 | 說明 |
|------|------|
| **結構** | 高點逐漸降低，低點逐漸升高，形成收斂三角形 |
| **方向** | 中性，向上或向下突破皆可，但通常沿原趨勢方向突破 |
| **測量目標價** | 三角形最寬處的距離，從突破點量起 |
| **成功率** | 約60-65%（方向不確定，需等突破） |
| **最佳突破位** | 三角形形成的50-75%位置處突破最可靠 |

**上升三角形（Ascending Triangle）：**

| 項目 | 說明 |
|------|------|
| **結構** | 水平上邊界（壓力）+ 上升的下邊界（支撐逐步抬高） |
| **偏向** | 看漲（約75%向上突破） |
| **成功率** | 向上突破成功率約75-80% |
| **測量目標價** | 三角形垂直高度從突破點向上量 |

**下降三角形（Descending Triangle）：**

| 項目 | 說明 |
|------|------|
| **結構** | 水平下邊界（支撐）+ 下降的上邊界（壓力逐步降低） |
| **偏向** | 看跌（約75%向下突破） |
| **成功率** | 向下突破成功率約75-87% |

#### (C) 楔形（Wedge）

**上升楔形（Rising Wedge）：**

| 項目 | 說明 |
|------|------|
| **結構** | 兩條向上收斂的趨勢線，上邊界傾斜角度小於下邊界 |
| **意義** | 在上升趨勢中為看跌反轉；在下跌趨勢中為短暫反彈（看跌持續） |
| **成功率** | 約65-70% |
| **確認** | 跌破下邊界確認 |

**下降楔形（Falling Wedge）：**

| 項目 | 說明 |
|------|------|
| **結構** | 兩條向下收斂的趨勢線 |
| **意義** | 在下跌趨勢中為看漲反轉；在上升趨勢中為回調（看漲持續） |
| **成功率** | 約65-70% |

#### (D) 矩形（Rectangle）

| 項目 | 說明 |
|------|------|
| **結構** | 價格在水平的支撐和壓力之間橫向整理 |
| **方向** | 通常沿原趨勢方向突破（約65%） |
| **測量目標價** | 矩形的高度（上下邊界距離）從突破點量起 |
| **確認條件** | 突破時放量 |
| **時間特徵** | 持續越久，突破後動能越大 |

## 1.3 缺口理論

### 1.3.1 缺口的四種類型

#### (A) 普通缺口（Common Gap）

| 項目 | 說明 |
|------|------|
| **定義** | 在無重大事件下出現的小缺口，常見於交投清淡時 |
| **發生場景** | 盤整區間內、低成交量時段 |
| **回補機率** | 極高（>90%），通常在數日內回補 |
| **交易價值** | 低，不宜作為交易依據 |
| **特徵** | 缺口小、成交量無明顯變化 |

#### (B) 突破缺口（Breakaway Gap）

| 項目 | 說明 |
|------|------|
| **定義** | 價格突破重要整理區間時形成的缺口 |
| **發生場景** | 形態突破（如三角形、矩形突破）、重大事件驅動 |
| **回補機率** | 低（<30%），強趨勢通常不回補 |
| **成交量** | 顯著放大 |
| **交易價值** | 極高，是新趨勢啟動的重要信號 |
| **操作建議** | 順勢跟進，不宜等待回補再進場 |

#### (C) 持續缺口 / 測量缺口（Runaway Gap / Measuring Gap）

| 項目 | 說明 |
|------|------|
| **定義** | 在趨勢進行中出現的缺口，通常出現在趨勢的中段 |
| **發生場景** | 強趨勢中期，追買/追賣力道加速 |
| **回補機率** | 低（<35%） |
| **特殊用途** | 可用於測量目標價（因通常出現在趨勢中點） |
| **目標價計算** | 從趨勢起點到持續缺口的距離，從缺口向前等距延伸 |
| **交易價值** | 高，確認趨勢強度並提供目標價參考 |

#### (D) 衰竭缺口（Exhaustion Gap）

| 項目 | 說明 |
|------|------|
| **定義** | 在趨勢末端出現的缺口，代表最後一波衝力 |
| **發生場景** | 趨勢尾聲，情緒達到極端 |
| **回補機率** | 極高（>85%），通常在數日內回補 |
| **成交量** | 極大量（天量），但後續迅速萎縮 |
| **交易價值** | 極高，反轉信號 |
| **判斷方法** | 若在持續缺口後再出現缺口且很快被回補，即為衰竭缺口 |

### 1.3.2 缺口判斷流程

```
出現缺口
  │
  ├─ 在盤整區間內？ → 普通缺口（忽略）
  │
  ├─ 突破重要支撐/壓力？ → 突破缺口（順勢操作）
  │
  ├─ 在趨勢中段，方向一致？ → 持續缺口（加碼/測量目標）
  │
  └─ 在趨勢末段，成交量極大？ → 衰竭缺口（準備反向操作）
```

### 1.3.3 島狀反轉（Island Reversal）

當衰竭缺口與反方向的突破缺口連續出現，中間的K線組合被兩個缺口隔開，形成「島嶼」。這是極強的反轉信號，出現頻率低但可靠度極高（>80%）。

# 二、趨勢分析系統

## 2.1 道氏理論（Dow Theory）

道氏理論由 Charles H. Dow 創立，後經 S.A. Nelson、William Hamilton、Robert Rhea 完善，是技術分析的理論基礎。

### 六大基本原則

#### 原則一：市場反映一切（The Market Discounts Everything）

所有已知的資訊 — 經濟數據、政治事件、市場心理、未來預期 — 都已經反映在市場價格中。唯一無法預測的是「天災」等不可抗力事件，但市場會迅速消化這些衝擊。

**實際意義：** 不需要過度分析基本面「為什麼」，專注於價格「怎麼走」。

#### 原則二：市場有三種趨勢（Three Types of Trends）

| 趨勢類型 | 持續時間 | 幅度 | 類比 |
|----------|----------|------|------|
| **主要趨勢（Primary）** | 數月至數年 | >20% | 海浪的潮汐 |
| **次級趨勢（Secondary）** | 數週至數月 | 主趨勢的33-66%回調 | 海浪的波浪 |
| **短期趨勢（Minor）** | 數日至數週 | 次級趨勢的小波動 | 海浪的漣漪 |

**實際意義：** 交易前先判斷你參與的是哪一級別的趨勢，使用對應的時間框架分析。

#### 原則三：主要趨勢有三個階段（Three Phases）

**多頭市場三階段：**
1. **積累階段（Accumulation）**：聰明錢開始買入，大眾仍悲觀，成交量低
2. **大眾參與階段（Public Participation）**：趨勢明確，技術訊號明顯，成交量放大
3. **過度階段（Excess/Distribution）**：全民狂熱，新聞一片看好，聰明錢開始賣出

**空頭市場三階段：**
1. **派發階段（Distribution）**：聰明錢退出，大眾仍樂觀
2. **恐慌階段（Panic）**：急跌，大量拋售
3. **絕望階段（Despair）**：壞消息不斷但股價不再大跌，築底過程

#### 原則四：指數必須相互確認（Indices Must Confirm）

主要趨勢的信號必須在不同的市場指數中得到確認。例如道瓊工業指數和道瓊運輸指數必須同時出現趨勢信號。

**現代應用：** 標普500與那斯達克同方向確認；或股票指數與債券市場的確認。

#### 原則五：成交量必須確認趨勢（Volume Must Confirm）

- 在多頭趨勢中：上漲時放量，回調時縮量
- 在空頭趨勢中：下跌時放量，反彈時縮量
- 量價背離是趨勢即將改變的重要警告

#### 原則六：趨勢持續直到明確反轉（Trend Persists Until Reversal）

主要趨勢會持續運行，直到出現明確的反轉信號。不應因為短期的逆向波動就假設趨勢已經改變。

**實際意義：** 不要過早預測頂底，順勢交易，直到趨勢結構被打破。

## 2.2 趨勢線畫法與突破確認

### 趨勢線的基本規則

**上升趨勢線：**
- 連接兩個以上的顯著低點（越多觸點越有效）
- 價格在趨勢線上方運行

**下降趨勢線：**
- 連接兩個以上的顯著高點
- 價格在趨勢線下方運行

### 趨勢線有效性判斷

| 條件 | 越高越有效 |
|------|------------|
| 觸點數量 | 3次以上觸碰且未跌破 |
| 時間長度 | 趨勢線持續時間越長，突破意義越大 |
| 傾斜角度 | 30-45度最穩定；太陡（>60度）易被突破 |
| 時間框架 | 周線 > 日線 > 小時線 |

### 趨勢線突破的確認標準

| 確認方法 | 說明 |
|----------|------|
| **收盤價突破** | 收盤價（非盤中價）必須位於趨勢線另一側 |
| **3%突破法則** | 價格突破趨勢線至少3%（適用於長期趨勢線） |
| **2日法則** | 收盤價連續2天位於趨勢線另一側 |
| **成交量確認** | 突破時成交量顯著放大 |
| **回測確認** | 突破後回測趨勢線不再跌回，確認角色轉換（支撐變壓力或反之） |

## 2.3 移動平均線系統

### 2.3.1 三種主要移動平均線

#### SMA（簡單移動平均線）

```
公式：SMA(n) = (P1 + P2 + ... + Pn) / n
其中 P 為收盤價，n 為週期數
```

| 特性 | 說明 |
|------|------|
| **優點** | 計算簡單，平滑度高，不易被單一極端值影響 |
| **缺點** | 反應較慢，對最新價格和最舊價格賦予相同權重 |
| **適用場景** | 長期趨勢判斷（SMA200、SMA50） |

#### EMA（指數移動平均線）

```
公式：EMA(t) = Price(t) x K + EMA(t-1) x (1-K)
其中 K = 2 / (n+1)，n 為週期數
例如 EMA(12) 的 K = 2/13 = 0.1538
```

| 特性 | 說明 |
|------|------|
| **優點** | 對最新價格賦予更高權重，反應更靈敏 |
| **缺點** | 更容易產生假信號 |
| **適用場景** | 中短期交易，MACD計算基礎 |

#### WMA（加權移動平均線）

```
公式：WMA(n) = (n x Pn + (n-1) x Pn-1 + ... + 1 x P1) / (n + (n-1) + ... + 1)
```

| 特性 | 說明 |
|------|------|
| **優點** | 線性加權，最新價格權重最高 |
| **缺點** | 反應介於SMA與EMA之間 |
| **使用頻率** | 較少單獨使用 |

### SMA vs EMA vs WMA 對比

| 項目 | SMA | EMA | WMA |
|------|-----|-----|-----|
| 反應速度 | 最慢 | 最快 | 居中 |
| 平滑度 | 最高 | 最低 | 居中 |
| 假信號 | 最少 | 最多 | 居中 |
| 滯後性 | 最大 | 最小 | 居中 |
| 適用場景 | 長期趨勢 | 短期交易 | 較少使用 |

### 2.3.2 常用均線週期與意義

| 均線 | 意義 | 適用場景 |
|------|------|----------|
| MA5 | 一週均線，短線交易指標 | 日內/極短線 |
| MA10 | 兩週均線，短線趨勢 | 短線 |
| MA20 | 月線，中短期趨勢 | 波段交易 |
| MA50 | 季線，中期趨勢 | 波段/中期 |
| MA60 | 季線（亞洲市場常用） | 中期 |
| MA120 | 半年線 | 中長期 |
| MA200 | 年線，最重要的長期趨勢線 | 長期投資 |
| MA250 | 年線（亞洲市場常用，約一年交易日數） | 長期 |

### 2.3.3 黃金交叉與死亡交叉

#### 黃金交叉（Golden Cross）

```
定義：短期均線由下方向上穿越長期均線
經典組合：MA50 上穿 MA200

判斷條件：
1. MA200 走平或開始上彎
2. MA50 由下向上穿越 MA200
3. 價格位於兩條均線上方
4. 成交量配合放大
```

| 項目 | 說明 |
|------|------|
| **信號強度** | 強看漲信號 |
| **可靠度** | 約65-70%（50/200日組合） |
| **滯後性** | 較大，通常在趨勢已走一段後才出現 |
| **假信號** | 在盤整行情中容易出現假黃金交叉 |
| **最佳確認** | 價格站穩均線上方 + 成交量放大 + ADX上升 |

#### 死亡交叉（Death Cross）

```
定義：短期均線由上方向下穿越長期均線
經典組合：MA50 下穿 MA200
```

| 項目 | 說明 |
|------|------|
| **信號強度** | 強看跌信號 |
| **可靠度** | 約60-65%（略低於黃金交叉） |
| **注意** | 有時出現在下跌已經接近尾聲時，需配合其他指標確認 |

### 2.3.4 均線扣抵原理

```
核心概念：
  SMA 的計算中，新加入一個數據的同時會「扣除」最舊的一個數據。
  因此均線的走向不僅取決於今天的價格，也取決於被「扣除」的那個舊價格。

判斷方法：
  - 如果即將扣抵的舊價格較低 → 即使今天價格不漲，均線也可能上彎
  - 如果即將扣抵的舊價格較高 → 即使今天價格不跌，均線也可能下彎

實際應用：
  1. 預判均線即將轉向的時間點
  2. 判斷均線支撐/壓力是否即將失效
  3. 在轉折點提前佈局
```

### 2.3.5 Guppy 多重移動平均線（GMMA）

由澳洲交易員 Daryl Guppy 開發，使用12條EMA組成兩個群組。

**短期群組（代表交易者行為）：**
- EMA 3, 5, 8, 10, 12, 15

**長期群組（代表投資者行為）：**
- EMA 30, 35, 40, 45, 50, 60

| 信號 | 條件 | 意義 |
|------|------|------|
| **強多頭** | 短期群組在長期群組上方，兩組間距大 | 趨勢強勁，順勢做多 |
| **弱多頭** | 短期群組在上方但間距收窄 | 趨勢減弱，準備獲利了結 |
| **買入信號** | 短期群組由下向上穿越長期群組 | 新多頭趨勢確立 |
| **賣出信號** | 短期群組由上向下穿越長期群組 | 新空頭趨勢確立 |
| **盤整** | 兩組均線交織糾纏 | 觀望，等待方向明確 |
| **趨勢強度** | 同組均線間距寬（發散） | 趨勢強；間距窄（收斂）= 趨勢弱 |

**優勢：** 同時觀察短線交易者與長線投資者的行為，比單一均線更全面。

### 2.3.6 均線乖離率（BIAS）

```
公式：BIAS(n) = (收盤價 - MA(n)) / MA(n) x 100%
```

| 項目 | 說明 |
|------|------|
| **理論基礎** | 價格終究會向均線回歸（均值回歸效應） |
| **正乖離過大** | 價格遠高於均線，超買，有回調壓力 |
| **負乖離過大** | 價格遠低於均線，超賣，有反彈動力 |

**常用參數與超買超賣閾值：**

| 週期 | 超買參考值 | 超賣參考值 | 適用場景 |
|------|-----------|-----------|----------|
| BIAS(5) | +3.5% ~ +5% | -3.5% ~ -5% | 短線 |
| BIAS(10) | +5% ~ +8% | -5% ~ -8% | 短中期 |
| BIAS(20) | +8% ~ +12% | -8% ~ -12% | 中期 |
| BIAS(60) | +15% ~ +20% | -15% ~ -20% | 中長期 |

> **注意：** 不同市場、不同個股的合理乖離值差異很大。高波動股票的合理乖離更大，需根據歷史數據校準。

## 2.4 波浪理論（Elliott Wave）

由 Ralph Nelson Elliott 在1930年代提出，認為市場走勢呈現可辨識的波浪模式，反映群眾心理的週期性變化。

### 2.4.1 基本結構：五浪上升 + 三浪下降

```
完整週期結構（8浪）：

         5
        / \
       /   \
    3 /     \ A
   / \       \ /\
  /   \    4  \/  \ B
 /     \  / \      \/\
/   2   \/   \        \ C
1  / \               \
  /   \                \
```

**推動浪（Impulse Waves）1-2-3-4-5：**

| 浪序 | 特徵 | 心理狀態 |
|------|------|----------|
| **第1浪** | 趨勢初始，力度較小，常被誤認為反彈 | 極少人注意到趨勢改變 |
| **第2浪** | 對第1浪的回調，幅度通常較深 | 恐懼再次蔓延，以為下跌趨勢未結束 |
| **第3浪** | 通常最長最強的一浪，成交量最大 | 大眾開始認知新趨勢，積極參與 |
| **第4浪** | 對第3浪的調整，幅度通常較淺 | 獲利了結但仍看好後市 |
| **第5浪** | 最後一波上攻，動能往往弱於第3浪 | 全民狂熱，新聞一片看好 |

**調整浪（Corrective Waves）A-B-C：**

| 浪序 | 特徵 | 心理狀態 |
|------|------|----------|
| **A浪** | 下跌開始，常被誤認為回調 | 大多數人仍看多 |
| **B浪** | 反彈，但無法超越前高 | 仍有人認為牛市持續（多頭陷阱） |
| **C浪** | 急跌，力度通常與A浪相當 | 恐慌蔓延，市場情緒極度悲觀 |

### 2.4.2 波浪規則（不可違反）

1. **第2浪不可回撤超過第1浪的100%**（即第2浪低點不能低於第1浪起點）
2. **第3浪不可以是最短的推動浪**（在1、3、5浪中）
3. **第4浪不可與第1浪的價格範圍重疊**（在非對角三角形中）

### 2.4.3 費波那契比例在波浪中的應用

| 浪序關係 | 常見費波那契比例 |
|----------|-----------------|
| 第2浪回撤第1浪 | 50%、61.8%（最常見）、78.6% |
| 第3浪延伸第1浪 | 161.8%（最常見）、200%、261.8% |
| 第4浪回撤第3浪 | 23.6%、38.2%（最常見）、50% |
| 第5浪與第1浪 | 等長、或第1浪的61.8%、或第1浪的161.8% |
| C浪與A浪 | 等長（最常見）、161.8%的A浪 |
| B浪回撤A浪 | 38.2%、50%、61.8% |

### 2.4.4 浪的延伸、截短、交替

**延伸（Extension）：**
- 推動浪中的某一浪會延伸，通常是第3浪
- 延伸浪自身也呈現5浪結構
- 第1、3、5浪中通常只有一浪延伸

**截短（Truncation/Failure）：**
- 第5浪未能超過第3浪的高點
- 表示趨勢極度疲弱，後續調整可能更深

**交替原則（Alternation）：**
- 如果第2浪是急跌（簡單）調整，第4浪通常是橫盤（複雜）調整
- 反之亦然
- 這是指導原則而非硬性規則

### 2.4.5 調整浪型態

| 型態 | 結構 | 特徵 |
|------|------|------|
| **鋸齒形（Zigzag）** | 5-3-5 | 急跌急漲，B浪不超過起點 |
| **平台形（Flat）** | 3-3-5 | 橫向整理，B浪接近起點 |
| **三角形（Triangle）** | 3-3-3-3-3 | 五個三浪子波收斂 |
| **雙重/三重組合** | 複合型 | 以上型態的組合 |

### 2.4.6 實際操作的難點與批評

| 批評 | 說明 |
|------|------|
| **主觀性強** | 同一段行情，不同分析師可能數出不同的浪數 |
| **事後驗證** | 往往在趨勢完成後才能確認正確的數浪 |
| **規則彈性大** | 延伸、截短等使得幾乎任何走勢都可以「解釋」 |
| **無法量化** | 難以寫成明確的交易規則進行回測 |
| **時間框架問題** | 不同時間框架可能得出矛盾的結論 |

**實務建議：** 將波浪理論作為趨勢框架參考，而非精確的交易系統。搭配其他指標確認進出場，不要執著於精確數浪。

## 2.5 葛蘭碧八大法則

由 Joseph Granville 提出，分析價格與移動平均線（通常為200日MA）的關係，提供4個買入信號和4個賣出信號。

### 四大買入信號

#### 買入信號1：均線由下降轉為走平或上升，價格從下方突破均線

```
場景：長期下跌趨勢結束，均線止跌走平，價格突破均線向上
意義：新多頭趨勢可能啟動
強度：最強的買入信號（趨勢轉折）
操作：中長線建倉，設停損於近期低點下方
```

#### 買入信號2：均線持續上升，價格短暫跌破均線後迅速收回

```
場景：上升趨勢中的正常回調，價格暫跌破均線但快速站回
意義：回調買入機會，趨勢未改變
強度：中強（順勢加碼）
操作：在價格站回均線上方時買入
```

#### 買入信號3：均線持續上升，價格回落接近均線但未跌破即反彈

```
場景：上升趨勢中，價格回測均線支撐後反彈
意義：均線支撐有效，趨勢健康
強度：中（順勢操作）
操作：在均線附近或反彈確認後買入
```

#### 買入信號4：價格大幅跌破下降均線後，急速反彈向均線靠攏

```
場景：下跌趨勢中價格偏離均線過多（乖離率過大），技術性反彈
意義：超賣反彈（短線機會，非趨勢改變）
強度：最弱的買入信號（逆勢操作，僅適合短線）
操作：短線交易，目標為均線附近，嚴設停損
```

### 四大賣出信號

#### 賣出信號5：均線由上升轉為走平或下降，價格從上方跌破均線

```
場景：長期上升趨勢結束，均線止升走平，價格跌破
意義：新空頭趨勢可能啟動
強度：最強的賣出信號（趨勢轉折）
操作：中長線出清多單或建立空單
```

#### 賣出信號6：均線持續下降，價格短暫突破均線後迅速回落

```
場景：下跌趨勢中的反彈，價格暫突破均線但快速跌回
意義：反彈賣出機會，下跌趨勢未改
強度：中強（順勢減碼或做空）
```

#### 賣出信號7：均線持續下降，價格反彈接近均線但未突破即回落

```
場景：下跌趨勢中，價格上攻至均線壓力後被壓回
意義：均線壓力有效，趨勢持續
強度：中（做空或減碼）
```

#### 賣出信號8：價格大幅突破上升均線後，急速回落向均線靠攏

```
場景：上升趨勢中價格偏離均線過多（正乖離過大），回調修正
意義：超買回調（短線賣出機會，非趨勢改變）
強度：最弱的賣出信號
```

### 葛蘭碧法則的均線選擇

| 均線週期 | 適用 | 特性 |
|----------|------|------|
| 5日/10日 | 短線 | 信號多但假信號也多 |
| 20日 | 波段 | 中等靈敏度 |
| 50日 | 中期波段 | 減少假信號 |
| 120日/200日 | 中長期 | 信號最可靠但最滯後 |

## 3.1 RSI（相對強弱指標）

### 計算公式

```
RSI = 100 - [100 / (1 + RS)]
RS = 平均上漲幅度 / 平均下跌幅度

首次計算（以14日為例）：
  平均上漲 = 14日內上漲日的漲幅總和 / 14
  平均下跌 = 14日內下跌日的跌幅總和 / 14

後續平滑計算：
  平均上漲 = (前一日平均上漲 x 13 + 今日上漲幅度) / 14
  平均下跌 = (前一日平均下跌 x 13 + 今日下跌幅度) / 14
```

### 最佳參數設定

| 時間框架 | 建議RSI週期 | 超買線 | 超賣線 |
|----------|-------------|--------|--------|
| 日內（1-5分鐘） | 7-9 | 80 | 20 |
| 短線（15分-1小時） | 9-14 | 70 | 30 |
| 日線（標準） | 14（默認值） | 70 | 30 |
| 周線 | 14 | 70 | 30 |
| 月線 | 14 | 70 | 30 |
| 強趨勢市場 | 14 | 80 | 20（放寬閾值） |

### 買賣信號判斷標準

#### (A) 超買超賣

| 信號 | 條件 | 操作 | 可靠度 |
|------|------|------|--------|
| 超買 | RSI > 70 | 考慮賣出/減碼 | 中（在強趨勢中可能長期超買） |
| 超賣 | RSI < 30 | 考慮買入/加碼 | 中（在強跌勢中可能長期超賣） |
| 極端超買 | RSI > 80 | 高概率回調 | 中高 |
| 極端超賣 | RSI < 20 | 高概率反彈 | 中高 |

#### (B) 背離（Divergence）

| 類型 | 條件 | 意義 | 可靠度 |
|------|------|------|--------|
| **看漲背離** | 價格創新低，RSI未創新低 | 下跌動能減弱，可能反轉向上 | 高（約70%） |
| **看跌背離** | 價格創新高，RSI未創新高 | 上漲動能減弱，可能反轉向下 | 高（約70%） |
| **隱藏看漲背離** | 價格低點抬高，RSI低點降低 | 上升趨勢中的順勢信號 | 中高 |
| **隱藏看跌背離** | 價格高點降低，RSI高點抬高 | 下跌趨勢中的順勢信號 | 中高 |

#### (C) 中軸交叉（50線）

| 信號 | 條件 | 意義 |
|------|------|------|
| 看漲 | RSI由下向上穿越50 | 動能由空轉多 |
| 看跌 | RSI由上向下穿越50 | 動能由多轉空 |

#### (D) RSI多時間框架應用

```
方法：在至少兩個時間框架上確認RSI信號

例如：
  周線RSI > 50（大趨勢向上）
  + 日線RSI回落至30-40區域後反彈
  = 高概率的買入時機

  周線RSI < 50（大趨勢向下）
  + 日線RSI反彈至60-70區域後回落
  = 高概率的賣出時機
```

### 局限性

- 在強趨勢中，RSI可能長期處於超買/超賣區域而不回調（「鈍化」）
- 單獨使用RSI的超買超賣信號容易被震盪行情消磨
- 背離信號可能持續多次而價格不反轉

### 搭配建議

| 搭配指標 | 用途 |
|----------|------|
| MACD | RSI超賣 + MACD金叉 = 強買入信號 |
| 布林通道 | RSI超賣 + 觸碰布林下軌 = 高概率反彈 |
| 成交量 | RSI背離 + 量能萎縮 = 反轉信號更可靠 |
| 支撐壓力位 | RSI極值 + 關鍵價位 = 最佳進場點 |

## 3.2 MACD

### 計算公式

```
DIF線（快線/MACD線）= EMA(12) - EMA(26)
DEA線（慢線/Signal線）= DIF的EMA(9)
MACD柱狀圖（Histogram）= (DIF - DEA) x 2
```

### 最佳參數設定

| 時間框架 | 快線 | 慢線 | 信號線 | 說明 |
|----------|------|------|--------|------|
| 日內短線 | 5 | 13 | 1 | 更靈敏，假信號多 |
| 日線（標準） | 12 | 26 | 9 | 默認值，最廣泛使用 |
| 周線 | 12 | 26 | 9 | 中長期趨勢 |
| 保守設定 | 19 | 39 | 9 | 更少假信號 |

### 買賣信號判斷標準

#### (A) 金叉與死叉

| 信號 | 條件 | 意義 | 可靠度 |
|------|------|------|--------|
| **金叉（買入）** | DIF從下方穿越DEA | 看漲信號 | 零軸以上：中高；零軸以下：中 |
| **死叉（賣出）** | DIF從上方穿越DEA | 看跌信號 | 零軸以下：中高；零軸以上：中 |

**金叉強度分級：**
- **強金叉：** 零軸以上的金叉（上升趨勢中的回調結束），成功率約65-70%
- **中金叉：** 零軸附近的金叉（趨勢可能轉折），成功率約60%
- **弱金叉：** 零軸下方遠處的金叉（可能只是下跌中的反彈），成功率約50%

#### (B) 零軸穿越

| 信號 | 條件 | 意義 |
|------|------|------|
| **看漲** | DIF由負轉正（上穿零軸） | 中期趨勢轉多 |
| **看跌** | DIF由正轉負（下穿零軸） | 中期趨勢轉空 |

#### (C) 背離

| 類型 | 條件 | 意義 | 可靠度 |
|------|------|------|--------|
| **頂背離** | 價格新高，DIF/柱狀圖未新高 | 上漲動能衰竭 | 高（約70%） |
| **底背離** | 價格新低，DIF/柱狀圖未新低 | 下跌動能衰竭 | 高（約70%） |

#### (D) MACD柱狀圖分析

```
柱狀圖的變化比DIF/DEA交叉更早反映動量變化：

  柱狀圖由負轉正：動量由空轉多（金叉即將發生）
  柱狀圖由正轉負：動量由多轉空（死叉即將發生）
  柱狀圖持續變長：趨勢加速
  柱狀圖開始縮短：趨勢減速（即將交叉）
```

### 局限性

- 在盤整行情中產生頻繁的假交叉信號
- 滯後性明顯，不適合捕捉急速反轉
- 單一參數設定無法同時適應趨勢市和震盪市

### 搭配建議

| 搭配 | 用途 |
|------|------|
| RSI | MACD金叉 + RSI超賣區域反彈 = 強買入 |
| ADX | ADX>25確認趨勢存在 + MACD方向 = 順勢交易 |
| 均線 | 價格在MA200上方 + MACD金叉 = 多頭趨勢中的進場點 |
| 布林通道 | 布林收口後開口 + MACD方向確認 = 突破方向 |

## 3.3 KD隨機指標（Stochastic Oscillator）

### 計算公式

```
%K（快線）= [(收盤價 - N日最低價) / (N日最高價 - N日最低價)] x 100
%D（慢線）= %K的M日移動平均

快速隨機指標（Fast Stochastic）：
  %K = 原始計算值
  %D = %K的3日SMA

慢速隨機指標（Slow Stochastic）：常用
  %K = Fast %D（即原始%K的3日SMA）
  %D = Slow %K的3日SMA
```

### 最佳參數設定

| 時間框架 | %K週期 | %K平滑 | %D平滑 | 說明 |
|----------|--------|--------|--------|------|
| 日內短線 | 5 | 3 | 3 | 更靈敏 |
| 日線（標準） | 14 | 3 | 3 | 默認值 |
| 中期波段 | 21 | 3 | 3 | 更平滑 |
| 低波動市場 | 9 | 3 | 3 | 提高靈敏度 |
| 高波動市場 | 14-21 | 5 | 5 | 減少假信號 |

### 買賣信號判斷標準

#### (A) 超買超賣

| 信號 | 條件 | 操作 |
|------|------|------|
| 超買 | %K > 80 | 準備賣出（等待%K下穿%D確認） |
| 超賣 | %K < 20 | 準備買入（等待%K上穿%D確認） |

#### (B) 交叉信號

| 信號 | 條件 | 可靠度 |
|------|------|--------|
| **買入** | %K在超賣區（<20）向上穿越%D | 高 |
| **賣出** | %K在超買區（>80）向下穿越%D | 高 |
| **一般買入** | %K在50以下向上穿越%D | 中 |
| **一般賣出** | %K在50以上向下穿越%D | 中 |

#### (C) 背離

| 類型 | 條件 | 意義 |
|------|------|------|
| 看漲背離 | 價格新低，%K未新低 | 下跌動能減弱 |
| 看跌背離 | 價格新高，%K未新高 | 上漲動能減弱 |

#### (D) KD鈍化處理方法

```
問題：在強趨勢中，KD可能長期停留在超買/超賣區域（鈍化），
      此時超買賣出或超賣買入會造成大量虧損。

解決方法：
1. 搭配ADX判斷趨勢強度 — ADX>25時忽略KD超買超賣信號
2. 在KD鈍化時只做順勢交易 — KD在超買區只做多不做空
3. 使用更長週期參數 — 延長KD週期減少鈍化頻率
4. 多時間框架確認 — 高時間框架趨勢方向 + 低時間框架KD信號
5. 改用RSI或CCI替代 — 不同指標在不同行情中表現不同
```

### 局限性

- 盤整行情中頻繁產生假信號
- 強趨勢中嚴重鈍化
- 對價格跳空反應不佳

### 搭配建議

- KD超賣 + RSI超賣 + 支撐位 = 高概率反彈買入點
- KD金叉 + MACD金叉 = 雙重確認
- KD + ADX：ADX判斷是否有趨勢，KD負責進場時機

## 3.4 威廉指標（Williams %R）

### 計算公式

```
%R = [(N日最高價 - 收盤價) / (N日最高價 - N日最低價)] x (-100)

值域：0 至 -100
```

### 最佳參數設定

| 參數 | 說明 |
|------|------|
| 標準週期 | 14日 |
| 短線 | 7-10日 |
| 中期 | 21日 |

### 買賣信號

| 區域 | 數值 | 意義 |
|------|------|------|
| 超買 | 0 至 -20 | 價格接近近期高點，可能回調 |
| 超賣 | -80 至 -100 | 價格接近近期低點，可能反彈 |
| 中性 | -20 至 -80 | 無明確信號 |

### 特性與局限

- 與KD隨機指標高度相似（數學上為倒數關係）
- 反應更靈敏，但假信號也更多
- 最適合在震盪行情中使用
- 在趨勢行情中同樣會鈍化

### 搭配建議

- 與RSI、KD擇一使用即可，不需同時使用
- 配合趨勢指標（MA、ADX）過濾假信號

---

## 3.5 CCI（商品通道指數）

### 計算公式

```
CCI = (TP - SMA(TP, n)) / (0.015 x MD)

其中：
  TP（典型價格）= (最高價 + 最低價 + 收盤價) / 3
  SMA(TP, n) = TP的n日簡單移動平均
  MD（平均絕對偏差）= Σ|TPi - SMA(TP)| / n
  0.015 = Lambert常數（使得約70-80%的CCI值落在-100到+100之間）
```

### 最佳參數設定

| 時間框架 | 週期 | 說明 |
|----------|------|------|
| 日內 | 10-14 | 更靈敏 |
| 日線（標準） | 20 | 默認值 |
| 周線 | 20 | 中長期趨勢 |

### 買賣信號

| 信號 | 條件 | 意義 |
|------|------|------|
| **看漲** | CCI從-100下方回升穿越-100 | 超賣區域反彈，買入信號 |
| **看跌** | CCI從+100上方回落穿越+100 | 超買區域回調，賣出信號 |
| **強趨勢** | CCI持續在+100以上 | 強多頭趨勢（不宜做空） |
| **弱趨勢** | CCI持續在-100以下 | 強空頭趨勢（不宜做多） |
| **背離** | 價格新高/低但CCI未跟隨 | 反轉預警 |

### 特殊之處

- CCI是無界限震盪指標（可以超過+200或低於-200）
- 超過+100/-100不一定代表會反轉，在強趨勢中可能是趨勢加強的信號
- 約70-80%的時間CCI在+100到-100之間
- 產生交易信號的時間只佔20-30%

### 搭配建議

- CCI突破+100 + ADX>25 = 強趨勢確認
- CCI背離 + RSI背離 = 高概率反轉
- CCI零線交叉可作為趨勢方向的輔助確認

---

## 3.6 DMI/ADX（趨勢強度指標）

### 計算公式

```
步驟一：計算方向性運動（DM）
  +DM = 今日最高價 - 昨日最高價（若為正值且大於-DM，否則為0）
  -DM = 昨日最低價 - 今日最低價（若為正值且大於+DM，否則為0）

步驟二：計算真實波幅（TR）
  TR = max(今日最高-今日最低, |今日最高-昨日收盤|, |今日最低-昨日收盤|)

步驟三：計算方向性指標（DI）
  +DI14 = (14日平滑+DM / 14日平滑TR) x 100
  -DI14 = (14日平滑-DM / 14日平滑TR) x 100

步驟四：計算DX
  DX = (|+DI14 - -DI14| / (+DI14 + -DI14)) x 100

步驟五：計算ADX
  ADX = 14日平滑DX
  首次ADX = 14日DX的平均值
  後續ADX = (前日ADX x 13 + 今日DX) / 14
```

### 最佳參數設定

| 參數 | 標準值 | 說明 |
|------|--------|------|
| DI週期 | 14日 | Wilder原始設定 |
| ADX週期 | 14日 | 同上 |
| 短線變體 | 7-10日 | 更靈敏 |

### ADX閾值判斷

| ADX數值 | 趨勢強度 | 建議策略 |
|----------|----------|----------|
| 0-15 | 無趨勢/極弱 | 使用震盪指標，區間交易 |
| 15-20 | 弱趨勢形成中 | 觀望或輕倉 |
| 20-25 | 趨勢確立中 | 準備順勢交易 |
| 25-40 | 中強趨勢 | 積極順勢交易 |
| 40-50 | 強趨勢 | 持續持有，注意可能接近極端 |
| >50 | 極強趨勢 | 趨勢可能即將結束，準備反轉 |

### 買賣信號

| 信號 | 條件 | 意義 |
|------|------|------|
| **買入** | +DI上穿-DI，且ADX>25 | 多頭趨勢確認 |
| **賣出** | -DI上穿+DI，且ADX>25 | 空頭趨勢確認 |
| **趨勢加速** | ADX上升且>25 | 趨勢增強，繼續持有 |
| **趨勢減弱** | ADX開始下降 | 趨勢可能結束，準備出場 |
| **盤整警告** | ADX<20且走平 | 避免趨勢交易 |

### 特殊技巧

- ADX僅衡量趨勢「強度」，不判斷方向（上升或下降皆可使ADX上升）
- ADX的方向比數值更重要：ADX上升=趨勢增強，ADX下降=趨勢減弱
- 當ADX從高位（>40）開始下降時，不代表趨勢反轉，而是趨勢「減速」

### 搭配建議

- ADX>25 → 使用趨勢指標（MACD、均線交叉）
- ADX<20 → 使用震盪指標（RSI、KD、布林通道）
- +DI/-DI交叉 + ADX上升 = 最可靠的趨勢信號

## 3.7 動量指標（Momentum / ROC）

### 計算公式

```
動量值（Momentum）= 今日收盤價 - N日前收盤價
變動率（ROC）= [(今日收盤價 - N日前收盤價) / N日前收盤價] x 100%
```

### 最佳參數

| 參數 | 適用 |
|------|------|
| 10日 | 短線動量 |
| 14日 | 標準設定 |
| 20日 | 中期動量 |

### 買賣信號

| 信號 | 條件 |
|------|------|
| 看漲 | 動量由負轉正（上穿零線） |
| 看跌 | 動量由正轉負（下穿零線） |
| 背離 | 價格新高/新低但動量未跟隨 |

### 局限性

- 極為簡單的指標，信號粗糙
- 通常作為輔助確認而非獨立交易依據

## 4.1 布林通道（Bollinger Bands）

### 計算公式

```
中軌 = SMA(n)           （通常n=20）
上軌 = SMA(n) + k x σ   （通常k=2）
下軌 = SMA(n) - k x σ

其中 σ = n日收盤價的標準差

衍生指標：
  %B = (收盤價 - 下軌) / (上軌 - 下軌)
  帶寬（Bandwidth）= (上軌 - 下軌) / 中軌 x 100%
```

### 最佳參數設定

| 時間框架 | 週期(n) | 倍數(k) | 說明 |
|----------|---------|---------|------|
| 日內（5分鐘） | 10-12 | 1.5 | 更靈敏 |
| 日線（標準） | 20 | 2 | 默認值，涵蓋約95%的價格波動 |
| 周線 | 20 | 2 | 中長期 |
| 保守設定 | 20 | 2.5 | 更少假突破 |
| 激進設定 | 10 | 1.5 | 短線交易 |

### 買賣信號與交易策略

#### (A) 收口→開口策略（Bollinger Squeeze）

```
核心邏輯：
  低波動期（布林收口）→ 高波動期（布林開口）的轉換是必然的。
  擠壓（Squeeze）結束後的方向性突破通常力度強大。

判斷方法：
  1. 帶寬（Bandwidth）降至近6個月低點 = 擠壓中
  2. 布林通道收入Keltner通道內 = 更精確的擠壓判斷
  3. 等待價格突破上軌或下軌確認方向

操作：
  - 布林收口時觀望（不做方向性交易）
  - 突破上軌 + 放量 → 做多
  - 突破下軌 + 放量 → 做空
  - 停損設在中軌
```

#### (B) 通道內交易（均值回歸）

```
適用條件：帶寬穩定，無明顯趨勢（ADX<20）

操作：
  - 價格觸碰下軌 + RSI超賣 → 買入，目標中軌
  - 價格觸碰上軌 + RSI超買 → 賣出，目標中軌
  - 停損設在通道外1個ATR處
```

#### (C) 趨勢交易（Walking the Bands）

```
在強趨勢中，價格會「沿著」上軌或下軌移動（band walking）。

判斷方法：
  - 價格反覆觸碰上軌且不回到中軌 = 強上升趨勢
  - 價格反覆觸碰下軌且不回到中軌 = 強下跌趨勢

操作：
  - 不要在趨勢中做均值回歸交易！
  - 使用中軌作為順勢交易的支撐/壓力
```

#### (D) %B指標應用

| %B值 | 意義 |
|------|------|
| >1.0 | 收盤價在上軌之上（極強或超買） |
| 0.5 | 收盤價在中軌 |
| <0 | 收盤價在下軌之下（極弱或超賣） |

### 局限性

- 不提供方向性判斷，需配合其他指標確認方向
- 在強趨勢中，布林通道上下軌的觸碰不是反轉信號
- 參數（週期和標準差倍數）需根據市場波動率調整

### 搭配建議

- 布林收口 + Keltner通道 = Squeeze策略
- 布林觸軌 + RSI超買超賣 = 震盪交易
- 布林 + ADX = 判斷該用趨勢策略還是震盪策略

---

## 4.2 ATR（平均真實波幅）

### 計算公式

```
真實波幅（TR）= max(
  今日最高 - 今日最低,
  |今日最高 - 昨日收盤|,
  |今日最低 - 昨日收盤|
)

ATR(n) = TR的n日指數移動平均（或簡單移動平均）
標準週期：n = 14
```

### 最佳參數設定

| 用途 | 週期 | 說明 |
|------|------|------|
| 短線止損 | 7-10 | 更敏感的波動率衡量 |
| 標準 | 14 | 默認值 |
| 倉位管理 | 14-20 | 穩定的波動率參考 |

### 核心應用

#### (A) ATR止損設定

```
方法：以ATR的倍數設定停損距離

做多止損 = 進場價 - N x ATR
做空止損 = 進場價 + N x ATR

常用倍數：
  - 日內交易：1.0-1.5 x ATR
  - 波段交易：2.0-3.0 x ATR
  - 中長期：3.0-4.0 x ATR

優勢：止損距離隨市場波動率自動調整
  - 高波動市場 → ATR大 → 止損距離大 → 不易被洗出
  - 低波動市場 → ATR小 → 止損距離小 → 風險控制更緊
```

#### (B) ATR倉位管理

```
基於ATR的倉位計算公式：

倉位大小 = (帳戶風險金額) / (N x ATR x 每點價值)

例如：
  帳戶100萬，風險1%（1萬元）
  ATR = 5元，止損倍數N = 2
  倉位 = 10,000 / (2 x 5) = 1,000股
```

#### (C) Keltner Channel（ATR通道）

```
中軌 = EMA(20)
上軌 = EMA(20) + 2 x ATR(10)
下軌 = EMA(20) - 2 x ATR(10)
```

| 應用 | 說明 |
|------|------|
| 趨勢判斷 | 價格在上軌上方 = 強多頭；下軌下方 = 強空頭 |
| 與布林搭配 | 布林通道收入Keltner通道內 = Squeeze（低波動期） |
| 均值回歸 | 價格偏離通道後回到中軌 |

### ATR不是方向性指標

- ATR只衡量波動「幅度」，不判斷方向
- ATR上升 = 波動加大（不等於上漲）
- ATR下降 = 波動收窄（不等於下跌）

---

## 4.3 歷史波動率 vs 隱含波動率

### 歷史波動率（HV / Historical Volatility）

```
計算方法：
  1. 計算每日對數報酬率：r = ln(今日收盤 / 昨日收盤)
  2. 計算r的標準差（通常用20日或30日）
  3. 年化：HV = 標準差 x √252（252為年交易日數）

常用週期：
  - HV(10)：短期波動率
  - HV(20)：標準
  - HV(30)：中期
  - HV(60)：長期
```

### 隱含波動率（IV / Implied Volatility）

```
定義：從選擇權（期權）價格反推出來的市場預期波動率

特性：
  - 前瞻性（反映市場對未來波動的預期）
  - 受供需影響（不僅僅反映統計波動率）
  - 存在「波動率微笑」（不同履約價的IV不同）
```

### HV vs IV 的交易應用

| 情況 | 意義 | 操作建議 |
|------|------|----------|
| IV > HV | 市場過度恐慌或期權被高估 | 賣出期權策略（賣Put、鐵鷹等） |
| IV < HV | 市場過度樂觀或期權被低估 | 買入期權策略（買Call/Put） |
| IV百分位 > 80% | IV處於歷史高位 | 傾向賣出波動率 |
| IV百分位 < 20% | IV處於歷史低位 | 傾向買入波動率 |

### IV均值回歸特性

隱含波動率具有強烈的均值回歸特性。極端高IV或低IV往往會回歸到歷史平均水準，這是波動率交易策略的理論基礎。

---

## 4.4 VIX恐慌指數

### 定義與計算

```
VIX = CBOE波動率指數
衡量S&P 500指數期權未來30天的隱含波動率
常被稱為「恐慌指數」（Fear Gauge）

VIX數值大約等於市場預期未來30天的年化波動率
例如：VIX = 20 → 市場預期未來30天S&P 500的年化波動率約20%
     → 月波動率約 20/√12 ≈ 5.8%
```

### VIX閾值判斷

| VIX水準 | 市場狀態 | 歷史對應 |
|---------|----------|----------|
| <12 | 極度樂觀/自滿 | 可能是市場頂部的警告 |
| 12-17 | 正常偏低 | 牛市正常水準 |
| 17-25 | 中等波動 | 正常市場波動 |
| 25-35 | 高度不安 | 市場修正期間 |
| 35-50 | 恐慌 | 市場大幅下跌 |
| >50 | 極度恐慌 | 歷史性危機（如2008、2020年3月） |

### VIX交易應用

| 策略 | 說明 |
|------|------|
| **逆向操作** | VIX極高時（>35）是買入股票的機會（恐慌時貪婪） |
| **自滿警告** | VIX極低時（<12）警惕潛在的市場修正 |
| **VIX期貨曲線** | 正常：遠月>近月（正價差，Contango）；恐慌：近月>遠月（逆價差，Backwardation） |
| **波動率產品** | 可交易VIX期貨、VIX選擇權、VXX/UVXY等ETN/ETF |

### VIX的16法則

```
VIX / 16 ≈ 市場預期的每日波動幅度
（因為 √252 ≈ 16）

例如：VIX = 20 → 每日預期波動 ≈ 20/16 = 1.25%
      VIX = 40 → 每日預期波動 ≈ 40/16 = 2.5%
```

# 五、成交量分析

---

## 5.1 量價關係核心邏輯

成交量是技術分析的「第二維度」，反映市場參與者的「力度」與「意願」。

### 經典量價關係

| 量價組合 | 意義 | 可靠度 |
|----------|------|--------|
| **量增價升** | 健康上漲，多方主導，趨勢持續 | 高 |
| **量縮價升** | 上漲動能減弱，警惕反轉 | 中高（警告信號） |
| **量增價跌** | 恐慌拋售或主力出貨，趨勢持續下跌 | 高 |
| **量縮價跌** | 下跌動能減弱，可能接近底部 | 中（可能只是中繼） |
| **量增價平** | 多空激烈爭奪，即將選擇方向 | 中（需觀察突破方向） |
| **量縮價平** | 市場觀望，缺乏興趣 | 低（缺乏交易價值） |

### 重要量價原則

- **天量天價：** 成交量創歷史新高 + 價格在高位 = 極可能是頂部。主力需要巨量來出貨。
- **地量地價：** 成交量萎縮至極低 + 價格在低位 = 可能接近底部。賣壓枯竭。
- **量在價先：** 成交量的變化往往領先價格的變化。量能先行放大/萎縮是趨勢變化的先行指標。
- **突破需量：** 有效的價格突破必須伴隨成交量放大（至少為前幾日均量的1.5-2倍以上）。
- **無量反彈是假反彈：** 在下跌趨勢中的反彈若無量配合，通常是誘多陷阱。

---

## 5.2 OBV（能量潮）

### 計算公式

```
如果今日收盤 > 昨日收盤：OBV = 昨日OBV + 今日成交量
如果今日收盤 < 昨日收盤：OBV = 昨日OBV - 今日成交量
如果今日收盤 = 昨日收盤：OBV = 昨日OBV
```

### 買賣信號

| 信號 | 條件 | 意義 | 可靠度 |
|------|------|------|--------|
| **看漲背離** | 價格新低，OBV未新低 | 買盤暗中累積，即將反轉上漲 | 高（約65-70%） |
| **看跌背離** | 價格新高，OBV未新高 | 買盤力道不足，即將反轉下跌 | 高（約65-70%） |
| **趨勢確認** | OBV隨價格同步創新高/低 | 趨勢健康，持續持有 | 高 |
| **突破確認** | 價格突破 + OBV同步突破 | 突破有效 | 高 |

### 優勢與局限

- **優勢：** 簡單直觀，對於偵測「聰明錢」的累積/派發非常有效
- **局限：** OBV的絕對值無意義，只看趨勢方向；在盤整中可能出現假信號

---

## 5.3 VWAP（成交量加權平均價格）

### 計算公式

```
VWAP = Σ(成交量i x 典型價格i) / Σ(成交量i)
其中 典型價格 = (最高 + 最低 + 收盤) / 3

VWAP通常以日為週期重新計算（每日重置）
```

### 交易應用

| 應用 | 說明 |
|------|------|
| **機構基準** | 機構交易以VWAP為基準，買入低於VWAP視為良好執行 |
| **多空分界** | 價格 > VWAP = 多方控制；價格 < VWAP = 空方控制 |
| **支撐壓力** | VWAP本身作為動態支撐/壓力線 |
| **進場參考** | 在VWAP附近建倉，風險報酬比通常較好 |

### VWAP延伸線

```
上方1個標準差 = VWAP + 1σ
上方2個標準差 = VWAP + 2σ
下方1個標準差 = VWAP - 1σ
下方2個標準差 = VWAP - 2σ

類似布林通道，但以成交量加權
```

### 最佳參數

- 日內交易：使用當日VWAP
- 波段交易：使用周VWAP或月VWAP（錨定VWAP）
- 錨定VWAP（Anchored VWAP）：從特定日期（如財報日、重要低點）開始計算

### 局限性

- 在盤整行情中價格反覆穿越VWAP，信號雜訊大
- 隨交易日推進，VWAP的敏感度逐漸降低
- 主要適用於流動性好的市場

---

## 5.4 A/D Line（累積分配線）

### 計算公式

```
CLV（收盤位置值）= [(收盤 - 最低) - (最高 - 收盤)] / (最高 - 最低)
A/D = 前日A/D + CLV x 當日成交量
```

### 應用

| 信號 | 條件 | 意義 |
|------|------|------|
| 看漲背離 | 價格新低，A/D未新低 | 資金流入（累積） |
| 看跌背離 | 價格新高，A/D未新高 | 資金流出（派發） |
| 趨勢確認 | A/D與價格同步 | 趨勢健康 |

### 與OBV的差異

- OBV只看收盤價相對昨日是漲是跌（全有或全無）
- A/D考慮收盤價在當日高低點中的位置（更精細）

---

## 5.5 CMF（蔡金資金流量）

### 計算公式

```
CMF(n) = Σ(CLV x Volume) / Σ(Volume)    （n日期間）
CLV = [(收盤 - 最低) - (最高 - 收盤)] / (最高 - 最低)
標準週期：n = 20-21
```

### 買賣信號

| CMF值 | 意義 |
|-------|------|
| > 0 | 資金流入，看漲 |
| < 0 | 資金流出，看跌 |
| > +0.25 | 強買壓 |
| < -0.25 | 強賣壓 |

### 特點

- 有界限（-1到+1之間）
- 比A/D Line更容易判讀
- 標準週期20-21日（約一個月交易日）

---

## 5.6 成交量分布（Volume Profile）

### 核心概念

與傳統的時間成交量不同，Volume Profile顯示在每個價格水準的成交量分布。

### 關鍵元素

#### POC（Point of Control / 控制點）

```
定義：成交量最大的價格水準
意義：市場「公允價格」，具有強力支撐/壓力作用
應用：
  - 價格偏離POC後通常會被「拉回」
  - POC是當日最重要的支撐壓力位
  - 多個時間框架的POC重疊 = 超強支撐/壓力
```

#### VA（Value Area / 價值區域）

```
定義：包含總成交量70%（或68%）的價格範圍
VA High = 價值區域上界
VA Low = 價值區域下界

意義：
  - 市場認為「合理」的交易範圍
  - 價格在價值區域外通常會被拉回
  - 突破價值區域可能啟動新趨勢
```

#### HVN（High Volume Node / 高成交量節點）

```
定義：成交量集中的價格區域
特性：
  - 具有磁吸效應（價格傾向在此盤整）
  - 作為強支撐/壓力區域
  - 價格通過HVN時速度較慢
```

#### LVN（Low Volume Node / 低成交量節點）

```
定義：成交量稀疏的價格區域
特性：
  - 價格快速通過（無人感興趣的價位）
  - 不作為支撐/壓力
  - 價格一旦進入LVN區域通常會快速穿越到下一個HVN
```

### Volume Profile交易策略

| 策略 | 說明 |
|------|------|
| **POC回歸** | 價格偏離POC後做回歸交易，目標POC |
| **VA突破** | 價格突破VA High做多，跌破VA Low做空 |
| **LVN突破** | 利用LVN區域的低阻力快速獲利 |
| **HVN反轉** | 在HVN區域尋找反轉信號 |
| **裸POC** | 前幾日未被回測的POC具有磁吸效應 |

# 六、支撐壓力分析

---

## 6.1 斐波那契工具

### 6.1.1 斐波那契回撤（Fibonacci Retracement）

```
核心比例：
  23.6% — 淺回撤（強趨勢中的回調）
  38.2% — 中淺回撤（常見的第一支撐/壓力位）
  50.0% — 中間回撤（非斐波那契數，但廣泛使用）
  61.8% — 黃金比例回撤（最重要的支撐/壓力位）
  78.6% — 深度回撤（趨勢可能已改變的警告線）

畫法：
  上升趨勢回調：從低點畫到高點
  下跌趨勢反彈：從高點畫到低點
```

### 各回撤水準的特性

| 水準 | 特性 | 可靠度 | 使用建議 |
|------|------|--------|----------|
| 23.6% | 強趨勢中的微幅回調 | 低 | 僅在極強趨勢中考慮 |
| 38.2% | 常見的第一回撤支撐 | 中高 | 最常見的順勢建倉位 |
| 50.0% | 心理中間點 | 中 | 結合其他支撐使用 |
| 61.8% | 黃金比例，最關鍵的支撐 | 高 | 最重要的判斷位，跌破則趨勢可能改變 |
| 78.6% | 深度回撤 | 中 | 最後防線，跌破幾乎確認趨勢反轉 |

### 6.1.2 斐波那契延伸（Fibonacci Extension）

```
用途：在突破前高/前低後，預測價格可能達到的目標位

核心比例：
  100%   — 等距延伸
  127.2% — 第一目標位
  161.8% — 黃金延伸（最重要的目標位）
  200%   — 雙倍延伸
  261.8% — 強趨勢延伸目標

畫法：從三個點定義（波段起點、波段終點、回調終點）
```

### 6.1.3 斐波那契扇形（Fan）

```
定義：從起點畫出基於斐波那契比例（38.2%、50%、61.8%）的扇形趨勢線
用途：提供動態的支撐/壓力線
特性：隨時間推移，支撐/壓力水準會改變
```

### 6.1.4 斐波那契弧形（Arc）

```
定義：以波段高低點為直徑，畫出38.2%、50%、61.8%的弧形
用途：同時考慮價格和時間的支撐/壓力
特性：結合了價格回撤和時間因素
```

### 6.1.5 斐波那契時區（Time Zone）

```
定義：在時間軸上按斐波那契數列（1,1,2,3,5,8,13,21,34...）標記垂直線
用途：預測可能出現重要轉折的時間點
可靠度：較低，通常作為輔助參考
```

### 使用建議

- 多個斐波那契水準匯聚（Confluence）的區域是最強的支撐/壓力
- 不同波段的38.2%和61.8%交叉重疊 = 超強位
- 斐波那契不是精確的價格，而是一個「區域」（通常上下偏差1-2%）
- 永遠需要其他指標確認（K線型態、成交量、其他指標）

---

## 6.2 樞軸點（Pivot Points）

### 6.2.1 標準樞軸點

```
計算公式（基於前一交易日的高、低、收盤價）：

PP（樞軸點）= (H + L + C) / 3

第一支撐 S1 = 2 x PP - H
第一壓力 R1 = 2 x PP - L

第二支撐 S2 = PP - (H - L)
第二壓力 R2 = PP + (H - L)

第三支撐 S3 = L - 2 x (H - PP)
第三壓力 R3 = H + 2 x (PP - L)
```

### 6.2.2 Fibonacci樞軸點

```
PP = (H + L + C) / 3

S1 = PP - 0.382 x (H - L)
S2 = PP - 0.618 x (H - L)
S3 = PP - 1.000 x (H - L)

R1 = PP + 0.382 x (H - L)
R2 = PP + 0.618 x (H - L)
R3 = PP + 1.000 x (H - L)
```

### 6.2.3 Camarilla樞軸點

```
C = 收盤價, H = 最高, L = 最低, Range = H - L

S1 = C - Range x 1.1/12
S2 = C - Range x 1.1/6
S3 = C - Range x 1.1/4    ← 關鍵支撐
S4 = C - Range x 1.1/2

R1 = C + Range x 1.1/12
R2 = C + Range x 1.1/6
R3 = C + Range x 1.1/4    ← 關鍵壓力
R4 = C + Range x 1.1/2

交易策略：
  - 價格在S3-R3之間：做區間交易（S3買入，R3賣出）
  - 突破R4：做多（突破交易）
  - 跌破S4：做空（突破交易）
```

### 6.2.4 Woodie樞軸點

```
PP = (H + L + 2C) / 4    ← 給予收盤價更高權重

R1 = 2 x PP - L
R2 = PP + (H - L)

S1 = 2 x PP - H
S2 = PP - (H - L)
```

### 樞軸點的應用

| 用途 | 說明 |
|------|------|
| **日內交易** | PP以上做多，PP以下做空 |
| **目標位** | R1/R2/R3作為做多目標；S1/S2/S3作為做空目標 |
| **止損** | 在下一個支撐/壓力位設置止損 |
| **突破交易** | 突破R1後目標R2，跌破S1後目標S2 |
| **多時間框架** | 周/月樞軸點比日樞軸點更具參考價值 |

---

## 6.3 心理關卡與歷史高低點

### 心理關卡（整數位）

```
原理：人類傾向於在整數位下單（買入/賣出/停損/停利）

主要心理關卡：
  - 整百元位：100, 200, 500, 1000...
  - 指數整千點：10000, 15000, 20000...
  - 匯率整數位：1.0000, 1.1000, 1.2000...

交易應用：
  - 心理關卡附近通常有較大的掛單量
  - 突破整數位可能觸發大量停損/停利單，導致加速
  - 在整數位前方（例如99.5元）常有較強壓力
  - 跌至整數位附近（例如100元）常有較強支撐
```

### 歷史高低點

```
重要性排序：
  1. 歷史最高/最低（All-Time High/Low）
  2. 52週高低點
  3. 近期重要的擺動高點/低點（Swing High/Low）
  4. 前日/前週/前月的高低點

應用原則：
  - 歷史高點被突破後變成支撐
  - 歷史低點被跌破後變成壓力（角色轉換）
  - 越多次被測試且守住的位，被突破時的意義越大
  - 長期未回測的高低點仍具有效力
```

# 七、市場寬度指標

市場寬度指標衡量整體市場的內部健康狀況，而非個別股票或指數。這些指標能揭示指數表現是否得到廣泛個股的支持。

---

## 7.1 漲跌家數比（A/D Ratio）

### 計算公式

```
A/D比率 = 上漲家數 / 下跌家數

A/D線（累積）：
  如果上漲家數 > 下跌家數：A/D線 = 前日值 + (上漲家數 - 下跌家數)
  如果下跌家數 > 上漲家數：A/D線 = 前日值 - (下跌家數 - 上漲家數)
```

### 應用

| 信號 | 條件 | 意義 |
|------|------|------|
| 健康上漲 | 指數上漲 + A/D線上漲 | 多數個股參與上漲，趨勢可持續 |
| 看跌背離 | 指數創新高 + A/D線未創新高 | 上漲僅由少數權值股帶動，頭部警告 |
| 看漲背離 | 指數創新低 + A/D線未創新低 | 下跌家數減少，底部可能形成 |
| 弱勢反彈 | 指數反彈 + A/D線持平/下跌 | 反彈廣度不足，可能是假反彈 |

---

## 7.2 新高新低比

### 計算公式

```
淨新高 = 創52週新高的個股數 - 創52週新低的個股數
新高新低比率 = 新高家數 / (新高家數 + 新低家數)
```

### 應用

| 信號 | 條件 | 意義 |
|------|------|------|
| 強多頭 | 新高家數持續增加，新低家數極少 | 市場內部強健 |
| 頭部警告 | 指數新高但新高家數減少 | 參與度下降，可能見頂 |
| 底部信號 | 新低家數開始減少並轉正 | 拋售壓力消退 |
| 投降信號 | 新低家數暴增（>40%的個股創新低） | 恐慌性拋售，可能接近底部 |

---

## 7.3 McClellan Oscillator / Summation Index

### McClellan Oscillator 計算公式

```
淨上漲家數（Net Advances）= 上漲家數 - 下跌家數

McClellan Oscillator = 淨上漲家數的19日EMA - 淨上漲家數的39日EMA

（與MACD概念類似，但應用於市場寬度數據）
```

### McClellan Oscillator 信號

| 數值 | 意義 |
|------|------|
| > 0 | 市場寬度偏多 |
| < 0 | 市場寬度偏空 |
| > +100 | 極度看漲（寬度推力），通常預示持續上漲 |
| < -100 | 極度看跌（寬度崩塌），可能接近超賣反彈 |
| 穿越零線 | 中期趨勢可能改變 |

### 寬度推力（Breadth Thrust）

```
定義：McClellan Oscillator從極端負值（如-150以下）快速回升到正值
意義：強力的看漲信號，通常預示中期以上的持續上漲
歷史紀錄：寬度推力後3-6個月市場上漲的機率約85%以上
```

### McClellan Summation Index

```
計算：McClellan Oscillator的累積總和
用途：衡量中長期市場寬度趨勢

信號：
  - Summation Index > 0 且上升 = 中期偏多
  - Summation Index < 0 且下降 = 中期偏空
  - 與指數的背離提供長期反轉預警
```

---

## 7.4 騰落指標

### 計算公式

```
騰落指標（ADL / Advance-Decline Line）=
  Σ(每日上漲家數 - 每日下跌家數)

這是漲跌家數差的累積值
```

### 應用

與A/D比率類似，但以累積線的形式呈現，更容易觀察長期趨勢和背離。

| 關鍵應用 | 說明 |
|----------|------|
| 牛市確認 | ADL隨指數同步創新高 |
| 熊市預警 | 指數新高但ADL走平或下跌（2007年即出現此信號） |
| 底部確認 | ADL領先指數向上反轉 |

---

## 7.5 百分比指標

### 定義

```
% of Stocks Above 200MA = 股價高於200日均線的個股數 / 總個股數 x 100%
% of Stocks Above 50MA  = 股價高於50日均線的個股數 / 總個股數 x 100%
```

### 應用

| 數值 | 200MA版本意義 | 50MA版本意義 |
|------|--------------|--------------|
| >70% | 市場強勢（可能過熱） | 短期極度樂觀 |
| 50-70% | 健康多頭 | 正常偏多 |
| 30-50% | 市場轉弱 | 中性偏弱 |
| <30% | 市場極弱（可能超賣） | 短期極度悲觀 |

### 背離信號

- 指數創新高但百分比指標下降 = 上漲收窄，頂部警告
- 指數創新低但百分比指標上升 = 內部轉強，底部信號

## 8.1 海龜交易法則

由 Richard Dennis 和 William Eckhardt 在1983年設計的完整趨勢追蹤系統。

### 完整規則

#### 市場選擇

```
標準：高流動性、波動率合理的市場
原始涵蓋：商品期貨、外匯、債券期貨
現代應用：個股、ETF、加密貨幣均可
跨市場分散：選擇低相關性的多個市場
```

#### 倉位管理（Position Sizing）

```
核心變數 N = 20日ATR（衡量市場波動率）

1個單位（Unit）的計算：
  Unit = (帳戶權益 x 1%) / (N x 每點價值)

含義：每個Unit承擔帳戶1%的風險

最大持倉限制：
  - 單一市場：最多4個Units
  - 高度相關市場：最多6個Units
  - 單一方向（多或空）：最多12個Units
  - 總持倉：最多20個Units
```

#### 進場規則

**系統一（短期，20日突破）：**

```
買入：價格突破20日最高點（且上一筆信號未獲利）
賣出：價格跌破20日最低點
如果上一筆信號獲利 → 跳過本次信號（由系統二接手）
```

**系統二（長期，55日突破）：**

```
買入：價格突破55日最高點
賣出：價格跌破55日最低點
所有信號都執行（不跳過）
```

#### 加碼規則（Pyramiding）

```
最多加碼3次（共4個Units）
加碼間隔：每上漲0.5N加碼1個Unit

例如：
  第1個Unit：突破價格
  第2個Unit：突破價格 + 0.5N
  第3個Unit：突破價格 + 1.0N
  第4個Unit：突破價格 + 1.5N
```

#### 停損規則

```
基於ATR的停損：
  止損距離 = 2N（即2個ATR）

每次加碼後調整所有持倉的停損：
  新停損 = 最新建倉價 - 2N

最大風險 = 帳戶權益的2%（單一交易）
```

#### 出場規則

**系統一出場：**
```
做多出場：價格跌破10日最低點
做空出場：價格突破10日最高點
```

**系統二出場：**
```
做多出場：價格跌破20日最低點
做空出場：價格突破20日最高點
```

### 海龜系統的績效特徵

| 指標 | 典型值 |
|------|--------|
| 勝率 | 35-40%（低勝率） |
| 盈虧比 | 2.5:1 至 4:1（高盈虧比） |
| 最大回撤 | 30-40% |
| 年化報酬 | 15-25%（歷史回測） |

### 現代適用性

- 原始參數在現代市場中效果可能下降（更多人知道此系統）
- 核心理念（趨勢追蹤、ATR倉位管理、嚴格紀律）依然有效
- 建議根據市場特性調整突破週期和ATR倍數

## 8.2 均值回歸系統

### 核心理念

```
假設：價格偏離均值後會回歸均值
適用環境：盤整行情、無明顯趨勢時期
```

### 系統設計範例

```
進場規則：
  做多：RSI(2) < 10 且 收盤價 < MA(200)（或 > MA(200)取決於風險偏好）
  做空：RSI(2) > 90 且 收盤價 > MA(200)

出場規則：
  做多出場：收盤價 > MA(5) 或持有超過5天
  做空出場：收盤價 < MA(5) 或持有超過5天

止損：2個ATR

替代方案：
  - 布林通道下軌買入，中軌賣出
  - 價格跌至VWAP下方2個標準差買入
  - 乖離率超過一定閾值時反向操作
```

### 績效特徵

| 指標 | 典型值 |
|------|--------|
| 勝率 | 60-75%（高勝率） |
| 盈虧比 | 0.5:1 至 1.5:1（低盈虧比） |
| 持倉時間 | 短（1-10天） |
| 最大風險 | 趨勢市場中會大幅虧損 |

### 注意事項

- 均值回歸策略在趨勢市場中會嚴重虧損
- 必須搭配趨勢過濾器（如ADX<25時才啟用）
- 止損必須嚴格執行（否則一次虧損可能抵消多次獲利）

---

## 8.3 突破交易系統

### 核心理念

```
假設：價格突破重要關卡後會繼續朝突破方向運動
適用環境：盤整結束後、波動率由低轉高
```

### 系統設計範例

```
進場規則：
  做多：收盤價突破N日最高點 + 成交量 > 20日均量的1.5倍 + ADX>25
  做空：收盤價跌破N日最低點 + 成交量放大 + ADX>25

N的選擇：
  - 日內：5-10日
  - 短線：20日
  - 中線：55日

出場規則：
  - 追蹤停損：最高價（做多時）回落2個ATR
  - 時間停損：突破後5日內未再創新高則出場
  - 目標停利：以斐波那契延伸或前波高度為目標

止損：
  - 突破前的盤整區間中點
  - 或突破價位下方1-2個ATR
```

### 績效特徵

| 指標 | 典型值 |
|------|--------|
| 勝率 | 40-55% |
| 盈虧比 | 2:1 至 3:1 |
| 假突破率 | 30-50% |

### 假突破的處理

```
減少假突破的方法：
1. 等待收盤確認（不用盤中突破）
2. 要求成交量配合（至少1.5倍均量）
3. 使用ATR過濾（突破幅度 > 0.5 ATR）
4. 要求ADX確認趨勢存在
5. 突破後第二天確認（不立即進場）
```

## 8.4 多指標組合策略

### 設計原則

```
黃金法則：
1. 不超過3-4個指標（太多會導致信號矛盾且過度最佳化）
2. 每個指標負責不同面向：
   - 趨勢方向（MA、ADX）
   - 進場時機（RSI、KD、MACD）
   - 風險管理（ATR、布林通道）
   - 確認信號（成交量、OBV）
3. 避免使用功能重複的指標（如同時用RSI和KD）
```

### 推薦組合一：趨勢追蹤

```
指標組合：
  1. MA200 — 判斷主趨勢方向（價格在MA200上方=多頭）
  2. MACD — 進場時機（金叉買入，死叉賣出）
  3. ADX — 確認趨勢強度（>25才交易）
  4. ATR — 止損和倉位計算

進場條件（做多）：
  - 價格 > MA200（多頭環境）
  - ADX > 25（趨勢存在）
  - MACD金叉（進場信號）

出場條件：
  - MACD死叉 或 追蹤停損（2 ATR）
```

### 推薦組合二：波段交易

```
指標組合：
  1. 布林通道 — 波動率和超買超賣
  2. RSI — 動量確認
  3. 成交量均線 — 量能確認
  4. ATR — 止損設定

進場條件（做多）：
  - 價格觸碰布林下軌
  - RSI < 30（超賣）
  - 出現看漲K線型態

出場條件：
  - 價格觸碰布林中軌或上軌
  - RSI > 70
```

### 推薦組合三：多時間框架

```
框架：
  周線 — 判斷主趨勢（MA50方向）
  日線 — 找進場機會（MACD + RSI）
  4小時/1小時 — 精確進場點（K線型態）

規則：
  1. 周線確認方向（只做順勢方向）
  2. 日線等待回調到支撐位 + RSI超賣
  3. 低時間框架確認反轉K線後進場
```

## 8.5 資金管理規則

### 核心原則

#### 固定比例風險（最推薦）

```
每筆交易的最大風險 = 帳戶總值的1-2%

計算倉位大小：
  倉位 = (帳戶總值 x 風險比例) / (進場價 - 停損價)

例如：
  帳戶：100萬元
  風險比例：1%（1萬元）
  進場價：50元
  停損價：48元（差距2元）
  倉位 = 10,000 / 2 = 5,000股
```

#### Kelly公式（理論最佳）

```
f* = (bp - q) / b

f* = 最佳投入比例
b = 盈虧比（平均獲利 / 平均虧損）
p = 勝率
q = 敗率（1-p）

例如：勝率55%，盈虧比2:1
f* = (2 x 0.55 - 0.45) / 2 = 0.325 = 32.5%

實務上使用半Kelly（f*/2 = 16.25%）以降低波動
```

#### 固定金額法

```
每筆交易固定投入金額
例如：帳戶100萬，每筆投入10萬元
簡單但不考慮波動率差異
```

### 風險管理規則

```
必須遵守的鐵律：

1. 單筆交易風險 ≤ 帳戶的 1-2%
2. 同時持倉總風險 ≤ 帳戶的 6-10%
3. 單一板塊/市場風險 ≤ 帳戶的 3-5%
4. 連續虧損達到帳戶 10% 時，暫停交易並檢討
5. 單月最大虧損 ≤ 帳戶的 10-15%
6. 每筆交易在進場前必須設定停損
7. 永遠不要向下攤平虧損部位
```

### 停損策略

| 停損類型 | 方法 | 適用場景 |
|----------|------|----------|
| **固定ATR停損** | 進場價 +/- N x ATR | 通用 |
| **圖表停損** | 設在支撐/壓力位外 | 形態交易 |
| **追蹤停損** | 隨獲利移動停損 | 趨勢交易 |
| **時間停損** | 持倉N天未獲利則出場 | 短線交易 |
| **百分比停損** | 固定虧損百分比（如-3%） | 簡單管理 |
| **波動率停損** | 根據布林通道寬度調整 | 波動率交易 |

### 盈虧比的重要性

```
不同勝率下需要的最低盈虧比：

  勝率30% → 盈虧比至少 2.5:1 才能盈利
  勝率40% → 盈虧比至少 1.6:1
  勝率50% → 盈虧比至少 1.1:1
  勝率60% → 盈虧比至少 0.7:1
  勝率70% → 盈虧比至少 0.5:1

結論：低勝率系統（如趨勢追蹤）依賴高盈虧比
      高勝率系統（如均值回歸）可以接受低盈虧比
      但無論如何，盈虧比 x 勝率 > 1 才能長期盈利
```

## 8.6 交易日誌與績效評估

### 交易日誌記錄項目

```
每筆交易必須記錄：

基本資訊：
  - 日期/時間
  - 交易標的
  - 做多/做空
  - 進場價格與數量
  - 出場價格與數量
  - 手續費/稅費

策略資訊：
  - 使用的交易系統/策略
  - 進場理由（哪些指標/型態觸發）
  - 出場理由（停損/停利/其他）
  - 預設的停損位和目標位
  - 實際盈虧

主觀記錄：
  - 交易當時的市場環境（趨勢/盤整/波動率）
  - 情緒狀態（冷靜/衝動/恐懼/貪婪）
  - 執行品質（是否嚴格遵守規則）
  - 事後檢討（做對了什麼/做錯了什麼）
```

### 核心績效指標

| 指標 | 計算方式 | 優秀標準 |
|------|----------|----------|
| **總報酬率** | (期末資產 - 期初資產) / 期初資產 | 視市場而定 |
| **年化報酬率** | (1 + 總報酬率)^(252/交易日數) - 1 | >15% |
| **勝率** | 獲利筆數 / 總交易筆數 | 視策略而定 |
| **盈虧比** | 平均獲利 / 平均虧損 | >1.5:1 |
| **期望值** | (勝率 x 平均獲利) - (敗率 x 平均虧損) | > 0 |
| **夏普比率** | (年化報酬 - 無風險利率) / 年化標準差 | >1.0 |
| **索提諾比率** | (年化報酬 - 無風險利率) / 下行標準差 | >1.5 |
| **最大回撤** | 峰值到谷值的最大跌幅 | <20% |
| **Calmar比率** | 年化報酬 / 最大回撤 | >1.0 |
| **復原因子** | 總淨利 / 最大回撤 | >3.0 |
| **獲利因子** | 總獲利 / 總虧損 | >1.5 |

### 績效評估流程

```
每週檢討：
  1. 本週交易筆數、勝率、盈虧
  2. 是否嚴格遵守交易規則
  3. 有無情緒化交易
  4. 下週需要注意的重要事件

每月檢討：
  1. 月度績效指標計算
  2. 最佳與最差交易分析
  3. 策略有效性評估
  4. 資金曲線分析（是否平滑上升）

每季/每年檢討：
  1. 完整績效報告
  2. 策略參數是否需要調整
  3. 市場環境是否已改變
  4. 交易系統的改進方向
  5. 與基準（如S&P 500）的比較
```

| 錯誤 | 對策 |

| 過度交易（頻繁進出） | 設定每日/每週最大交易筆數 |
| 不設停損 | 進場前必須先設好停損，否則不開倉 |
| 移動停損（往虧損方向） | 停損只能移向獲利方向，永遠不向回移 |
| 報復性交易 | 連續虧損2-3筆後強制休息 |
| 過度槓桿 | 嚴格遵守每筆1-2%風險上限 |
| 憑感覺交易 | 建立明確的交易規則並嚴格執行 |
| 追高殺低 | 耐心等待回調到支撐位再進場 |
| 過度最佳化 | 使用Walk-Forward Analysis驗證，樣本外測試 |
| 忽視市場環境 | 每天先判斷市場環境再選擇策略 |
| 同時使用太多指標 | 限制在3-4個，每個負責不同面向 |
