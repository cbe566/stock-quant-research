## 標準分析流程（SOP）

```
第1步：宏觀掃描
  → 景氣階段判斷 → 央行政策方向 → 板塊輪動選擇

第2步：量化篩選
  → QVM多因子（質量→價值→動量）→ 因子中性化 → 排名

第3步：基本面確認
  → 財報分析 → DCF估值 → F-Score/Z-Score品質檢查

第4步：籌碼面確認
  → 法人/機構動向 → 大戶集中度 → 融資融券 → 內部人交易

第5步：技術面進場
  → 趨勢確認 → 動量指標 → 成交量驗證 → 支撐壓力位

第6步：風險管理
  → 倉位計算（ATR/Kelly） → 止損設定 → 組合相關性檢查

第7步：執行與監控
  → 交易執行 → 日誌記錄 → 定期回顧 → 動態調整
```
