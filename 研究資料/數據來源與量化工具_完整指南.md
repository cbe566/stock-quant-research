### 1.1 美股數據

#### 1.1.1 Yahoo Finance API（yfinance）

| 項目 | 說明 |
|------|------|
| **功能** | 股價（日/週/月/分鐘線）、財務報表、股息、分割、期權鏈、市場摘要、全球日曆 |
| **安裝** | `pip install yfinance` |
| **費用** | 完全免費（非官方 API，使用 Yahoo 公開端點） |
| **數據範圍** | 全球主要市場股票、ETF、指數、外匯、加密貨幣 |
| **限制** | 非官方 API，Yahoo 可能隨時變更端點；高頻請求可能被限速 |
| **版本** | v1.0+（2025年12月里程碑版本，新增全球日曆、市場摘要功能） |

**使用範例：**

```python
import yfinance as yf

# 下載單一股票歷史數據
aapl = yf.Ticker("AAPL")
hist = aapl.history(period="1y")  # 近一年日線
print(hist.head())

# 獲取財務報表
income_stmt = aapl.income_stmt        # 損益表
balance_sheet = aapl.balance_sheet     # 資產負債表
cashflow = aapl.cashflow               # 現金流量表

# 批量下載多檔股票
data = yf.download(["AAPL", "MSFT", "GOOGL"], start="2024-01-01", end="2026-03-01")
print(data["Close"].head())

# 獲取股息與分割資訊
dividends = aapl.dividends
splits = aapl.splits

# 獲取期權鏈
options = aapl.options  # 可用到期日列表
opt_chain = aapl.option_chain(options[0])  # 第一個到期日的期權資料
print(opt_chain.calls.head())

# 獲取即時報價資訊
info = aapl.info
print(f"市值: {info.get('marketCap')}")
print(f"本益比: {info.get('trailingPE')}")
```

**優點：**
- 完全免費，無需 API Key
- 數據範圍廣泛，覆蓋全球市場
- 與 pandas 深度整合，返回 DataFrame
- 社群活躍，持續更新維護
- v1.0 新增全球日曆（財報日、IPO、經濟事件、股票分割）

**缺點：**
- 非官方 API，穩定性無法保證
- 分鐘線數據僅保留近60天
- 無即時串流數據（有延遲）
- 高頻大量請求可能被限速

#### 1.1.2 FRED（Federal Reserve Economic Data）

| 項目 | 說明 |
|------|------|
| **功能** | 美國宏觀經濟指標：GDP、CPI、失業率、利率、貨幣供給等 |
| **安裝** | `pip install fredapi` |
| **費用** | 免費（需申請 API Key） |
| **數據範圍** | 超過 84 萬筆經濟時間序列，來自 118 個數據源 |
| **頻率** | 年、季、月、週、日 |

**使用範例：**

```python
from fredapi import Fred

# 初始化（需先到 https://fred.stlouisfed.org/docs/api/api_key.html 申請）
fred = Fred(api_key='你的API_KEY')

# 獲取美國 GDP 季度數據
gdp = fred.get_series('GDP')
print(gdp.tail())

# 獲取 CPI 消費者物價指數
cpi = fred.get_series('CPIAUCSL')

# 獲取聯邦基金利率
fed_rate = fred.get_series('FEDFUNDS')

# 獲取失業率
unemployment = fred.get_series('UNRATE')

# 獲取 10年期-2年期國債利差（殖利率曲線）
t10y2y = fred.get_series('T10Y2Y')

# 搜尋特定指標
results = fred.search('consumer price index')
print(results[['title', 'frequency']].head(10))

# 獲取 VIX 恐慌指數
vix = fred.get_series('VIXCLS')

# 獲取美元指數相關數據
dxy = fred.get_series('DTWEXBGS')  # 貿易加權美元指數
```

**優點：**
- 美國宏觀經濟數據最權威來源
- 數據品質極高，官方數據
- 歷史數據回溯至1940年代
- API 穩定，文檔完善
- 完全免費

**缺點：**
- 僅涵蓋宏觀經濟數據，無個股數據
- 以美國數據為主
- 部分指標更新頻率較低

#### 1.1.3 SEC EDGAR

| 項目 | 說明 |
|------|------|
| **功能** | SEC 監管文件：10-K、10-Q、8-K 年報/季報、13F 機構持倉、內部人交易 |
| **安裝** | `pip install edgartools` 或 `pip install sec-edgar-downloader` |
| **費用** | 完全免費（無需 API Key） |
| **數據範圍** | 所有在美國上市的公司監管文件 |

**使用範例：**

```python
# === 方法一：edgartools（推薦，功能最全） ===
from edgar import *

# 設置身份識別（SEC 要求）
set_identity("your_name your_email@example.com")

# 搜尋公司
company = Company("AAPL")
print(company)

# 獲取最新 10-K 年報
filing_10k = company.get_filings(form="10-K").latest(1)
print(filing_10k)

# 獲取 13F 機構持倉（追蹤對沖基金持倉）
berkshire = Company("BRK-B")
filings_13f = berkshire.get_filings(form="13F-HR")
latest_13f = filings_13f.latest(1)

# 獲取財務報表數據
financials = company.financials
print(financials.income_statement)

# === 方法二：sec-edgar-downloader ===
from sec_edgar_downloader import Downloader

dl = Downloader("MyCompany", "my_email@example.com")

# 下載蘋果公司最近5份 10-K 年報
dl.get("10-K", "AAPL", limit=5)

# 下載 13F 持倉報告
dl.get("13F-HR", "0001067983", limit=4)  # 巴菲特的波克夏
```

**優點：**
- 官方數據源，最權威的美股財報數據
- 完全免費，無需付費訂閱
- 13F 持倉可追蹤機構投資者動向
- edgartools 提供結構化 Python 物件

**缺點：**
- 原始數據需要解析（XBRL 格式）
- SEC 限制請求頻率（10次/秒）
- 數據更新有延遲（按監管報告週期）

| **數據範圍** | A股全市場、港股通、主要指數、基金、期貨 |
| **官網** | https://tushare.pro |

**使用範例：**

```python
import tushare as ts

# 設置 Token（需到 https://tushare.pro 註冊）
ts.set_token('你的TOKEN')
pro = ts.pro_api()

# 獲取 A 股日線數據
df = pro.daily(ts_code='000001.SZ', start_date='20250101', end_date='20260328')
print(df.head())

# 獲取股票基本資訊
stock_basic = pro.stock_basic(
    exchange='',
    list_status='L',  # L=上市
    fields='ts_code,symbol,name,area,industry,list_date'
)

# 獲取財務指標
fina = pro.fina_indicator(ts_code='000001.SZ', start_date='20240101')

# 獲取資金流向（北向資金/南向資金）
hsgt = pro.moneyflow_hsgt(start_date='20260101', end_date='20260328')

daily_basic = pro.daily_basic(
    ts_code='000001.SZ',
    trade_date='20260327',
    fields='ts_code,trade_date,turnover_rate,pe,pb,ps'

# 獲取三大法人（機構）持倉數據
inst_hold = pro.stk_holdernumber(ts_code='000001.SZ')
```

**優點：**
- A 股數據最全面的平台之一

#### 1.2.3 BaoStock（證券寶）

| 項目 | 說明 |
|------|------|
| **功能** | A 股歷史行情（日/週/月/5分鐘/15分鐘/30分鐘/60分鐘線）、財務數據、指數成分股 |
| **安裝** | `pip install baostock` |
| **費用** | 完全免費 |
| **數據範圍** | A 股（上海、深圳） |

**使用範例：**

```python
import baostock as bs
import pandas as pd

# 登入系統
lg = bs.login()

# 獲取歷史 A 股日線數據
rs = bs.query_history_k_data_plus(
    "sh.600000",  # 浦發銀行
    "date,code,open,high,low,close,volume,amount,adjustflag",
    start_date='2025-01-01',
    end_date='2026-03-28',
    frequency="d",       # d=日線, w=週線, m=月線
    adjustflag="3"        # 3=不復權, 1=後復權, 2=前復權
)

# 轉換為 DataFrame
data_list = []
while (rs.error_code == '0') & rs.next():
    data_list.append(rs.get_row_data())
df = pd.DataFrame(data_list, columns=rs.fields)
print(df.head())

# 獲取季度財務指標
profit = bs.query_profit_data(code="sh.600000", year=2025, quarter=4)

# 獲取滬深300成分股
hs300 = bs.query_hs300_stocks(date="2026-03-28")

# 登出
bs.logout()
```

**優點：**
- 完全免費，無積分限制
- 數據穩定，較少出錯
- 支持分鐘線數據
- 提供復權數據
- 適合初學者

**缺點：**
- 僅支持 A 股，無港股/美股
- 數據更新較慢（通常 T+1）
- 功能相比 Tushare/AKShare 較少
- 需要 login/logout，使用略繁瑣

```python
def neg_sharpe(weights, returns_mean, cov_matrix, rf=0.02):
    port_ret = np.dot(weights, returns_mean)
    port_vol = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
    return -(port_ret - rf) / port_vol

n_assets = 4
constraints = ({'type': 'eq', 'fun': lambda x: np.sum(x) - 1})
bounds = tuple((0, 1) for _ in range(n_assets))
x0 = np.array([1/n_assets] * n_assets)

result = optimize.minimize(
    neg_sharpe, x0,
    args=(returns.mean(), np.eye(n_assets) * 0.04, 0.02),
```

```python
import backtrader as bt

# 定義策略
class SmaCross(bt.Strategy):
    params = (
        ('fast', 10),
        ('slow', 30),
    )

    def __init__(self):
        # 計算移動平均
        sma_fast = bt.ind.SMA(period=self.p.fast)
        sma_slow = bt.ind.SMA(period=self.p.slow)
        # 交叉信號
        self.crossover = bt.ind.CrossOver(sma_fast, sma_slow)

    def next(self):
        if not self.position:
            if self.crossover > 0:  # 黃金交叉 → 買入
                self.buy()
        elif self.crossover < 0:    # 死亡交叉 → 賣出
            self.close()

cerebro.addstrategy(SmaCross)
```

#### 2.6.1 PyTorch

| **適用** | LSTM 股價預測、Transformer 模型 |

```python
import torch
import torch.nn as nn

# LSTM 股價預測模型
class StockLSTM(nn.Module):
    def __init__(self, input_size=5, hidden_size=64, num_layers=2, output_size=1):
        super().__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers,
                           batch_first=True, dropout=0.2)
        self.fc = nn.Linear(hidden_size, output_size)

    def forward(self, x):
        # x 形狀: (batch, seq_len, features)
        lstm_out, _ = self.lstm(x)
        # 取最後一個時間步
        out = self.fc(lstm_out[:, -1, :])
        return out

# 初始化模型
model = StockLSTM(input_size=5, hidden_size=64)
criterion = nn.MSELoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

# 訓練循環
for epoch in range(100):
    model.train()
    optimizer.zero_grad()
    output = model(X_train_tensor)
    loss = criterion(output, y_train_tensor)
    loss.backward()
    optimizer.step()

    if (epoch + 1) % 10 == 0:
        print(f'Epoch [{epoch+1}/100], Loss: {loss.item():.6f}')
```

```python
class MovingAverageStrategy(QCAlgorithm):
    def Initialize(self):
        self.SetStartDate(2023, 1, 1)
        self.SetEndDate(2025, 12, 31)
        self.SetCash(100000)

        self.spy = self.AddEquity("SPY", Resolution.Daily).Symbol
        self.fast = self.SMA(self.spy, 10)
        self.slow = self.SMA(self.spy, 30)

    def OnData(self, data):
        if not self.fast.IsReady:
            return

        if self.fast.Current.Value > self.slow.Current.Value:
            self.SetHoldings(self.spy, 1.0)
        else:
            self.Liquidate()
```

### 完整量化分析流程範例

```python
"""
完整量化分析流程：從數據獲取到策略回測
"""
import yfinance as yf
import pandas as pd
import pandas_ta as ta
import numpy as np
from pypfopt import EfficientFrontier, risk_models, expected_returns

# ========== 第一步：數據獲取 ==========
tickers = ['AAPL', 'MSFT', 'GOOGL', 'AMZN', 'JPM', 'JNJ', 'XOM', 'GLD']
data = yf.download(tickers, start='2023-01-01', end='2026-03-28')
close = data['Close']
returns = close.pct_change().dropna()

# ========== 第二步：技術分析（以 AAPL 為例） ==========
aapl = yf.download('AAPL', start='2023-01-01', end='2026-03-28')
aapl.ta.strategy("all")  # 計算所有技術指標

# ========== 第三步：因子篩選（QVM 策略） ==========
# 質量因子：ROE > 10%
# 價值因子：PE < 20
# 動量因子：過去 6 個月漲幅 > 中位數
momentum_6m = close.pct_change(126).iloc[-1]
selected = momentum_6m[momentum_6m > momentum_6m.median()].index.tolist()
print(f"動量篩選通過: {selected}")

# ========== 第四步：投資組合優化 ==========
mu = expected_returns.mean_historical_return(close[selected])
S = risk_models.sample_cov(close[selected])
ef = EfficientFrontier(mu, S)
weights = ef.max_sharpe(risk_free_rate=0.04)
cleaned = ef.clean_weights()
print("\n最優配置:")
for t, w in cleaned.items():
    if w > 0:
        print(f"  {t}: {w:.1%}")
ef.portfolio_performance(verbose=True, risk_free_rate=0.04)

# ========== 第五步：風險評估 ==========
portfolio_returns = returns[selected].dot(pd.Series(cleaned)[selected])
sharpe = portfolio_returns.mean() / portfolio_returns.std() * np.sqrt(252)
max_dd = (portfolio_returns.cumsum() - portfolio_returns.cumsum().cummax()).min()
var_95 = np.percentile(portfolio_returns, 5)

print(f"\n風險指標:")
print(f"年化夏普比率: {sharpe:.2f}")
print(f"最大回撤: {max_dd:.2%}")
print(f"VaR(95%): {var_95:.2%}")
```

## 附錄：常見問題與注意事項

### 1. 數據品質檢查清單

- [ ] 是否有缺失值？（停牌、假日）
- [ ] 是否需要前復權/後復權？
- [ ] 時區是否統一？
- [ ] 是否存在生存者偏差？（退市股票）
- [ ] 是否存在前視偏差？（Look-ahead bias）
- [ ] 數據頻率是否一致？

### 2. 常見陷阱

| 陷阱 | 說明 | 解決方案 |
|------|------|----------|
| 前視偏差 | 使用了未來才知道的數據 | 嚴格使用時間序列交叉驗證 |
| 過擬合 | 在歷史數據上表現完美 | 樣本外測試、Walk-forward 分析 |
| 生存者偏差 | 只用了存活的股票 | 包含已退市股票數據 |
| 交易成本忽略 | 回測不含手續費、滑點 | 設置合理的交易成本 |
| 流動性假設 | 假設可以按歷史價格成交 | 加入滑點模型、限制持倉比例 |
