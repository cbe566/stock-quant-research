## 一、市場結構差異

### 1.1 美國市場

| 項目 | 說明 |
|------|------|
| **交易所** | NYSE、NASDAQ、AMEX、OTC Markets |
| **交易時間** | 東部時間 9:30–16:00；盤前 4:00–9:30、盤後 16:00–20:00 |
| **結算制度** | T+1（2024年5月28日起由T+2縮短） |
| **做空機制** | 完善：融券、賣空（Reg SHO）、期權策略、反向ETF |
| **漲跌幅限制** | 無個股漲跌停限制；全市場熔斷機制（S&P 500 跌7%/13%/20%） |
| **上市制度** | 註冊制，退市制度成熟（不符標準自動啟動退市程序） |
| **投資者結構** | 機構主導，約80%交易量由機構與演算法交易貢獻 |
| **高頻交易** | 占交易量50%以上；演算法交易占60–75%整體交易量 |
| **做市商** | 做市商制度成熟，Citadel Securities 約占美股交易量20% |
| **計價貨幣** | 美元（USD） |
| **佣金成本** | 主流券商零佣金（Robinhood、Schwab、Fidelity等），期權另加$0.65/合約 |

**關鍵特徵**：
- 全球流動性最好的市場，市值約50萬億美元
- 盤前盤後交易活躍，重大消息常在盤外反映
- 期權市場極度發達，0DTE（當日到期期權）交易量爆發性增長
- SPAC、Direct Listing 等創新上市方式
- Meme Stock 現象（散戶透過社群媒體集體行動）

### 1.2 中國市場（A股）

| 項目 | 說明 |
|------|------|
| **交易所** | 上交所（SSE）、深交所（SZSE）、北交所（BSE） |
| **板塊** | 主板（滬深）、創業板（深）、科創板（滬）、北交所 |
| **交易時間** | 9:15–9:25 集合競價、9:30–11:30 連續競價、13:00–15:00 連續競價 |
| **結算制度** | T+1（買入當日不能賣出，次日方可賣出） |
| **漲跌幅限制** | 主板 ±10%、創業板/科創板 ±20%、北交所 ±30%、ST股 ±5% |
| **新股首日** | 主板首日漲幅上限44%；創業板/科創板前5日無漲跌幅 |
| **做空機制** | 極度受限：2024年7月起全面暫停轉融券業務，融券餘額大幅萎縮 |
| **投資者結構** | 散戶交易量占約80%，但持股市值僅約20%；機構持股約80% |
| **上市制度** | 由審批制轉向註冊制（2023年全面推行），退市常態化加速 |
| **計價貨幣** | 人民幣（CNY），外資通過陸港通/QFII參與 |
| **交易成本** | 佣金0.015%–0.3%（雙向）；印花稅0.1%（僅賣出）；過戶費0.001% |

**關鍵特徵**：
- 典型「政策市」：政策對市場影響遠超基本面
- 散戶主導交易→短期波動大、投機氛圍重
- 散戶貢獻80%交易量但僅獲得約10%利潤（Neuberger Berman 研究）
- T+1制度限制日內交易，催生獨特的「打板」策略
- 2024年退市23家公司，2019年至今累計退市212家，退市常態化
- 「國家隊」（匯金、社保基金等）在市場低迷時托市
- 北向資金（外資）被視為「聰明錢」，2025年中長期資金新增入市超1萬億元
- 融券幾乎停擺：2024年7月暫停轉融券後，融券餘額由700億降至不足300億

### 1.3 歐洲市場

| 項目 | 說明 |
|------|------|
| **主要交易所** | 倫敦LSE、法蘭克福Xetra、巴黎/阿姆斯特丹Euronext、瑞士SIX、馬德里BME |
| **主要指數** | STOXX 600、FTSE 100、DAX 40、CAC 40、IBEX 35、SMI |
| **交易時間** | 各國略異，大致CET 9:00–17:30 |
| **結算制度** | T+2（歐盟正在討論縮短至T+1） |
| **多幣種** | 歐元（EUR）、英鎊（GBP）、瑞士法郎（CHF）、瑞典克朗（SEK）等 |
| **做空機制** | 完善但有臨時禁空歷史（2008年金融危機、2020年COVID期間6國曾禁空） |
| **做空揭露** | 淨空頭倉位 ≥0.1% 須向監管報告；≥0.5% 須公開披露 |
| **投資者結構** | 機構主導，主權財富基金、退休基金為重要參與者 |
| **監管框架** | MiFID II/MiFIR（2025年9月完成修訂轉換）、MAR、UCITS |
| **交易成本** | 因國家而異；英國印花稅0.5%（買入）；法國金融交易稅0.3%（大型股） |

**關鍵特徵**：
- 多國多幣種→匯率風險是必須面對的核心變數
- 分紅文化濃厚，特別是英國：STOXX 600 股息率約2.5%
- ESG投資全球領先，2026年7月起ESG評級機構須由ESMA授權監管
- 市場分散性高，跨交易所套利機會存在
- 奢侈品（LVMH、Hermès）、工業（Siemens）、醫藥（Roche、Novo Nordisk）為特色產業
- 能源價格（特別是天然氣）對歐洲工業股影響極大
- MiFID II 要求更高的交易透明度與最佳執行義務
- 2026年起允許研究費用與執行費用重新捆綁（但需透明披露）

## 二、基本面分析差異

### 2.1 會計準則差異

| 維度 | US GAAP（美國） | CAS/ASBE（中國） | IFRS（歐洲） |
|------|----------------|-----------------|-------------|
| 準則體系 | 規則導向（Rules-based） | 與IFRS趨同90-95%，但有延遲 | 原則導向（Principles-based） |
| 固定資產計價 | 歷史成本 | 僅允許歷史成本 | 可選歷史成本或重估值 |
| 減值回轉 | 不允許回轉 | 不允許回轉 | 允許回轉（商譽除外） |
| 研發費用 | 全部費用化（軟體除外） | 開發階段可資本化 | 開發階段可資本化 |
| 存貨計價 | 允許LIFO | 不允許LIFO | 不允許LIFO |
| 收入確認 | ASC 606 | 與IFRS 15趨同 | IFRS 15 |
| 租賃 | ASC 842 | 與IFRS 16趨同 | IFRS 16 |
| 財報頻率 | 季報（10-Q）+年報（10-K） | 季報+半年報+年報 | 主要為半年報+年報 |

**分析師注意事項**：
- 比較中美公司財務指標時，需注意研發費用資本化差異對利潤的影響
- IFRS允許減值回轉可能使歐洲公司利潤波動更大
- 中國會計準則對新IFRS的採納有延遲，跨國比較需確認是否適用同版準則
- US GAAP的Non-GAAP指標（Adjusted EBITDA等）在美股分析中極為重要

---

### 2.2 估值方法差異

#### 美股估值重點

| 估值指標 | 適用場景 | 說明 |
|----------|---------|------|
| Forward P/E | 最廣泛使用 | 基於未來12個月預期EPS |
| PEG Ratio | 成長股篩選 | P/E除以EPS增長率，<1被視為低估 |
| EV/EBITDA | 併購估值標準 | 排除資本結構差異 |
| P/S（營收比） | 虧損型科技股 | SaaS公司、初創期企業 |
| DCF模型 | 機構標準方法 | 自由現金流折現，WACC為折現率 |
| Non-GAAP指標 | 科技股常用 | Adjusted EBITDA、Adjusted EPS |
| SaaS指標 | 軟體即服務 | ARR、NDR、Rule of 40、CAC/LTV |
| 回購調整EPS | 大型股 | 回購對EPS增長的貢獻拆分 |

**美股估值關鍵邏輯**：
- 市場高度前瞻→Forward指標比Trailing更受重視
- 科技股估值接受度高→P/S 10x、20x在高增長公司屬正常
- 回購是重要的股東回報方式→需拆分「內生增長」與「回購增長」
- 分析師一致預期（Consensus Estimate）對股價影響極大
- Earnings Surprise（盈餘驚喜）驅動短期股價反應

#### A股估值重點

| 估值指標 | 適用場景 | 說明 |
|----------|---------|------|
| P/E（TTM） | 最常用 | 滾動12個月市盈率 |
| P/B | 銀行、地產、國企 | 國企P/B<1常見，不代表低估 |
| P/E（行業分位數） | 板塊比較 | 因板塊估值差異大，須用分位數 |
| 政策紅利估值 | 新能源、半導體等 | 政策扶持帶來的額外估值溢價 |
| 殼價值 | 小盤股 | 被收購重組的潛在價值（註冊制後大幅下降） |
| PEV（P/EV） | 保險股 | 內含價值倍數 |
| NAV折價 | 地產股 | 淨資產價值折價 |

**A股估值關鍵邏輯**：
- 散戶主導→概念炒作頻繁，估值泡沫化常見
- 不同板塊估值差異極大（科創板可達100x P/E，銀行股僅5x）
- 國企估值長期偏低→「中特估」（中國特色估值體系）概念試圖修正
- 殼價值因註冊制推進和退市常態化正在消失
- 政策對估值的影響可能超過基本面本身
- 需特別關注關聯交易、少數股東權益、政府補貼對利潤的影響

#### 歐股估值重點

| 估值指標 | 適用場景 | 說明 |
|----------|---------|------|
| Dividend Yield | 最受重視 | 歐洲投資者極重視現金回報 |
| EV/EBITDA | 主流估值 | 跨國比較標準工具 |
| P/E（Shiller CAPE） | 長期估值 | 週期調整後更穩定 |
| FCF Yield | 價值投資 | 自由現金流收益率 |
| Sum-of-Parts | 多元集團 | 歐洲大型集團常用分部估值法 |
| ESG調整估值 | 日益重要 | ESG評分高的公司享估值溢價 |

**歐股估值關鍵邏輯**：
- 分紅是核心：歐洲投資者對Dividend Yield的要求遠高於美國
- 匯率是隱藏變數：跨國公司營收受匯率波動影響，需做幣種拆分
- IFRS vs GAAP差異影響跨大西洋比較（特別是減值與研發資本化）
- ESG評分正在實質影響估值溢價和資金配置
- 集團折價（Conglomerate Discount）在歐洲仍然普遍存在
- 部分行業（奢侈品、半導體設備）享有全球稀缺性溢價

| 市場 | 核心驅動力 | 強勢行業 | 分析重點 |
|------|-----------|---------|---------|
| **美國** | 科技創新、平台經濟 | 科技（Mag 7）、AI、醫藥、金融 | 技術護城河、TAM、用戶增長 |

## 三、技術分析差異

### 3.1 美股技術分析特點

**適用性：★★★★☆（趨勢性強，技術分析有效性高）**

| 特點 | 說明 |
|------|------|
| 趨勢性強 | 美股趨勢延續性好，均線策略表現優異 |
| 均線系統 | 50日、200日均線是機構重要參考，「黃金交叉」和「死亡交叉」有效 |
| 缺口理論 | 跳空缺口有較高的回補概率，缺口分析可靠 |
| OPEX效應 | 期權到期日（每月第3個週五）對股價產生磁吸效應（Max Pain理論） |
| 0DTE期權 | 當日到期期權交易量爆發，導致盤中波動放大 |
| 算法影響 | 50%+ 交易為算法執行→經典形態可能被「假突破」利用 |
| 盤前盤後 | 重大消息在盤外反映，隔日開盤可能跳空→需關注盤前價格 |
| 支撐壓力 | 整數關口、歷史高點、斐波那契位置有效 |

**最有效技術工具**：
1. 均線系統（MA50、MA200）
2. MACD（趨勢確認）
3. RSI（超買超賣，但美股超買可以持續更久）
4. 布林通道（波動率通道）
5. VWAP（日內重要參考價）
6. 期權 Gamma Exposure（GEX）分析

**美股特有注意事項**：
- 美股可以連續上漲數月不回調，RSI超買不代表立即回落
- 高頻交易造成的「閃崩」（Flash Crash）需設好止損
- 財報季前後技術分析可靠度下降（基本面事件主導）

---

### 3.2 A股技術分析特點

**適用性：★★★★★（散戶多→行為可預測→技術分析有時更有效）**

| 特點 | 說明 |
|------|------|
| 漲跌停扭曲 | 漲跌停板使技術指標失真，須特殊處理 |
| T+1限制 | 買入後當日無法賣出→日內策略受限，「半日戰法」盛行 |
| 散戶行為 | 追漲殺跌模式明顯→短期反轉策略有效 |
| 打板策略 | 追漲停板（漲停板戰法）是A股獨有的技術流派 |
| 連板股 | 連續漲停的「妖股」有獨特技術形態（一字板、T字板等） |
| 分時圖 | 因T+1限制，分時圖分析對次日操作具參考價值 |
| 龍頭戰法 | 每輪行情的龍頭股識別是A股技術分析的核心 |
| 缺口理論 | A股缺口回補概率較美股更高 |
| 量價關係 | 成交量分析在A股極為重要（「天量天價、地量地價」） |

**最有效技術工具**：
1. 量價關係（成交量是A股的「靈魂指標」）
2. 漲停板分析（封板量、開板次數）
3. 均線系統（MA5、MA10短期；MA20、MA60中期）
4. MACD（底背離、頂背離）
5. KD隨機指標（適合震盪市）
6. 籌碼分布圖（成本分布）

**A股特有注意事項**：
- 政策面可以瞬間改變技術趨勢，「一根大陽線改變信仰」
- 漲停板封死時技術指標失效（成交量極低、價格不動）
- 集合競價（9:15–9:25）的分析對當日走勢判斷重要
- 午後（13:00–14:30）是主力常見的操作時段
- 尾盤（14:30–15:00）拉升或殺跌對次日開盤有暗示作用

---

### 3.3 歐股技術分析特點

**適用性：★★★☆☆（機構主導、波動溫和、均值回歸特徵）**

| 特點 | 說明 |
|------|------|
| 趨勢溫和 | 歐股趨勢性不如美股強，均值回歸特徵更明顯 |
| 多交易所 | 同一股票在多交易所上市→價差套利機會 |
| 匯率干擾 | 技術分析需區分「本幣」與「外幣」圖形 |
| 分紅影響 | 高分紅導致除息缺口頻繁，技術圖形易失真 |
| 交易時段 | 歐股開盤前美股盤後已發生→歐股常跟隨美股跳空 |
| 波動率 | 整體波動率低於美股和A股 |

**最有效技術工具**：
1. 布林通道（均值回歸策略的核心）
2. 均線系統（MA50、MA200）
3. RSI（歐股的超買超賣信號比美股更可靠）
4. 支撐壓力位（歷史價位有效性高）
5. ATR（平均真實波幅，衡量波動率）

**歐股特有注意事項**：
- 歐股對美股隔夜走勢高度敏感→開盤跳空常見
- 除息日技術圖形需用「還原權息」圖表分析
- 歐洲多國假期不同→流動性在假期前後顯著下降
- 英國脫歐後，英股與歐陸股技術節奏可能不同步

## 四、籌碼分析差異

### 4.1 美股籌碼分析體系

| 資料來源 | 更新頻率 | 說明 | 重要性 |
|----------|---------|------|--------|
| **13F持倉報告** | 季度（滯後45天） | 管理規模≥1億美元的機構須披露持倉 | ★★★★★ |
| **Form 4（內部人交易）** | 2個工作日內 | CEO、CFO、董事買賣股票 | ★★★★☆ |
| **Short Interest** | 每月兩次（FINRA） | 空頭持倉比例、Days to Cover | ★★★★☆ |
| **Options Flow** | 即時 | 大單期權交易、異常活動監測 | ★★★★★ |
| **Dark Pool** | 延遲 | 暗池交易占比約40%，大額機構交易常在此完成 | ★★★☆☆ |
| **ETF資金流** | 每日 | 追蹤SPY、QQQ等ETF的淨流入/流出 | ★★★★☆ |
| **分析師評級** | 持續 | 一致預期、目標價變化、升降級 | ★★★★☆ |
| **Proxy Statements** | 年度 | 股東大會提案、機構投票立場 | ★★☆☆☆ |

**美股籌碼分析核心邏輯**：
- 13F報告雖有45天滯後，但可追蹤頂級基金（Berkshire、Bridgewater等）的持倉變化趨勢
- Options Flow是最即時的機構動向指標：大額Call買入=看多、大額Put買入=看空或避險
- Short Interest超過20%的股票有「軋空」（Short Squeeze）風險
- 內部人（Insider）買入比賣出更有參考價值（賣出可能僅是多元化需求）
- Dark Pool大額買入常預示機構建倉

### 4.2 A股籌碼分析體系

| 資料來源 | 更新頻率 | 說明 | 重要性 |
|----------|---------|------|--------|
| **北向資金** | 每日（即時） | 外資通過滬港通/深港通買賣A股，被視為「聰明錢」 | ★★★★★ |
| **三大法人** | 每日 | 公募/險資/社保/QFII、私募、遊資（自營商） | ★★★★★ |
| **龍虎榜** | 每日 | 漲跌幅>7%、換手率>20%等條件觸發，披露買賣前5大券商分點 | ★★★★☆ |
| **十大流通股東** | 季度 | 前10大股東持股變化 | ★★★★☆ |
| **股東人數** | 季度/半年 | 股東人數減少=籌碼集中=看多信號 | ★★★★☆ |
| **千張大戶** | 每週 | 大戶持股比例變化（台股概念，A股看前100大股東集中度） | ★★★☆☆ |
| **融資融券** | 每日 | 融資餘額變化反映散戶槓桿情緒（融券2024年7月後幾乎停擺） | ★★★★☆ |
| **大宗交易** | 每日 | 折價/溢價率、買方營業部 | ★★★☆☆ |
| **限售股解禁** | 可預期 | 解禁前後拋壓預期 | ★★★★☆ |
| **券商分點追蹤** | 每日 | 特定營業部的交易風格識別（隔日沖、趨勢型等） | ★★★☆☆ |

**A股籌碼分析核心邏輯**：
- **北向資金**是最重要的單一指標：持續淨買入=外資看好→中期看多
  - 2025年中長期資金新增入市超1萬億元
  - 北向資金持股市值約2.41萬億元
- **龍虎榜**可識別「遊資」操作模式（如「隔日沖」席位）
- **股東人數持續減少**是籌碼集中的強信號
- **融資餘額**上升+股價上漲=健康；融資上升+股價不漲=警惕出貨
- **限售股解禁**是已知的確定性利空，須提前評估拋壓
- 融券暫停後，A股實質上只能做多，下跌時缺乏對沖工具

### 4.3 歐股籌碼分析體系

| 資料來源 | 更新頻率 | 說明 | 重要性 |
|----------|---------|------|--------|
| **重大持股披露** | 事件觸發 | 持股超過3%/5%/10%等門檻須公告 | ★★★★★ |
| **空頭持倉披露** | 每日更新 | 淨空頭≥0.1%須報告監管；≥0.5%須公開 | ★★★★★ |
| **機構投資者持倉** | 季度/年度 | 歐洲機構透明度高，UCITS基金須定期披露 | ★★★★☆ |
| **主權財富基金** | 年度 | 挪威主權基金（NBIM）、阿布達比投資局等持倉追蹤 | ★★★☆☆ |
| **管理層交易** | 事件觸發 | MAR要求內部人交易須在3個工作日內披露 | ★★★★☆ |
| **ETF資金流** | 每日 | UCITS ETF資金流向 | ★★★☆☆ |

**歐股籌碼分析核心邏輯**：
- 空頭披露制度是歐股獨有的強大工具：可直接看到誰在做空、空了多少
- 重大持股披露門檻低（3%起），比美國（5%以上才需13D/13G）更透明
- 歐洲的臨時禁空令歷史表明：監管可能突然干預做空（2008、2020年均有先例）
- 挪威主權財富基金（全球最大，超過1.7萬億美元）的持倉變化具指標意義
- 英國脫歐後，英國採用新的輕觸式做空披露制度（The Short Selling Regulations 2025）

## 六、量化策略的地區適應性

### 6.1 因子有效性三市場對比

| 因子 | 美股 | A股 | 歐股 | 差異原因 |
|------|------|-----|------|---------|
| **價值（HML）** | 長期有效但近年被成長股壓制 | 有效，但須排除殼價值干擾（註冊制後改善） | 相對穩定且持續有效 | 美股科技權重大；A股散戶偏好題材 |
| **動量（MOM）** | ★★★★★ 最有效 | ★★☆☆☆ 中長期動量弱，短期反轉更有效 | ★★★☆☆ 介於兩者之間 | A股散戶追漲殺跌→反轉效應顯著 |
| **規模（SMB）** | 大盤股長期跑贏 | 小盤股效應曾經顯著（註冊制後減弱） | 小盤溢價溫和 | A股殼價值推高小盤股；美股大型股享壟斷溢價 |
| **質量（QMJ）** | ★★★★☆ 穩定有效 | ★★★★☆ 三市場都有效 | ★★★★★ 表現最穩定 | 質量因子跨市場普遍有效 |
| **低波動（BAB）** | ★★★★☆ 穩定 | ★★☆☆☆ 不穩定（投機氛圍） | ★★★★☆ 穩定 | A股散戶偏好高波動→低波動股被冷落 |
| **高股息** | ★★★☆☆ | ★★★★★ 近年A股最強因子之一 | ★★★★★ 歐洲傳統強項 | A股「中特估」+低利率環境推動高股息策略 |
| **反轉（Short-term Reversal）** | ★★☆☆☆ 較弱 | ★★★★★ 月度反轉利潤1.2%/月 | ★★★☆☆ | A股散戶追漲殺跌→反轉效應全球最強 |

**MSCI研究關鍵發現**（2010–2025年）：
- A股中高股息和最小波動率因子長期跑贏基準
- A股中純價格動量因子表現差→集體主義投資者行為導致
- 因子配置在市場穩定期主導回報；板塊配置在動盪期更有效
- 混合策略（因子+板塊）在所有市場環境中表現最穩定

### 6.2 回測注意事項

| 注意事項 | 美股 | A股 | 歐股 |
|----------|------|-----|------|
| **數據品質** | 優秀（CRSP、Compustat） | 須注意：除權除息處理、ST標記、停牌、財務造假 | 良好（Datastream、Bloomberg） |

| **指數成分變動** | 數據完備 | 須處理板塊轉換（主板→科創板等） | 須處理跨國指數變動 |

## 七、監管與合規差異

### 7.1 美國監管體系

| 監管機構/法規 | 內容 | 對分析師的影響 |
|-------------|------|-------------|
| **SEC（證券交易委員會）** | 全面監管證券市場、信息披露、反欺詐 | 信息披露最完整，分析資料最豐富 |
| **FINRA** | 自律監管組織，監管券商和交易員 | Short Interest數據來源 |
| **SOX法案** | CEO/CFO須對財報準確性負刑責 | 財報可信度相對最高 |
| **Reg NMS** | 全國市場系統，確保最佳執行 | 保障投資者獲得最佳價格 |
| **Reg FD** | 禁止選擇性信息披露 | 信息面相對公平 |
| **Pattern Day Trader** | 保證金帳戶<$25,000不得頻繁日內交易 | 小額帳戶受限 |
| **Wash Sale Rule** | 30天內買回已虧損賣出的股票不得抵稅 | 稅務規劃須注意 |
| **13F/13D/13G** | 機構持倉、大額持股披露 | 籌碼分析核心數據來源 |

### 7.2 中國監管體系

| 監管機構/法規 | 內容 | 對分析師的影響 |
|-------------|------|-------------|
| **CSRC（證監會）** | 全面監管，權力高度集中 | 監管政策變化可瞬間改變市場格局 |
| **減持新規** | 大股東、董監高減持須公告且受比例限制 | 可預判拋壓 |
| **轉融券暫停** | 2024年7月起全面暫停轉融通業務 | 實質上無法做空個股 |
| **退市新規** | 財務類、交易類、規範類、重大違法四類退市 | 須評估退市風險（面值退市、市值退市加速） |
| **註冊制** | 2023年全面推行，降低上市門檻 | 新股供給增加→殼價值消失、打新收益下降 |
| **IPO窗口指導** | 監管可暫停或放緩IPO節奏 | IPO暫停=市場穩定信號 |
| **跨境投資限制** | QFII/RQFII、陸港通額度 | 外資進出受制度約束 |
| **「窗口指導」** | 非正式的行政指導，無書面法規 | 不可預測的監管風險 |

## 九、各市場分析實戰SOP

### 9.1 美股分析SOP

```
步驟1：宏觀環境確認
├── Fed政策路徑（加息/降息/暫停）
├── 10Y美債殖利率趨勢
├── 美元指數方向
└── VIX水平（<15平靜、15-25正常、>25警戒、>35恐慌）

步驟2：板塊選擇
├── 根據景氣循環選強勢板塊
├── 確認板塊ETF資金流入方向
└── 觀察板塊間相對強弱（RS Rating）

步驟3：個股篩選（多因子）
├── 質量篩選：ROE>15%, 負債率<50%, FCF為正
├── 動量確認：過去6個月漲幅前30%
├── 估值合理：Forward P/E < 行業中位數 或 PEG<1.5
└── 盈餘品質：連續3季Earnings Beat

步驟4：籌碼確認
├── 13F：頂級基金是否持有/增持
├── Options Flow：是否有大額Call買入
├── Short Interest：空頭比例是否異常
└── Insider：內部人近期有無買入

步驟5：技術進場
├── 均線多頭排列（MA50>MA200）
├── 突破前高/盤整區間
├── 成交量放大確認
└── VWAP之上

步驟6：風控設定
├── 止損：-8%至-10%（或關鍵支撐位下方）
├── 止盈：技術目標位或基本面目標價
├── 倉位：單一個股不超過總資金5-10%
└── 期權保護：必要時買入Put對沖
```

---

### 9.2 A股分析SOP

```
步驟1：政策面判斷（最重要！）
├── 近期中央/國務院政策方向
├── 證監會最新監管動態
├── 人民銀行貨幣政策操作
└── 行業政策（利好還是利空？）

步驟2：資金面確認
├── 北向資金連續淨買入/賣出天數
├── 融資餘額變化趨勢
├── 社融/M2數據走向
└── 國家隊是否入場

步驟3：板塊選擇
├── 政策利好板塊優先
├── 北向資金集中買入的板塊
├── 板塊輪動節奏（領漲→補漲→滯漲）
└── 避開近期被監管整頓的行業

步驟4：個股篩選
├── 質量篩選：ROE>10%, 現金流為正
├── 高股息篩選：股息率>3%, 連續3年穩定派息
├── 籌碼確認：股東人數減少、北向持續買入
└── 估值合理：P/E < 行業中位數

步驟5：技術進場
├── 成交量分析（量在價先）
├── 均線支撐確認（MA20是短期生命線）
├── MACD底背離/金叉
└── 注意漲跌停板影響

步驟6：風控設定（A股風控更重要！）
├── 止損：-5%至-8%
├── 注意T+1限制：買入當天不能止損
├── 單一個股不超過總資金10%
├── 避開限售股解禁期
├── 關注政策面突變風險
└── 無法做空對沖→倉位控制是唯一防禦
```

---

### 9.3 歐股分析SOP

```
步驟1：宏觀環境
├── ECB利率政策路徑
├── 歐元區PMI與通脹趨勢
├── 能源價格（天然氣、油價）
├── 匯率走勢（EUR/USD、GBP/USD）
└── 地緣政治風險評估

步驟2：板塊選擇
├── 歐洲景氣循環定位
├── 能源價格對工業股的影響判斷
├── ESG政策受益板塊
└── 分紅型板塊（公用事業、電信、金融）

步驟3：個股篩選
├── 質量篩選：ROE>12%, 穩定派息歷史
├── 股息篩選：Dividend Yield>3%, 派息覆蓋率>1.5x
├── 價值篩選：EV/EBITDA < 行業中位數
├── ESG篩選：ESG評分>行業平均
└── 匯率敞口分析：營收幣種結構

步驟4：籌碼確認
├── 空頭持倉披露（>0.5%公開數據）
├── 重大持股變動（3%/5%/10%門檻）
├── 管理層交易記錄
└── 主權財富基金持倉變化

步驟5：技術進場
├── 均值回歸信號（布林通道觸及下軌）
├── RSI超賣確認（歐股RSI信號更可靠）
├── 均線支撐
└── 注意除息日對技術圖形的影響

步驟6：風控設定
├── 止損：-6%至-10%
├── 匯率對沖：考慮FX Forward或ETF對沖
├── 分散多國配置降低單一國家風險
├── 注意歐洲假期流動性下降
└── 監控臨時禁空令風險
```
