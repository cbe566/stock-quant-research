# 第一篇：財務報表分析

## 一、三大報表深度解讀

三大財務報表（資產負債表、損益表、現金流量表）是所有基本面分析的根基。三者相互連結，缺一不可。淨利潤從損益表流入資產負債表的保留盈餘，同時作為現金流量表的起點。

---

### 1.1 資產負債表（Balance Sheet）

**本質**：反映企業在某一特定時點的財務狀況快照。

**核心公式**：資產 = 負債 + 股東權益

#### 1.1.1 流動資產（Current Assets）

| 科目 | 說明 | 分析重點 |
|------|------|----------|
| 現金及等價物 | 銀行存款、短期國債、貨幣市場基金 | 過多可能代表資本配置效率低；過少則有流動性風險 |
| 應收帳款 | 已確認收入但尚未收到的款項 | 增長速度遠超營收增長 → 可能有虛增收入風險 |
| 存貨 | 原材料、在製品、製成品 | 存貨異常堆積 → 需求放緩或過度生產的訊號 |
| 預付費用 | 預先支付但尚未消耗的費用 | 金額較小，通常非分析重點 |
| 短期投資 | 一年內到期的有價證券 | 注意公允價值變動對損益的影響 |

**分析要點**：
- 應收帳款天數（DSO）= 應收帳款 / 日均營收，若持續上升需警覺
- 存貨天數（DIO）= 存貨 / 日均銷貨成本，行業間差異大需橫向對比
- 流動資產品質：現金 > 應收帳款 > 存貨（依流動性排序）

#### 1.1.2 非流動資產（Non-Current Assets）

| 科目 | 說明 | 分析重點 |
|------|------|----------|
| 固定資產（PP&E） | 土地、廠房、機器設備 | 資本密集型行業佔比高；注意折舊政策變更 |
| 無形資產 | 專利、商標、商譽 | 商譽過高需警惕減值風險 |
| 長期投資 | 持有超過一年的投資 | 關注投資收益波動對核心獲利的干擾 |
| 使用權資產（IFRS 16） | 租賃資產資本化 | 2019年後新準則，影響負債比率 |

**分析要點**：
- 商譽 / 總資產比率 > 30% → 減值風險顯著
- 資本支出（CapEx）趨勢 → 判斷公司是否在擴張或收縮
- 固定資產年齡 = 累計折舊 / 當期折舊費用 → 設備老化程度

#### 1.1.3 負債結構（Liabilities）

| 類別 | 科目 | 風險等級 |
|------|------|----------|
| 流動負債 | 應付帳款、短期借款、一年內到期長債 | 高（短期償還壓力） |
| 非流動負債 | 長期借款、公司債、退休金負債 | 中（關注到期日分佈） |
| 或有負債 | 訴訟、擔保、承諾 | 不確定（報表附註揭露） |

**分析要點**：
- 短期借款佔比過高 → 「短債長用」風險
- 債務到期時程分佈 → 是否有集中到期的再融資壓力
- 利率結構：固定利率 vs. 浮動利率 → 利率風險暴露

#### 1.1.4 股東權益（Shareholders' Equity）

| 科目 | 說明 |
|------|------|
| 股本 | 已發行股份面值 |
| 資本公積 | 超過面值的溢價發行收入 |
| 保留盈餘 | 累計淨利潤扣除已發放股利 |
| 其他綜合損益 | 外幣換算、金融資產未實現損益等 |
| 庫藏股 | 公司買回的自家股票（減項） |

**分析要點**：
- 保留盈餘持續增長 → 公司具備長期獲利能力
- 庫藏股增加 → 公司認為股價被低估（正面訊號）或管理層提升EPS
- 其他綜合損益波動大 → 可能隱藏匯率或投資風險

---

### 1.2 損益表（Income Statement）

**本質**：反映企業在一段期間內的經營成果（通常為季度或年度）。

#### 完整損益結構

```
營業收入（Revenue）
 - 營業成本（COGS）
═══════════════════════
= 毛利（Gross Profit）
 - 營業費用（Operating Expenses）
   - 銷售費用（S&M）
   - 管理費用（G&A）
   - 研發費用（R&D）
   - 折舊與攤銷（D&A）
═══════════════════════
= 營業利潤（Operating Income / EBIT）
 + 營業外收入
 - 營業外支出
 - 利息費用
═══════════════════════
= 稅前利潤（EBT）
 - 所得稅
═══════════════════════
= 淨利潤（Net Income）
 / 加權平均股數
═══════════════════════
= 每股盈餘（EPS）
```

#### 關鍵利潤指標

| 指標 | 公式 | 意義 |
|------|------|------|
| 毛利率 | 毛利 / 營收 | 產品競爭力與定價能力 |
| 營業利潤率 | 營業利潤 / 營收 | 核心業務獲利能力 |
| EBITDA利潤率 | EBITDA / 營收 | 排除折舊攤銷的現金獲利能力 |
| 淨利率 | 淨利 / 營收 | 最終股東可獲得的利潤比例 |
| EPS | 淨利 / 加權平均股數 | 每股獲利能力（最常用的估值基礎） |

**分析要點**：
- **營收品質**：有機增長 vs. 併購增長 vs. 匯率效應
- **毛利率趨勢**：穩定或上升 → 定價能力強或成本控制佳
- **營業槓桿**：固定成本佔比高的公司，營收增長時利潤率擴張更快
- **一次性項目**：剔除非經常性損益後的「調整後淨利」更能反映真實獲利
- **EPS稀釋**：注意股票選擇權、可轉債等對稀釋EPS的影響

#### 常見陷阱

1. **收入確認操縱**：提前認列收入（channel stuffing）、虛構訂單
2. **費用資本化**：將應費用化的支出計入資產（如WorldCom案例）
3. **一次性收益美化**：出售資產創造非經常性利潤掩蓋本業衰退
4. **關聯方交易**：與關聯企業的非常規交易需特別注意

---

### 1.3 現金流量表（Cash Flow Statement）

**本質**：反映企業在一段期間內的現金流入與流出，是最難被操縱的報表。

#### 三大現金流分類

##### （1）經營活動現金流（CFO - Cash Flow from Operations）

**間接法起點**：淨利潤

**常見調整項目**：
- (+) 折舊與攤銷（非現金費用加回）
- (+/-) 營運資金變動（應收帳款、存貨、應付帳款）
- (+/-) 其他非現金項目（股份支付、遞延稅項）

**判斷標準**：
- CFO > 淨利潤 → 盈餘品質良好（現金確實收到）
- CFO < 淨利潤 → 盈餘品質可疑（利潤停留在紙面上）
- CFO 持續為負 → 核心業務無法產生現金，高度危險

##### （2）投資活動現金流（CFI - Cash Flow from Investing）

| 項目 | 現金流方向 | 解讀 |
|------|-----------|------|
| 購買固定資產（CapEx） | 流出 | 擴張投資 |
| 出售固定資產 | 流入 | 可能縮減業務 |
| 收購企業 | 流出 | 外部增長策略 |
| 出售投資 | 流入 | 注意是否為美化現金流 |

**判斷標準**：
- 成長期公司CFI通常為大額負值（大量投資）
- 成熟期公司CFI趨於穩定
- CFI突然大幅為正 → 可能在變賣資產

##### （3）融資活動現金流（CFF - Cash Flow from Financing）

| 項目 | 現金流方向 | 解讀 |
|------|-----------|------|
| 發行股票 | 流入 | 股權融資（注意稀釋效應） |
| 回購股票 | 流出 | 回饋股東或提升EPS |
| 發行債券/借款 | 流入 | 債務融資（注意槓桿上升） |
| 償還債務 | 流出 | 降低槓桿 |
| 發放股利 | 流出 | 回饋股東 |

#### 自由現金流（Free Cash Flow, FCF）

**公式**：FCF = CFO - CapEx

**意義**：企業在維持營運與投資後，可自由分配給所有資本提供者的現金。

**判斷標準**：
- FCF 持續為正且增長 → 財務健康的核心指標
- FCF Yield（FCF / 市值）> 5% → 可能被低估
- FCF 轉換率 = FCF / 淨利，理想值 > 80%

#### 現金流組合解讀

| CFO | CFI | CFF | 解讀 |
|-----|-----|-----|------|
| + | - | - | **最佳狀態**：營運賺錢，投資擴張，回饋股東/還債 |
| + | - | + | 成長期：營運有現金但不足，需額外融資投資 |
| + | + | - | 成熟/衰退：營運加賣資產用來還債 |
| - | - | + | 燒錢期：核心不賺錢，靠融資撐住（新創常見） |
| - | + | + | 危險：靠賣資產和借錢維生 |
| - | + | - | 高度危險：賣資產還債，營運虧損 |

## 二、關鍵財務比率

### 2.1 盈利能力指標

#### ROE — 股東權益報酬率（杜邦分析拆解）

**公式**：ROE = 淨利潤 / 股東權益

**三因子杜邦分析**（DuPont Analysis）：

```
ROE = 淨利率 × 資產周轉率 × 權益乘數

    = (淨利/營收) × (營收/總資產) × (總資產/股東權益)
```

| 因子 | 提升策略 | 風險 |
|------|----------|------|
| 淨利率 | 提高售價、降低成本 | 需求彈性限制 |
| 資產周轉率 | 提升營運效率、輕資產模式 | 品質可能下降 |
| 權益乘數 | 增加財務槓桿 | 財務風險上升 |

**五因子杜邦分析**（Bloomberg版本）：

```
ROE = 稅負比率 × 利息負擔比率 × EBIT利潤率 × 資產周轉率 × 權益乘數

    = (淨利/EBT) × (EBT/EBIT) × (EBIT/營收) × (營收/總資產) × (總資產/權益)
```

**判斷標準**：
- ROE > 15% → 優秀（長期穩定更佳）
- ROE > 20% → 卓越（需確認是否來自高槓桿）
- ROE持續下降 → 競爭優勢侵蝕的訊號
- 高ROE + 低槓桿 → 真正的優質企業

**常見陷阱**：
- 高槓桿可人為推升ROE，但增加財務風險
- 大量回購庫藏股導致股東權益下降，ROE虛高
- 虧損導致股東權益為負時ROE失去意義

#### ROA — 總資產報酬率

**公式**：ROA = 淨利潤 / 平均總資產

**判斷標準**：
- 行業差異極大：銀行業約1%，科技業可達15%+
- 同行比較最有意義
- ROA趨勢比絕對值更重要

#### ROIC — 投入資本報酬率

**公式**：ROIC = NOPAT / 投入資本 = 稅後營業利潤 / (股東權益 + 有息負債 - 現金)

**判斷標準**：
- ROIC > WACC → 企業在創造價值
- ROIC < WACC → 企業在毀滅價值
- ROIC > 15% 且持續多年 → 具備競爭優勢的強力證據

**為何ROIC優於ROE**：
- 不受資本結構影響，更能反映核心業務的真實回報
- 避免了高槓桿對ROE的扭曲
- 是Warren Buffett和Charlie Munger最看重的指標之一

#### 毛利率與淨利率

| 指標 | 優秀標準 | 注意事項 |
|------|----------|----------|
| 毛利率 | > 40%（因行業而異） | 趨勢比絕對值重要；軟體業可達80%+，零售業可能僅20% |
| 營業利潤率 | > 15% | 反映管理效率與規模經濟 |
| 淨利率 | > 10% | 注意非經常性損益的扭曲 |
| EBITDA利潤率 | > 20% | 適合資本密集型行業對比 |

---

### 2.2 營運效率指標

| 指標 | 公式 | 判斷標準 | 意義 |
|------|------|----------|------|
| 存貨周轉率 | 銷貨成本 / 平均存貨 | 越高越好（同行比較） | 存貨管理效率 |
| 存貨周轉天數 | 365 / 存貨周轉率 | 越低越好 | 存貨變現速度 |
| 應收帳款周轉率 | 營收 / 平均應收帳款 | 越高越好 | 收款效率 |
| 應收帳款天數（DSO） | 365 / 應收帳款周轉率 | < 45天較佳 | 平均收款時間 |
| 應付帳款天數（DPO） | 365 × 平均應付帳款 / COGS | 較長代表議價能力強 | 對供應商付款時間 |
| 總資產周轉率 | 營收 / 平均總資產 | 因行業而異 | 資產利用效率 |

**現金轉換週期（CCC）**：

```
CCC = DIO + DSO - DPO
```

- CCC為負 → 如Amazon模式，先收客戶錢再付供應商（極優）
- CCC下降趨勢 → 營運效率改善
- CCC上升趨勢 → 資金被佔用增加，效率惡化

---

### 2.3 償債能力指標

#### 短期償債能力

| 指標 | 公式 | 健康標準 | 風險標準 |
|------|------|----------|----------|
| 流動比率 | 流動資產 / 流動負債 | > 1.5 | < 1.0 |
| 速動比率 | (流動資產 - 存貨) / 流動負債 | > 1.0 | < 0.5 |
| 現金比率 | 現金 / 流動負債 | > 0.3 | < 0.1 |

**注意**：流動比率過高（> 3.0）可能代表資產閒置、資本配置效率低。

#### 長期償債能力

| 指標 | 公式 | 健康標準 | 風險標準 |
|------|------|----------|----------|
| 負債比率 | 總負債 / 總資產 | < 50% | > 70% |
| 負債/權益比 | 總負債 / 股東權益 | < 1.0 | > 2.0 |
| 淨負債/EBITDA | (有息負債-現金) / EBITDA | < 2.0x | > 4.0x |
| 利息覆蓋倍數 | EBIT / 利息費用 | > 5.0x | < 2.0x |

**常見陷阱**：
- 表外負債（經營租賃、特殊目的實體）可能被忽略
- IFRS 16實施後租賃負債入表，需注意新舊準則比較
- 行業特性：公用事業、房地產行業天然高槓桿
- 銀行業使用資本適足率（CAR）而非一般負債比率

---

### 2.4 估值比率

| 指標 | 公式 | 常見判斷標準 | 適用場景 |
|------|------|-------------|----------|
| P/E（本益比） | 股價 / EPS | 成長股 20-40x；價值股 8-15x | 獲利穩定的公司 |
| Forward P/E | 股價 / 預估EPS | 低於歷史均值可能被低估 | 有可靠盈餘預測時 |
| P/B（股價淨值比） | 股價 / 每股淨值 | < 1.0 可能被低估 | 資產密集型（銀行、製造） |
| P/S（股價營收比） | 股價 / 每股營收 | < 2.0 較為合理 | 尚未獲利的高成長公司 |
| EV/EBITDA | 企業價值 / EBITDA | < 10x 可能被低估 | 跨國、跨行業比較最佳 |
| EV/EBIT | 企業價值 / EBIT | < 15x 較為合理 | 資本支出差異大時使用 |
| PEG | P/E / 盈餘成長率 | < 1.0 → 成長被低估 | 成長股估值 |
| FCF Yield | FCF / 市值 | > 5% 可能被低估 | 成熟現金流穩定公司 |
| 股息殖利率 | 年度股利 / 股價 | > 3% 對收益型投資人有吸引力 | 成熟穩定派息公司 |

**企業價值（EV）**：

```
EV = 市值 + 有息負債 + 優先股 + 少數股東權益 - 現金及等價物
```

**估值比率的局限**：
- 週期性行業：低P/E可能是高峰期（即將下滑），高P/E可能是谷底期（即將回升）
- 不同會計準則（IFRS vs. US GAAP）影響比較
- 需搭配成長率、品質指標綜合評估
- 單一估值指標永遠不夠，至少使用2-3個交叉驗證

---

## 三、財務品質分析

### 3.1 應計比率與盈餘品質（Accrual Analysis）

#### 理論基礎

1996年，密西根大學教授Richard Sloan發表突破性研究，發現盈餘中「應計項目」（非現金部分）佔比越高的公司，未來股價表現越差。這被稱為「應計異常」（Accrual Anomaly），是最被廣泛研究的市場無效率現象之一。

**核心邏輯**：盈餘 = 現金流 + 應計項目。應計項目比例越高，盈餘品質越低，因為管理層有更大的操縱空間。

#### Sloan應計比率計算

**資產負債表法**：
```
應計項目 = (ΔCA - ΔCash) - (ΔCL - ΔSTD - ΔITP) - D&A

其中：
ΔCA = 流動資產變動
ΔCash = 現金變動
ΔCL = 流動負債變動
ΔSTD = 短期借款變動
ΔITP = 應付所得稅變動
D&A = 折舊攤銷
```

**現金流量表法（更簡易）**：
```
應計比率 = (淨利潤 - 經營活動現金流) / 平均總資產
```

#### 判斷標準

| 應計比率 | 盈餘品質 | 投資意義 |
|----------|----------|----------|
| < -5% | 極高 | 現金流遠超帳面利潤，最安全 |
| -5% ~ 0% | 高 | 健康範圍 |
| 0% ~ 5% | 中等 | 需進一步分析 |
| 5% ~ 10% | 低 | 警告訊號 |
| > 10% | 極低 | 可能存在盈餘操縱 |

#### 歷史績效驗證

Sloan的原始研究（1962-2001年）顯示，做多低應計比率股票、做空高應計比率股票的策略，年化回報近18%，是同期S&P 500的7.4%的兩倍以上。

#### 常見陷阱
- 高成長公司自然會有較高的應計項目（應收帳款、存貨增長），不一定代表造假
- 需搭配營收成長率一起分析，有機成長帶來的應計增加較為合理
- 行業差異大，應在同行業內比較

---

### 3.2 Beneish M-Score（財務舞弊偵測模型）

#### 理論基礎

1999年由Indiana University教授Messod D. Beneish提出的統計模型，利用8個財務比率加權計算，偵測公司是否存在盈餘操縱的可能性。該模型成功預警了Enron等重大財務舞弊案件。

#### 8個變數詳解

| 變數 | 名稱 | 公式 | 意義 |
|------|------|------|------|
| DSRI | 應收帳款天數指數 | DSO(t) / DSO(t-1) | > 1 表示應收帳款增長超過營收，可能虛增收入 |
| GMI | 毛利率指數 | 毛利率(t-1) / 毛利率(t) | > 1 表示毛利率下滑，有操縱動機 |
| AQI | 資產品質指數 | [1-(CA+PPE)/TA](t) / [1-(CA+PPE)/TA](t-1) | > 1 表示資產中無形/其他資產比例增加 |
| SGI | 營收增長指數 | 營收(t) / 營收(t-1) | 高增長公司更可能操縱，但非直接證據 |
| DEPI | 折舊指數 | 折舊率(t-1) / 折舊率(t) | > 1 表示放慢折舊速度以美化利潤 |
| SGAI | 管銷費用指數 | SGA%收入(t) / SGA%收入(t-1) | 費用異常變動的偵測 |
| TATA | 應計項目/總資產 | (淨利 - CFO) / 總資產 | 直接衡量盈餘品質 |
| LVGI | 槓桿指數 | 槓桿(t) / 槓桿(t-1) | > 1 表示借款增加 |

#### M-Score公式（8變數模型）

```
M-Score = -4.84 + 0.920×DSRI + 0.528×GMI + 0.404×AQI + 0.892×SGI
          + 0.115×DEPI - 0.172×SGAI + 4.679×TATA - 0.327×LVGI
```

#### 判斷標準

| M-Score | 判斷 |
|---------|------|
| < -2.22 | **安全**：操縱可能性極低（5變數模型門檻） |
| -2.22 ~ -1.78 | **灰色地帶**：需進一步調查 |
| > -1.78 | **警告**：很可能存在盈餘操縱（8變數模型門檻） |

#### 模型準確度

Beneish在樣本外測試中，正確識別76%的操縱公司，誤判率為17.5%的非操縱公司。

#### 常見陷阱
- 模型對高成長公司的SGI敏感，可能產生誤報
- 金融業（銀行、保險）因財務結構特殊，模型適用性較低
- M-Score是機率性工具，不是確定性判斷，需搭配其他分析方法
- 單一年度異常可能是合理的業務變化，需觀察連續趨勢

---

### 3.3 Altman Z-Score（破產風險預測模型）

#### 理論基礎

1968年由紐約大學Stern商學院教授Edward I. Altman提出，利用5個財務比率預測企業兩年內的破產機率。至今仍是最廣泛使用的信用風險評估工具之一。

#### 三個版本

##### 版本一：製造業上市公司（原始版本）

```
Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 1.0×X5
```

| 變數 | 比率 | 衡量面向 |
|------|------|----------|
| X1 | 營運資金 / 總資產 | 短期流動性 |
| X2 | 保留盈餘 / 總資產 | 累計獲利能力（公司年齡代理） |
| X3 | EBIT / 總資產 | 營運效率（權重最高 3.3） |
| X4 | 股票市值 / 總負債 | 市場對償債能力的信心 |
| X5 | 營收 / 總資產 | 資產周轉效率 |

**判斷標準**：

| Z-Score | 區域 | 解讀 |
|---------|------|------|
| > 2.99 | 安全區 | 破產風險極低 |
| 1.81 ~ 2.99 | 灰色區 | 需進一步關注 |
| < 1.81 | 危險區 | 破產風險高 |

##### 版本二：非上市公司（Z'-Score）

```
Z' = 0.717×X1 + 0.847×X2 + 3.107×X3 + 0.420×X4' + 0.998×X5
```

其中X4' = 股東權益帳面價值 / 總負債（替代市值）

**判斷標準**：安全 > 2.90；灰色 1.23~2.90；危險 < 1.23

##### 版本三：非製造業/新興市場（Z''-Score）

```
Z'' = 6.56×X1 + 3.26×X2 + 6.72×X3 + 1.05×X4'
```

移除了X5（資產周轉率），因其在不同行業間差異過大。

**判斷標準**：安全 > 2.60；灰色 1.10~2.60；危險 < 1.10

#### 常見陷阱
- 金融業、房地產業等高槓桿行業天然Z-Score偏低，需用行業專用模型
- 大型企業即使Z-Score偏低，也可能因政府救助等因素不會破產
- 模型基於歷史數據，對突發性危機（如疫情）預測能力有限
- 需動態追蹤Z-Score變化趨勢，單期數值參考價值有限

---

### 3.4 Piotroski F-Score（價值股篩選評分系統）

#### 理論基礎

2000年由Stanford大學會計學教授Joseph Piotroski提出，旨在從高帳面市值比（低P/B）的價值股中，進一步篩選出財務基本面健全的公司。原始研究顯示（1976-1996年），F-Score 8-9分的價值股年化超額回報達13.4%。

#### 9項評分標準

每項符合得1分，滿分9分。

##### A. 獲利能力（Profitability）— 4項

| # | 指標 | 得分條件 | 邏輯 |
|---|------|----------|------|
| F1 | ROA | 當年ROA > 0 | 公司能獲利 |
| F2 | 經營現金流 | 當年CFO > 0 | 有現金流入 |
| F3 | ROA變化 | ΔROA > 0（年增） | 獲利能力改善 |
| F4 | 盈餘品質 | CFO > 淨利 | 現金支撐盈餘 |

##### B. 槓桿/流動性/資金來源（Leverage）— 3項

| # | 指標 | 得分條件 | 邏輯 |
|---|------|----------|------|
| F5 | 長期負債比率變化 | 長期負債/總資產 下降 | 降低財務風險 |
| F6 | 流動比率變化 | 流動比率 上升 | 短期償債能力改善 |
| F7 | 股份稀釋 | 未發行新股（流通股數未增加） | 未稀釋股東權益 |

##### C. 營運效率（Operating Efficiency）— 2項

| # | 指標 | 得分條件 | 邏輯 |
|---|------|----------|------|
| F8 | 毛利率變化 | Δ毛利率 > 0（年增） | 定價能力或成本控制改善 |
| F9 | 資產周轉率變化 | Δ資產周轉率 > 0 | 資產利用效率提升 |

#### 判斷標準與應用

| F-Score | 評級 | 投資建議 |
|---------|------|----------|
| 8-9 | 強勢 | 優質價值股候選，考慮買入 |
| 5-7 | 中等 | 需進一步研究其他基本面 |
| 3-4 | 偏弱 | 謹慎，可能有價值陷阱 |
| 0-2 | 弱勢 | 考慮做空或迴避 |

#### 實際應用步驟
1. 先按P/B由低到高排序，選出前20%的價值股池
2. 計算每檔股票的F-Score（9項評分）
3. 選擇F-Score ≥ 7的股票進入最終名單
4. 搭配其他質量指標（ROIC、FCF）進行最終篩選

#### 常見陷阱
- 不適用於金融業（銀行、保險），因其財務結構特殊
- 不適用於高資本支出行業（電信、公用事業），高負債是行業常態
- 二元評分系統對邊界值敏感，微小變化可能改變分數
- 歷史回測績效不保證未來表現，需搭配其他分析方法

---

## 四、DCF估值模型

### 4.1 模型概述

DCF（Discounted Cash Flow，折現現金流模型）是最基礎也最重要的內在價值估值方法。核心思想是：一家公司的價值等於其未來所有現金流的現值之和。

```
企業價值 = Σ [FCFt / (1+r)^t] + 終值 / (1+r)^n
```

### 4.2 FCFF模型（Free Cash Flow to Firm）

**適用**：計算企業整體價值（Enterprise Value），再扣除淨負債得到股權價值。

**FCFF計算**：
```
FCFF = EBIT × (1 - 稅率) + 折舊攤銷 - 資本支出 - 營運資金增量

或

FCFF = 淨利 + 折舊攤銷 + 利息費用×(1-稅率) - 資本支出 - 營運資金增量
```

**折現率**：使用WACC

**得到**：企業價值（EV）→ 股權價值 = EV - 淨負債

### 4.3 FCFE模型（Free Cash Flow to Equity）

**適用**：直接計算股權價值，不需要扣除負債。

**FCFE計算**：
```
FCFE = 淨利 + 折舊攤銷 - 資本支出 - 營運資金增量 + 淨借款
```

**折現率**：使用股權成本（Cost of Equity）

**得到**：直接得到股權價值

### 4.4 WACC計算（加權平均資本成本）

```
WACC = E/(E+D) × Ke + D/(E+D) × Kd × (1-T)

其中：
E = 股權市值
D = 有息負債市值
Ke = 股權成本
Kd = 債務成本（稅前）
T = 有效稅率
```

#### 股權成本（CAPM模型）

```
Ke = Rf + β × (Rm - Rf) + 其他風險溢價

其中：
Rf = 無風險利率（通常用10年期國債殖利率）
β = 系統性風險係數
Rm - Rf = 股權風險溢價（ERP，通常4.5%-6.5%）
```

**Beta的選擇**：
- 使用5年月度回歸Beta（最常見）
- 小型股可加上規模溢價（Size Premium）
- 新興市場可加上國家風險溢價（Country Risk Premium）

#### 典型WACC範圍

| 公司類型 | 典型WACC |
|----------|----------|
| 大型穩定公司（公用事業） | 5%-7% |
| 成熟科技公司 | 8%-10% |
| 成長型公司 | 10%-13% |
| 高風險新創 | 15%-25% |

### 4.5 終值計算（Terminal Value）

終值通常佔DCF估值的60%-80%，是最敏感的假設之一。

#### 方法一：永續增長法（Gordon Growth Model）

```
終值 = FCF(n+1) / (WACC - g) = FCF(n) × (1+g) / (WACC - g)

其中：g = 永續增長率
```

**永續增長率（g）的選擇**：
- 通常設定為2%-3%，接近長期GDP成長率
- 不應超過長期通膨率 + 實質GDP成長率
- 超過4%幾乎都不合理（意味著公司最終超過整個經濟體）

#### 方法二：退出倍數法（Exit Multiple）

```
終值 = FCF(n) × 退出EV/EBITDA倍數

或

終值 = EBITDA(n) × 退出倍數
```

**退出倍數的選擇**：
- 參考同業當前交易倍數
- 考慮行業成熟度：成熟行業用較低倍數（6-8x），高成長行業可用更高倍數
- 兩種方法交叉驗證結果

### 4.6 敏感度分析（Sensitivity Analysis）

DCF模型對假設高度敏感，必須進行敏感度分析。

#### 雙變數敏感度表（最常用）

| | WACC 8% | WACC 9% | WACC 10% | WACC 11% | WACC 12% |
|---|---------|---------|----------|----------|----------|
| g=1.0% | $XX | $XX | $XX | $XX | $XX |
| g=1.5% | $XX | $XX | $XX | $XX | $XX |
| g=2.0% | $XX | $XX | **基準** | $XX | $XX |
| g=2.5% | $XX | $XX | $XX | $XX | $XX |
| g=3.0% | $XX | $XX | $XX | $XX | $XX |

**關鍵敏感度測試變數**：
1. WACC（±1-2%）
2. 永續增長率（±0.5-1%）
3. 營收增長率假設
4. 長期營業利潤率假設
5. 資本支出/營收比率

**經驗法則**：WACC變動1個百分點，估值變動10%-20%。

### 4.7 DCF實際操作步驟

1. **預測期間**：通常5-10年（高成長公司用較長期間）
2. **建立三表模型**：損益表、資產負債表、現金流量表互相連結
3. **計算每年FCFF或FCFE**
4. **確定WACC或Ke**
5. **計算終值**
6. **折現所有現金流與終值**
7. **得到企業價值 → 扣除淨負債 → 除以股數 → 每股內在價值**
8. **進行敏感度分析**
9. **與市場價格比較，判斷高估或低估**

### 4.8 DCF的局限與陷阱

- **GIGO原則**（Garbage In, Garbage Out）：輸入假設錯誤，結果一定錯誤
- **終值依賴**：終值佔比過高使估值容易被微小假設改變左右
- **不適用場景**：週期性公司（盈餘波動大）、虧損公司、金融業
- **過度精確的幻覺**：看似精確的數字可能建立在高度不確定的假設上
- **最佳實踐**：搭配相對估值法（P/E、EV/EBITDA可比較）交叉驗證

# 第二篇：行業分析框架

## 一、Porter五力分析

### 理論基礎

1979年由Harvard商學院教授Michael E. Porter在《Harvard Business Review》發表，是分析行業競爭結構最經典的框架。五力決定了一個行業的長期平均利潤率。

### 五力詳解

#### 力量一：現有競爭者的競爭強度（Rivalry Among Existing Competitors）

**影響因素**：

| 因素 | 高競爭（不利） | 低競爭（有利） |
|------|---------------|---------------|
| 競爭者數量 | 眾多且實力相當 | 少數，有明顯領導者 |
| 行業增長率 | 低增長（零和博弈） | 高增長（不需搶對手份額） |
| 產品差異化 | 低（同質化商品） | 高（品牌、技術壁壘） |
| 退出障礙 | 高（專用資產、情感因素） | 低（容易轉型） |
| 固定成本佔比 | 高（需維持高產能利用率） | 低 |

#### 力量二：新進入者的威脅（Threat of New Entrants）

**進入障礙**：

| 障礙類型 | 高壁壘（有利） | 低壁壘（不利） |
|----------|---------------|---------------|
| 資本需求 | 巨額投資（半導體fab） | 低資本（App開發） |
| 規模經濟 | 顯著（汽車製造） | 不顯著 |
| 品牌忠誠度 | 強（可口可樂） | 弱 |
| 轉換成本 | 高（企業軟體） | 低（消費品） |
| 法規/牌照 | 嚴格管制（銀行、電信） | 開放進入 |
| 專利/技術 | 大量專利保護（製藥） | 技術門檻低 |
| 通路掌握 | 渠道被壟斷 | 易取得通路 |

#### 力量三：替代品的威脅（Threat of Substitutes）

**分析維度**：
- 替代品的性價比是否持續改善？
- 買方的轉換成本高還是低？
- 替代品來自哪個行業？（跨界顛覆往往最具破壞力）

**範例**：
- 串流媒體對傳統有線電視的替代
- 電動車對燃油車的替代
- 視訊會議對商務旅行的部分替代

#### 力量四：買方的議價能力（Bargaining Power of Buyers）

**高議價能力的條件**：
- 買方集中度高（少數大客戶佔營收比例大）
- 產品標準化（容易換供應商）
- 買方對價格敏感
- 買方有後向整合的能力（自行生產）
- 買方資訊充分（網路時代更容易比價）

#### 力量五：供應商的議價能力（Bargaining Power of Suppliers）

**高議價能力的條件**：
- 供應商集中度高（少數壟斷供應）
- 原料/零組件難以替代
- 供應商有前向整合的能力
- 行業不是供應商的主要客戶

### 五力分析實際應用步驟

1. 明確定義分析的「行業」邊界
2. 逐一評估五力的強弱（低/中/高）
3. 綜合判斷行業吸引力
4. 識別最強的力量（決定策略重點）
5. 尋找可改變力量平衡的策略機會

### 常見陷阱
- 行業定義過廣或過窄會導致分析失準
- 五力是靜態框架，需定期更新以反映行業變化
- 忽略互補品的角色（Porter後來補充了第六力）
- 過度關注當前競爭者，忽略跨界顛覆

## 二、行業生命週期

### 四階段模型

```
 收入/利潤
    │
    │                    ╭────╮
    │               ╭───╯    │  成熟期
    │           ╭──╯         │
    │      成長期│            ╰──╮
    │     ╭─╯                    ╰── 衰退期
    │ 導入期
    │╭──╯
    └──────────────────────────────────── 時間
```

| 階段 | 特徵 | 增長率 | 競爭 | 利潤 | 現金流 | 投資重點 |
|------|------|--------|------|------|--------|----------|
| **導入期** | 新產品/技術，市場認知低 | 極高但基數小 | 少數先驅者 | 負或微利 | 大量燒錢 | 研發、市場教育 |
| **成長期** | 需求快速增加，新進入者湧入 | 20%+ | 增加中 | 快速改善 | 逐漸轉正 | 產能擴張、市場份額 |
| **成熟期** | 增長放緩，市場飽和 | GDP水準 | 穩定寡占 | 最高 | 穩定正向 | 效率優化、股東回饋 |
| **衰退期** | 需求萎縮，技術被替代 | 負增長 | 退出開始 | 下降 | 仍可為正 | 收割、轉型 |

### 各階段投資策略

| 階段 | 估值方法 | 風險特徵 | 適合投資者 |
|------|----------|----------|-----------|
| 導入期 | P/S、TAM滲透率 | 高風險高回報 | 風險承受能力高 |
| 成長期 | PEG、EV/營收增長率 | 中高風險 | 成長型投資者 |
| 成熟期 | P/E、EV/EBITDA、DCF | 中低風險 | 價值型/收益型 |
| 衰退期 | 清算價值、NAV | 價值陷阱風險 | 特殊狀況投資者 |

## 三、產業鏈分析

### 分析框架

```
上游（原材料/零組件）
    ↓ 供應商議價能力
中游（製造/加工/組裝）
    ↓ 附加價值
下游（品牌/通路/終端）
    ↓ 客戶議價能力
終端消費者
```

### 分析步驟

1. **繪製產業鏈地圖**：識別上中下游所有關鍵環節
2. **利潤分佈分析**：哪個環節賺最多？（微笑曲線理論：研發和品牌兩端利潤最高）
3. **瓶頸識別**：哪個環節是供應鏈瓶頸？（如先進製程晶圓代工）
4. **垂直整合趨勢**：是否有企業在往上下游延伸？
5. **技術變革衝擊**：新技術可能顛覆哪個環節？

### 微笑曲線（Smile Curve）

```
利潤率
 高│  研發設計        品牌服務
   │    ╲              ╱
   │     ╲            ╱
   │      ╲          ╱
 低│       ╲╌╌╌╌╌╌╌╱
   │        製造組裝
   └────────────────────
   上游      中游      下游
```

**投資含義**：優先投資微笑曲線兩端（高附加價值環節）的企業。

## 四、競爭格局與護城河（Moat）分析

### 護城河分類（Warren Buffett / Pat Dorsey框架）

#### 1. 無形資產護城河

| 類型 | 說明 | 範例 | 持久性 |
|------|------|------|--------|
| 品牌 | 消費者願意為品牌支付溢價 | 蘋果、愛馬仕、可口可樂 | 高 |
| 專利 | 法律保護的獨佔權 | 輝瑞、高通 | 中（有期限） |
| 法規許可 | 政府准入限制 | 銀行牌照、賭場執照 | 高 |

#### 2. 轉換成本護城河

客戶更換供應商的成本（金錢、時間、風險）極高。

**範例**：
- 企業軟體（SAP、Oracle）— 數據遷移成本極高
- 銀行帳戶 — 自動扣款、直接入帳轉換麻煩
- 醫療器材 — 醫生熟悉特定器材的操作

#### 3. 網路效應護城河

用戶越多，平台價值越高，吸引更多用戶加入。

**範例**：
- 社群媒體（Facebook、WeChat）
- 支付網路（Visa、Mastercard）
- 雙邊市場（Uber、Airbnb）

#### 4. 成本優勢護城河

| 來源 | 說明 | 範例 |
|------|------|------|
| 規模經濟 | 產量越大，單位成本越低 | Walmart、台積電 |
| 位置優勢 | 靠近原料或市場 | 水泥廠、砂石場 |
| 獨特資源 | 擁有低成本資源 | 沙特阿美（低成本石油） |
| 製程優勢 | 專有技術降低成本 | 台積電先進製程 |

#### 5. 有效規模護城河

市場規模有限，僅容納少數參與者獲利。新進入者加入會導致所有人虧損。

**範例**：特定地區的機場、鐵路、管線公司。

### 護城河評估步驟

1. **識別**：公司是否擁有上述任何護城河？
2. **量化**：護城河帶來多大的超額利潤？（ROIC > WACC的持續年限）
3. **持久性**：護城河是否會被侵蝕？技術變革、監管變化是否威脅護城河？
4. **趨勢**：護城河是在加寬還是變窄？

### 護城河強度判斷

| ROIC表現 | 持續年限 | 護城河評級 |
|----------|----------|-----------|
| ROIC > WACC + 10% | > 10年 | 寬闊護城河（Wide Moat） |
| ROIC > WACC + 5% | 5-10年 | 狹窄護城河（Narrow Moat） |
| ROIC ≈ WACC | < 5年 | 無護城河（No Moat） |

## 五、TAM/SAM/SOM市場規模估算

### 定義

```
TAM（Total Addressable Market）
 │ 全部潛在市場規模
 └──→ SAM（Serviceable Addressable Market）
       │ 公司產品/地域可觸及的市場
       └──→ SOM（Serviceable Obtainable Market）
             │ 考慮競爭後，實際可獲得的市場份額
```

| 指標 | 定義 | 計算範例 |
|------|------|----------|
| TAM | 如果100%市場滲透率，最大市場規模 | 全球智慧手機市場 = 80億人 × 某比例 × 均價 |
| SAM | 受限於產品特性、地理覆蓋、客戶群 | 高端智慧手機市場（>$500）× 公司進入的國家 |
| SOM | 基於競爭格局，合理可獲得的份額 | SAM × 目標市場佔有率（如5%-15%） |

### 兩種估算方法

#### 自上而下（Top-Down）
1. 從總體市場數據出發（市場研究報告）
2. 逐步縮小到目標市場
3. 優點：快速、宏觀
4. 缺點：可能過於樂觀

#### 自下而上（Bottom-Up）
1. 從單位經濟（客戶數 × 客單價）出發
2. 逐步加總得出市場規模
3. 優點：更貼近實際
4. 缺點：可能遺漏潛在市場

### 常見陷阱
- TAM膨脹（把不相關的市場算進去）
- 忽略市場增長率（靜態 vs. 動態市場）
- SOM過於樂觀（忽略競爭和執行風險）
- 混淆收入TAM和利潤TAM

# 第三篇：宏觀經濟分析

## 一、景氣循環四階段

### 理論基礎

經濟活動呈現週期性波動，典型週期為3-10年。NBER（美國國家經濟研究局）是官方的景氣循環判定機構。

### 四階段特徵

| 階段 | GDP | 通膨 | 利率 | 失業率 | 企業盈餘 | 股市表現 |
|------|-----|------|------|--------|----------|----------|
| **擴張期** | 加速增長 | 逐漸上升 | 央行開始升息 | 持續下降 | 快速增長 | 牛市（早中期最佳） |
| **高峰期** | 增速見頂 | 高位 | 持續升息/高位 | 接近最低 | 增速放緩 | 震盪，開始見頂 |
| **收縮期** | 負增長 | 開始回落 | 央行降息 | 快速上升 | 大幅下滑 | 熊市 |
| **谷底期** | 觸底回穩 | 低位 | 維持低利率 | 接近最高 | 開始回穩 | 築底反彈 |

### 景氣循環判斷方法

#### 方法一：領先指標法

| 指標類型 | 代表指標 | 領先時間 |
|----------|----------|----------|
| 領先指標 | ISM新訂單、收益率曲線、股市、建築許可、消費者預期 | 6-12個月 |
| 同步指標 | 工業生產、零售銷售、就業、個人收入 | 同步 |
| 落後指標 | 失業率、CPI、銀行放款、單位勞動成本 | 3-12個月 |

**美國諮議會（Conference Board）領先經濟指標（LEI）**：
- 包含10個領先指標的綜合指數
- 連續3個月下滑 → 衰退警告
- 年變動率轉負 → 強烈衰退訊號

#### 方法二：收益率曲線法

```
正常收益率曲線（經濟擴張）：
殖利率
  │          ╱──── 長期利率
  │     ╱───╯
  │  ╱─╯
  │╱
  └──────────── 年期
  1Y  2Y  5Y  10Y  30Y

倒掛收益率曲線（衰退預警）：
殖利率
  │╲
  │  ╲──╮
  │     ╰───╮
  │          ╰── 長期利率
  └──────────── 年期
```

**2Y/10Y利差**：
- 正常 > 0：經濟預期良好
- 倒掛 < 0：每次倒掛在過去50年都預示了衰退（領先12-24個月）
- 解除倒掛後：衰退通常在6-12個月內到來

#### 方法三：Fidelity景氣循環模型

Fidelity Investments使用四象限模型，結合多項指標判定當前所處階段：
1. 早週期（Early）：政策寬鬆，信貸改善，盈餘加速
2. 中週期（Mid）：增長穩健，信貸健康，最長階段
3. 晚週期（Late）：通膨上升，政策收緊，利差收窄
4. 衰退（Recession）：盈餘下滑，失業上升，政策轉鬆

## 二、核心經濟指標

### 2.1 GDP（國內生產毛額）

| 面向 | 說明 |
|------|------|
| 定義 | 一國在特定期間內所有最終商品與服務的市場價值 |
| 公式 | GDP = C + I + G + (X - M) |
| 公布頻率 | 季度（美國有初值、修正值、終值三次公布） |
| 市場關注 | 環比年率（QoQ annualized）更受關注 |

**判斷標準**：
- 連續兩季GDP負增長 → 技術性衰退的通用定義
- 增長率 > 3% → 強勁（美國標準）
- 增長率 2-3% → 溫和
- 增長率 < 1% → 疲弱

### 2.2 通膨指標

#### CPI（消費者物價指數）

| 指標 | 說明 | 判斷標準 |
|------|------|----------|
| 總CPI | 包含食品和能源 | 波動較大 |
| 核心CPI | 排除食品和能源 | Fed更關注此指標 |
| CPI年增率 | 與去年同期比較 | 目標約2%（多數央行） |
| CPI月增率 | 與上月比較 | > 0.3%/月需警覺 |

#### PCE（個人消費支出平減指數）

- 美聯儲偏好的通膨指標（與CPI不同的權重和計算方法）
- 核心PCE是Fed政策決策的核心依據
- 目標：2%年增率

#### PPI（生產者物價指數）

- 衡量生產端的價格壓力
- PPI上升往往領先CPI上升（成本傳導）
- 核心PPI排除食品和能源

### 2.3 就業指標

#### 非農就業報告（NFP）

| 指標 | 說明 | 判斷標準 |
|------|------|----------|
| 非農新增就業 | 每月新增非農就業人數 | > 200K 強勁；< 100K 疲弱 |
| 失業率 | U-3官方失業率 | < 4% 充分就業；> 6% 衰退 |
| U-6失業率 | 包含不充分就業和氣餒工人 | 更全面反映就業市場 |
| 勞動參與率 | 勞動力佔適齡人口比例 | 下降可能掩蓋真實失業 |
| 平均時薪年增率 | 薪資通膨壓力指標 | > 4% 可能推升通膨 |

#### 每週初領失業救濟金

- 高頻指標（每週公布）
- < 250K → 勞動市場健康
- > 300K → 放緩訊號
- 持續上升趨勢 → 衰退預警

### 2.4 PMI（採購經理人指數）

| 指標 | 公布機構 | 頻率 |
|------|----------|------|
| ISM製造業PMI | 美國供應管理協會 | 月度 |
| ISM服務業PMI | 美國供應管理協會 | 月度 |
| Markit PMI | S&P Global | 月度（有初值） |
| 財新PMI | 財新/Markit | 月度（中國） |

**判斷標準**：

| PMI值 | 解讀 |
|-------|------|
| > 55 | 強勁擴張 |
| 50-55 | 溫和擴張 |
| 50 | 榮枯分界線 |
| 45-50 | 溫和收縮 |
| < 45 | 嚴重收縮（通常伴隨衰退） |

**PMI子項目分析**：
- 新訂單 > 50 → 未來需求健康（最具領先性）
- 新訂單 - 存貨缺口 → 正向缺口預示生產回升
- 就業分項 → 反映企業招聘意願
- 價格分項 → 反映通膨壓力

### 2.5 消費者信心指數

| 指標 | 公布機構 | 側重 |
|------|----------|------|
| 密歇根大學消費者信心指數 | 密歇根大學 | 消費者對未來的預期 |
| 諮議會消費者信心指數 | Conference Board | 消費者對當前狀況的評估 |

**意義**：消費佔美國GDP約70%，消費者信心直接影響支出意願。

### 2.6 零售銷售與工業生產

| 指標 | 意義 | 判斷標準 |
|------|------|----------|
| 零售銷售月增率 | 消費端實時狀況 | > 0.5% 強勁 |
| 核心零售（排除汽車、汽油） | 排除波動大項目 | 趨勢比單月重要 |
| 工業生產指數 | 製造業產出 | 年增率與GDP高度相關 |
| 產能利用率 | 實際產出/潛在產出 | > 80% → 可能推升通膨 |

## 三、央行政策分析

### 3.1 美聯儲（Federal Reserve）政策框架

#### 雙重使命（Dual Mandate）

1. **物價穩定**：核心PCE年增率目標2%（2020年後採用「平均通膨目標」AIT）
2. **充分就業**：失業率接近自然失業率（約4%）

#### 政策工具

| 工具 | 說明 | 對市場影響 |
|------|------|-----------|
| 聯邦基金利率（FFR） | 主要政策工具 | 直接影響短期利率和信貸成本 |
| 量化寬鬆（QE） | 購買國債和MBS | 壓低長期利率、增加流動性 |
| 量化緊縮（QT） | 縮減資產負債表 | 推升長期利率、減少流動性 |
| 前瞻指引（Forward Guidance） | 口頭引導市場預期 | 管理預期，降低政策不確定性 |
| 逆回購/準備金利率 | 短期流動性管理 | 維持貨幣市場利率在目標區間 |

#### 點陣圖（Dot Plot）

- 每季FOMC會議公布（3/6/9/12月）
- 顯示每位Fed官員對未來利率路徑的預測
- 中位數點代表市場最關注的「共識預期」
- 與市場定價（Fed Funds Futures）的差距是重要交易訊號

#### 利率決議分析框架

1. **決議前**：
   - CME FedWatch工具查看市場定價機率
   - 關注最新經濟數據（CPI、NFP、GDP）
   - 注意Fed官員講話的「鷹派/鴿派」傾向

2. **決議當天**：
   - 利率決定（升/降/不變）
   - 聲明措辭變化（逐字比較前次聲明）
   - 經濟展望更新（SEP）
   - 點陣圖變化

3. **決議後**：
   - 主席記者會的口風
   - FOMC會議紀要（會後3週公布）

### 3.2 QE/QT對市場的影響

#### QE（量化寬鬆）的傳導機制

```
央行購買國債/MBS
    ↓
壓低長期利率
    ↓
降低企業融資成本 + 推升資產價格（股票、房地產）
    ↓
財富效應 → 促進消費與投資
    ↓
刺激經濟增長
```

**市場影響**：
- 股市：強烈正面（流動性充沛，TINA效應—There Is No Alternative）
- 債市：債券價格上升，殖利率下降
- 匯率：本幣傾向貶值
- 商品：受益於弱美元和通膨預期

#### QT（量化緊縮）的傳導機制

- 與QE相反：央行不再續購到期債券或主動出售
- 減少銀行體系準備金
- 推升長期利率
- 降低市場流動性
- 歷史經驗：2018年QT導致市場劇烈波動

### 3.3 收益率曲線分析

#### 正常形態（Normal）
- 短期利率 < 長期利率
- 反映正常的期限溢價和通膨預期
- 有利於銀行業（借短貸長賺利差）

#### 平坦化（Flattening）
- 短長期利差收窄
- 通常發生在升息週期中後期
- 市場開始擔憂經濟放緩

#### 倒掛（Inverted）
- 短期利率 > 長期利率（2Y > 10Y）
- **歷史上最可靠的衰退預警指標**
- 過去8次倒掛，每次都跟隨了經濟衰退
- 平均領先衰退12-18個月
- 但「假陽性」：並非每次倒掛後經濟都立即衰退，時間差異大

#### 陡峭化（Steepening）
- 短長期利差擴大
- 若由短端下降驅動（Bull Steepening）→ 市場預期降息
- 若由長端上升驅動（Bear Steepening）→ 通膨預期上升

# 第四篇：技術分析核心

## 一、趨勢分析

### 1.1 道氏理論（Dow Theory）

**六大基本原則**：

1. **市場反映一切資訊**：價格已包含所有已知資訊
2. **市場有三種趨勢**：
   - 主要趨勢（Primary）：持續1-3年
   - 次要趨勢（Secondary）：持續3週-3個月，回調主趨勢的33%-66%
   - 短期波動（Minor）：持續不到3週
3. **主要趨勢有三個階段**：
   - 累積期（聰明錢進場）
   - 公眾參與期（大眾跟進，趨勢加速）
   - 過度投機期（聰明錢出場，散戶接最後一棒）
4. **指數必須相互確認**：工業指數和運輸指數需同步創新高才確認牛市
5. **成交量確認趨勢**：量升價漲確認上升趨勢，量升價跌確認下降趨勢
6. **趨勢持續到明確的反轉訊號出現**

### 1.2 移動平均線系統（MA System）

| 均線 | 代表意義 | 常用策略 |
|------|----------|----------|
| MA5/MA10 | 短期趨勢 | 短線進出場 |
| MA20 | 月線/中短期 | 波段交易基準 |
| MA50 | 季線/中期 | 中期趨勢判斷 |
| MA60 | 近似季線 | 台股常用 |
| MA120 | 半年線 | 中長期支撐壓力 |
| MA200 | 年線/長期 | 牛熊分界線（機構最常用） |

**關鍵訊號**：

| 訊號 | 定義 | 解讀 |
|------|------|------|
| 黃金交叉 | 短期均線由下往上穿越長期均線 | 看漲訊號 |
| 死亡交叉 | 短期均線由上往下穿越長期均線 | 看跌訊號 |
| 均線多頭排列 | 短 > 中 > 長均線依序排列 | 強勢上升趨勢 |
| 均線空頭排列 | 短 < 中 < 長均線依序排列 | 強勢下降趨勢 |
| 股價站上MA200 | 價格在年線之上 | 長期趨勢向上 |
| 股價跌破MA200 | 價格在年線之下 | 長期趨勢向下 |

**EMA vs. SMA**：
- SMA（簡單移動平均）：等權重計算
- EMA（指數移動平均）：近期權重較高，反應更快
- 短線交易者偏好EMA，長線投資者偏好SMA

### 1.3 趨勢線繪製

**上升趨勢線**：連接兩個以上的波段低點
**下降趨勢線**：連接兩個以上的波段高點

**有效性判斷**：
- 至少3個觸及點才算可靠
- 觸及次數越多，趨勢線越可靠
- 突破趨勢線需配合成交量確認
- 假突破常見：等待收盤價確認（而非盤中突破）

## 二、形態分析

### 2.1 反轉形態

| 形態 | 出現位置 | 成功率 | 目標價計算 |
|------|----------|--------|-----------|
| 頭肩頂 | 上升趨勢末端 | 約83%（突破頸線後） | 頸線 - (頭到頸線距離) |
| 頭肩底 | 下降趨勢末端 | 約74% | 頸線 + (頭到頸線距離) |
| 雙重頂（M頭） | 上升趨勢末端 | 約73% | 頸線 - (頂到頸線距離) |
| 雙重底（W底） | 下降趨勢末端 | 約78% | 頸線 + (底到頸線距離) |
| 三重頂/底 | 較為罕見 | 類似雙重頂/底 | 同上 |

**確認條件**：
- 突破頸線需伴隨成交量放大
- 回測頸線而不跌破 → 確認有效突破
- 反轉形態前必須有明確的先前趨勢

### 2.2 持續形態

| 形態 | 說明 | 突破方向 |
|------|------|----------|
| 上升三角形 | 水平壓力線 + 上升支撐線 | 偏向上突破（約77%） |
| 下降三角形 | 下降壓力線 + 水平支撐線 | 偏向下突破（約72%） |
| 對稱三角形 | 收斂的壓力線和支撐線 | 順原趨勢（約60%） |
| 旗形（Flag） | 小幅反向平行通道 | 順原趨勢（高可靠性） |
| 楔形（Wedge） | 收斂通道，反向於主趨勢 | 順原趨勢 |
| 矩形（Rectangle） | 水平通道盤整 | 順原趨勢（約65%） |

## 三、動量指標

### 3.1 RSI（相對強弱指數）

**公式**：RSI = 100 - [100 / (1 + RS)]，RS = 平均漲幅 / 平均跌幅（通常14期）

| RSI值 | 解讀 | 操作建議 |
|-------|------|----------|
| > 70 | 超買 | 不一定要立即賣出，強勢股可持續超買 |
| 50-70 | 偏多 | 上升趨勢中正常範圍 |
| 30-50 | 偏空 | 下降趨勢中正常範圍 |
| < 30 | 超賣 | 可能反彈機會，但弱勢股可持續超賣 |

**進階用法**：
- **RSI背離**：價格創新高但RSI未創新高 → 看跌背離（最有價值的訊號）
- **RSI趨勢線**：在RSI圖上也可以畫趨勢線
- **RSI 50線**：多頭市場RSI多在40-90之間波動，空頭市場在10-60之間

### 3.2 MACD（移動平均收斂發散指標）

**組成**：
```
MACD線 = 12日EMA - 26日EMA
信號線 = MACD線的9日EMA
柱狀圖 = MACD線 - 信號線
```

| 訊號 | 描述 | 解讀 |
|------|------|------|
| MACD線 > 0 | 短期均線在長期均線之上 | 中期趨勢向上 |
| MACD線 < 0 | 短期均線在長期均線之下 | 中期趨勢向下 |
| 黃金交叉 | MACD線由下穿越信號線 | 買入訊號 |
| 死亡交叉 | MACD線由上穿越信號線 | 賣出訊號 |
| 柱狀圖由負轉正 | 下跌動能減弱 | 可能轉多 |
| MACD背離 | 價格新高但MACD未新高 | 趨勢可能反轉 |

### 3.3 KD隨機指標（Stochastic Oscillator）

**公式**：
```
%K = (收盤價 - N期最低價) / (N期最高價 - N期最低價) × 100
%D = %K的M期移動平均（通常M=3）
```

**判斷標準**：

| 訊號 | 條件 | 解讀 |
|------|------|------|
| 超買 | K > 80 | 可能回調 |
| 超賣 | K < 20 | 可能反彈 |
| 黃金交叉 | K由下往上穿越D（在20以下更有效） | 買入訊號 |
| 死亡交叉 | K由上往下穿越D（在80以上更有效） | 賣出訊號 |
| KD鈍化 | K持續在80以上或20以下 | 強勢趨勢，不宜逆勢操作 |

## 四、波動指標

### 4.1 布林通道（Bollinger Bands）

**組成**：
```
中軌 = 20日SMA
上軌 = 中軌 + 2倍標準差
下軌 = 中軌 - 2倍標準差
```

**統計原理**：約95%的價格落在2倍標準差範圍內。

| 訊號 | 描述 | 解讀 |
|------|------|------|
| 價格觸及上軌 | 接近統計上界 | 超買（但強勢可沿上軌上行） |
| 價格觸及下軌 | 接近統計下界 | 超賣（但弱勢可沿下軌下行） |
| 帶寬收窄（Squeeze） | 波動率降低 | 大行情即將發動（方向不確定） |
| 帶寬擴大 | 波動率增加 | 趨勢正在進行中 |
| 走出通道又回來 | 假突破 | 可能反向運動 |

**帶寬指標（Bandwidth）**：
```
帶寬 = (上軌 - 下軌) / 中軌
```
帶寬歷史低點 → 「Squeeze」訊號，預示即將出現大波動。

### 4.2 ATR（平均真實波幅）

**公式**：
```
True Range = max(當日最高-最低, |當日最高-前日收盤|, |當日最低-前日收盤|)
ATR = True Range的N期移動平均（通常14期）
```

**應用**：
- **停損設定**：常見使用2倍ATR作為追蹤停損距離
- **倉位管理**：ATR越大（波動越高），應降低倉位
  - 每股風險金額 = 2 × ATR
  - 交易股數 = 帳戶風險金額 / 每股風險金額
- **波動率比較**：不同股票的ATR/股價比率可比較相對波動率

## 五、成交量分析

### 5.1 量價關係

| 價格 | 成交量 | 解讀 | 可靠度 |
|------|--------|------|--------|
| 上漲 | 放量 | 多方強勢，趨勢健康 | 高 |
| 上漲 | 縮量 | 上漲乏力，可能回調 | 中 |
| 下跌 | 放量 | 空方強勢，可能破位 | 高 |
| 下跌 | 縮量 | 賣壓減弱，可能止跌 | 中 |
| 盤整 | 放量 | 多空分歧加大，變盤在即 | 中 |
| 盤整 | 縮量 | 觀望氣氛濃厚 | 低 |

### 5.2 OBV（能量潮指標）

**計算**：
- 若今日收盤 > 昨日收盤：OBV = 前日OBV + 今日成交量
- 若今日收盤 < 昨日收盤：OBV = 前日OBV - 今日成交量
- 若收盤不變：OBV不變

**應用**：
- OBV趨勢向上 + 價格上漲 → 趨勢確認
- OBV與價格背離 → 趨勢可能反轉（最有價值的訊號）
- OBV先於價格突破 → 領先指標

### 5.3 VWAP（成交量加權平均價格）

**公式**：VWAP = Σ(價格 × 成交量) / Σ(成交量)

**應用**：
- 機構交易者的重要基準價格
- 價格 > VWAP → 多方主導
- 價格 < VWAP → 空方主導
- VWAP附近常有支撐/壓力

## 六、斐波那契回撤與延伸

### 6.1 關鍵比率

| 比率 | 來源 | 在交易中的角色 |
|------|------|---------------|
| 23.6% | 1 - 0.618^2 ≈ 0.236 | 淺回撤（強趨勢） |
| 38.2% | 1 - 0.618 | 溫和回撤 |
| **50.0%** | 不是黃金比率但廣泛使用 | 心理關卡 |
| **61.8%** | 黃金比率 | 最重要的回撤位 |
| 78.6% | √0.618 | 深度回撤（趨勢可能反轉） |

### 6.2 斐波那契回撤（Retracement）

**繪製方法**：
- 上升趨勢：從波段低點拉到波段高點
- 下降趨勢：從波段高點拉到波段低點

**應用**：
- 價格回撤到38.2%後反彈 → 強勢趨勢（淺回撤）
- 價格回撤到61.8%後反彈 → 仍在正常範圍
- 價格跌破78.6% → 原趨勢可能已結束

### 6.3 斐波那契延伸（Extension）

| 延伸比率 | 應用 |
|----------|------|
| 100% | 等幅延伸（最常見目標價） |
| 127.2% | 中等延伸 |
| 161.8% | 黃金延伸（第二常見目標價） |
| 200% | 雙倍延伸 |
| 261.8% | 大幅延伸 |

**最佳實踐**：
- 與水平支撐壓力線交匯處更可靠
- 與布林通道邊界重合時形成機構級支撐壓力區
- 多個時間週期的斐波那契水平重合 → 極強支撐壓力（Fibonacci Cluster）

## 七、支撐壓力分析

### 識別方法

| 方法 | 說明 | 可靠度 |
|------|------|--------|
| 前高/前低 | 歷史高低點形成的水平線 | 高 |
| 整數關口 | 如100元、10000點 | 中（心理因素） |
| 跳空缺口 | 缺口上下緣 | 中高 |
| 成交密集區 | 大量成交發生的價格區間 | 高 |
| 均線支撐壓力 | MA20/50/200 | 中高 |
| 斐波那契水平 | 38.2%、50%、61.8% | 中 |
| 趨勢線 | 上升/下降趨勢線 | 中 |

### 支撐壓力轉換原則

- 支撐被跌破後轉為壓力
- 壓力被突破後轉為支撐
- 測試次數越多的支撐壓力越可靠
- 伴隨大量成交量的突破更可信

### 常見陷阱
- 支撐壓力是「區間」而非精確價位
- 假突破（Fakeout）非常常見，需等待確認
- 不同時間週期的支撐壓力重要性不同（週線 > 日線 > 小時線）
- 極端市場環境下（恐慌拋售），技術支撐可能完全失效

# 第五篇：籌碼分析

## 一、三大法人動向分析

### 1.1 外資（Foreign Institutional Investors）

| 特性 | 說明 |
|------|------|
| 資金規模 | 最大（佔台股成交量約30%） |
| 操作風格 | 中長線為主，偏好大型權值股 |
| 資訊來源 | 全球宏觀 + 個股研究報告 |
| 影響力 | 對指數影響最大 |

**分析方法**：

| 觀察指標 | 多方訊號 | 空方訊號 |
|----------|----------|----------|
| 現貨買賣超 | 連續5日以上買超 | 連續5日以上賣超 |
| 期貨未平倉 | 淨多單增加 | 淨空單增加 |
| 選擇權Put/Call比 | 偏向賣Put | 偏向買Put |
| 匯率走勢 | 台幣升值（資金匯入） | 台幣貶值（資金匯出） |

**常見陷阱**：
- 「假外資」：本國資金透過海外管道回流，偽裝外資操作
- 外資包含多家機構，方向不一定一致
- 期貨部位可能是避險而非方向性交易

### 1.2 投信（Domestic Fund Managers）

| 特性 | 說明 |
|------|------|
| 資金規模 | 中等 |
| 操作風格 | 波段操作，季底作帳行情 |
| 偏好標的 | 中小型股、成長股 |
| 獨特行為 | 季末（3/6/9/12月）拉抬持股，作帳效應明顯 |

**分析方法**：
- 投信連續買超個股 → 可能是基金經理人在佈局
- 投信重倉股在季末有被拉抬的傾向
- 投信認養股（長期持有不賣的個股）值得關注

### 1.3 自營商（Proprietary Trading Desks）

| 特性 | 說明 |
|------|------|
| 資金規模 | 較小 |
| 操作風格 | 短線為主，當沖比例高 |
| 分為兩類 | 自行買賣（方向性）+ 避險買賣（對沖持倉） |
| 參考價值 | 需分開看，避險部位不代表方向 |

**分析要點**：
- 自營商自行買賣的淨買超較有參考價值
- 避險買賣多與權證、選擇權對沖有關
- 大量買超可能是造市需求，需仔細區分

## 二、千張大戶持股變化

### 定義

持有某一股票超過1,000張（100萬股）的投資人被歸類為「千張大戶」。

### 判斷標準

| 千張大戶持股比例 | 籌碼狀態 | 投資意義 |
|-----------------|----------|----------|
| < 40% | 籌碼分散 | 散戶為主，股價較難上漲 |
| 40% - 70% | 甜蜜區間 | 大戶有信心，同時有足夠流動性 |
| > 80% | 過度集中 | 流動性不足，可能有出貨風險 |

### 籌碼集中度趨勢

| 趨勢 | 解讀 |
|------|------|
| 大戶持股連續增加 + 股價上漲 | 主力吸籌，後勢看好 |
| 大戶持股連續增加 + 股價不漲 | 主力默默吸籌，醞釀中 |
| 大戶持股減少 + 股價上漲 | 主力趁高出貨，危險訊號 |
| 大戶持股減少 + 股價下跌 | 主力出貨中，迴避 |

### 數據來源
- 台灣集中保管結算所（TDCC）每週公布
- 分為1-5張、5-10張……400-600張、600-800張、800-1000張、1000張以上

## 三、券商分點追蹤

### 分析方法

**步驟**：
1. 找出連續在同一個股大量買進的券商分點
2. 判斷該分點是否為已知的主力券商（如外資常用分點）
3. 觀察買進後的持股情況（是否持續持有）
4. 注意「隔日沖」券商（當日拉高隔日出貨的慣性操作）

### 關鍵觀察

| 模式 | 解讀 |
|------|------|
| 單一分點連續大量買入 | 可能是主力在吸籌 |
| 多個分點同步買入 | 可能是外資系統性佈局 |
| 買入分散在眾多小分點 | 散戶行為，不具參考價值 |
| 已知「隔日沖」分點大量買入 | 當日可跟但隔日需出場 |

### 常見陷阱
- 主力可能使用多個分點分散買入，不易追蹤
- 隔日沖券商的操作可能造成股價短期虛漲
- 分點資料有一天的延遲，追蹤即時性有限

## 四、融資融券變化

### 融資（Margin Buying）

| 指標 | 說明 |
|------|------|
| 融資餘額 | 市場上借錢買股的總金額 |
| 融資使用率 | 融資餘額 / 融資限額 |
| 融資增減 | 每日融資淨增減 |

**判斷邏輯**：

| 融資 + 股價 | 解讀 |
|-------------|------|
| 融資增加 + 股價上漲 | 多頭健康（但需注意過熱） |
| 融資增加 + 股價不漲/下跌 | 散戶搶進但主力在賣，危險 |
| 融資減少 + 股價下跌 | 多殺多，融資斷頭（可能接近底部） |
| 融資減少 + 股價上漲 | 籌碼沈澱，最健康的上漲 |

### 融券（Short Selling）

| 指標 | 說明 |
|------|------|
| 融券餘額 | 被借出放空的股票數量 |
| 券資比 | 融券餘額 / 融資餘額 |
| 融券回補 | 強制或自願買回還券 |

**判斷邏輯**：

| 融券 | 解讀 |
|------|------|
| 融券大增 | 看空力量增加，但也代表潛在回補買盤 |
| 券資比 > 30% | 高軋空風險（Short Squeeze潛力） |
| 融券強制回補（除權息前） | 短期買盤湧入，可能推升股價 |

### 融資維持率

```
融資維持率 = (股票市值 + 融資保證金) / 融資金額 × 100%
```

- 維持率 < 130% → 券商發出追繳通知
- 維持率 < 120% → 強制斷頭賣出
- 大盤急跌 → 融資斷頭連鎖反應 → 加速下跌（但也是底部訊號之一）

## 五、內部人持股變動

### 定義

內部人包括：董事、監察人、經理人、持股10%以上大股東。

### 法規要求
- 台灣：內部人轉讓持股需事先申報（證交法第22-2條）
- 美國：SEC Form 4（2個工作日內申報）
- 中國：大股東減持需提前15日公告

### 分析邏輯

| 行為 | 解讀 | 可靠度 |
|------|------|--------|
| 內部人大量買入 | 最強的看多訊號（用自己的錢） | 很高 |
| 多位內部人同時買入 | 集體看好，訊號更強 | 極高 |
| 內部人賣出 | 不一定看空（可能是個人理財需要） | 中 |
| 內部人異常大量賣出 | 需要警覺（可能知道未公開利空） | 高 |
| 持股質押比例高 | 大股東可能資金緊張 | 中高 |

**Peter Lynch名言**：「內部人賣股的理由有很多，但買股的理由只有一個——他們認為股價會上漲。」

### 常見陷阱
- 內部人賣出不一定是看空訊號（稅務規劃、多元化、生活需要）
- 需觀察賣出的規模相對於總持股的比例
- 小額賣出通常無意義，大額賣出才需關注
- 定期計畫性賣出（10b5-1 Plan）的訊號意義較低

## 快速篩選檢查清單

### 第一步：排除地雷股
- [ ] Altman Z-Score > 1.81（非危險區）
- [ ] Beneish M-Score < -1.78（非操縱嫌疑）
- [ ] 經營活動現金流 > 0（核心業務能產生現金）
- [ ] 利息覆蓋倍數 > 3x（能支付利息）

### 第二步：品質篩選
- [ ] Piotroski F-Score ≥ 7（財務基本面健全）
- [ ] ROE > 15%（高股東回報）
- [ ] ROIC > WACC（創造價值）
- [ ] 應計比率 < 5%（盈餘品質良好）
- [ ] FCF轉換率 > 80%（現金支撐利潤）

### 第三步：估值篩選
- [ ] P/E < 行業中位數（相對便宜）
- [ ] EV/EBITDA < 10x（或行業合理範圍）
- [ ] PEG < 1.0（成長被低估）
- [ ] FCF Yield > 5%（現金流回報率高）

### 第四步：籌碼確認
- [ ] 三大法人偏多（外資/投信連續買超）
- [ ] 千張大戶持股比例穩定或增加
- [ ] 融資餘額未異常飆升
- [ ] 無內部人異常大量賣出

### 第五步：技術確認
- [ ] 股價站上MA20/MA50
- [ ] RSI在40-70的健康多頭區間
- [ ] MACD柱狀圖為正或由負轉正
- [ ] 成交量配合價格走勢
